# Отчёты и логи

**[← Назад в корень](../README.md)**


В этой папке хранятся результаты работы скриптов анализа и аудита.

## Автоматические отчеты

*   `duplicates_report.json`: Результат работы `scripts/find_duplicates.py`. Содержит список ключей, которые повторяются в разных категориях.
*   `synonym_cleanup_report.md`: Отчет о чистке синонимов.

## Использование

Файлы здесь генерируются автоматически. Не редактируйте их вручную, так как они будут перезаписаны при следующем запуске скриптов.
Для сохранения исторических данных перемещайте отчеты в `archive/reports/`.
