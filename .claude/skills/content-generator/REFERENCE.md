## FAQ Templates

### Типичные вопросы по категориям

**Автохимия:**

- Почему {продукт} не работает?
- Можно ли использовать на {поверхность}?
- Как часто применять?
- Чем отличается от {альтернатива}?
- Какую концентрацию использовать?

**Защитные покрытия:**

- Сколько держится {покрытие}?
- Как подготовить поверхность?
- Сколько слоёв наносить?
- Можно ли наносить на {поверхность}?
- Как восстановить {покрытие}?

**Инструменты:**

- Какой размер выбрать?
- Совместим ли с {оборудование}?
- Как ухаживать?
- Когда менять?
- Можно ли использовать для {задача}?

## Writing Guidelines

### Intro (30-60 words)

**Structure:** What + Why + Key Condition

```
✅ Активная пена для мойки автомобиля (Pre-wash) размягчает
   80–90% грязи, чтобы вы не поцарапали лак при ручной мойке.
   Для керамики берите нейтральную (pH 7–9), для зимней грязи —
   щелочную (pH 10–12).

❌ Активная пена — это современное средство для мойки автомобилей,
   которое пользуется большой популярностью среди автовладельцев...
```

### Tables

**Always include:**

- Type/variant names
- Key differentiator (pH, size, effect)
- Use case / when to choose
- Pros/cons if relevant

```markdown
| Тип | pH | Назначение | Влияние на защиту |
|-----|----|------------|-------------------|
| Щелочная | 10-12 | Зимняя грязь | Смывает воск |
| Нейтральная | 7-9 | Регулярная мойка | Безопасна для воска |
```

### Selection Scenarios

**Format:** Condition → Recommendation

```markdown
**Сценарии выбора:**
- **Обычная грязь, пыль** → щелочная (pH 10-12)
- **Авто с керамикой** → нейтральная (pH 7-9)
- **Экономия** → концентрат 5л
```

### How-To Steps

**Format:** Numbered list with bold action

```markdown
1. **Развести концентрат** по инструкции
2. **Нанести на сухой кузов** снизу вверх
3. **Выдержать 3-5 минут** — не давать высохнуть
4. **Смыть** сверху вниз под давлением
```

### Common Mistakes

**Format:** Mistake → Consequence

```markdown
**Частые ошибки:**
- **Пена высохла** → белёсые пятна
- **Превысили концентрацию** → разводы на пластике
- **Нанесли на горячий кузов** → пятна
```

### FAQ Answers

**Expert tone:** Specific, actionable, with context

```markdown
### Пена стекает как вода. Что делать?

Чаще дело в оборудовании, а не в химии. Дешёвые пенокомплекты
имеют плохую сеточку. Попробуйте увеличить концентрацию (1:5
вместо 1:10). Если не помогло — замените таблетку в пенообразователе.
```

## Word Count Guidelines

**December 2025 approach:** Length by competitor analysis, not formula.

| Category Type | Typical Range | Notes |
|--------------|---------------|-------|
| Simple chemical | 300-500 | Intro + Table + Steps + 3 FAQ |
| Complex chemical | 500-800 | + detailed scenarios, + more FAQ |
| Equipment | 400-600 | Focus on selection criteria |
| Accessories | 300-400 | Shorter, more visual |

**John Mueller:** "90-95% of that text is unnecessary... some amount of text is useful."

---

## Quality Checklist

- [ ] Intro: 30-60 words, self-contained
- [ ] Has comparison table
- [ ] Has numbered how-to steps
- [ ] Has 3-5 FAQ questions
- [ ] Keywords used naturally
- [ ] No "water" / filler text
- [ ] No brand names/prices
- [ ] Expert tone throughout
- [ ] Actionable advice
