## Intro (30-60 слов)

**Формула:** {Что это} + {Главная функция} + {Важное условие}

```
✅ "Чернитель резины — финишный состав для защиты шин от UV
   и придания глубокого цвета. Бывает водный (матовый) и
   силиконовый (глянцевый). Наносится только на сухую резину."

❌ "В этой статье мы расскажем о чернителях резины..."
❌ "Широкий ассортимент чернителей резины в нашем магазине..."
```

**Почему 30-60:** AI использует первый блок для Direct Answer. Должен быть самодостаточным.

## Антипаттерны (НЕ делать!)

| ❌ Ошибка | ✅ Правильно |
|-----------|--------------|
| "В этой статье..." | Сразу факт (Definition) |
| "Широкий ассортимент" | Конкретика: типы, объёмы |
| "Купить X можно..." | Таблица с типами |
| H1 = "Купить антибитум" | H1 = "Антибітум" |
| Keyword density 2%+ | Natural writing |
| 800 слов SEO-текста | 30-60 слов intro + таблица |

## FAQ Rules

- **3-5 вопросов** — реальные запросы пользователей
- **Экспертный тон** — конкретные ответы
- **БЕЗ FAQPage schema** — не работает для e-commerce (Google Aug 2023)
- Контент оставить — полезно для AI Overviews, Voice Search

---

## Workflow

1. Прочитать `categories/{slug}/data/{slug}_clean.json` — keywords
2. Прочитать `categories/{slug}/meta/{slug}_meta.json` — H1, types
3. Прочитать `categories/{slug}/research/RESEARCH_DATA.md` — факты
4. Написать контент по структуре выше
5. Сохранить в `categories/{slug}/content/{slug}_ru.md`
6. Валидация: `python3 scripts/validate_content.py {path}`
