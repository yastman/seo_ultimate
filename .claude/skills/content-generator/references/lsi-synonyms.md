# LSI-синонимы для разбавления переспама

Когда stem-группа превышает 2.5%, заменить часть вхождений на синонимы.

**Правило:** Целевые ключи из `_clean.json` НЕ заменять. Заменять только тематические слова.

---

## Семантические дубли — НЕ добавлять

Google склеивает вариации одного смысла. Stuffing дублями = переспам без SEO-пользы.

**❌ НЕ добавлять:**

| Дубль 1 | ≈ Дубль 2 | ≈ Дубль 3 |
|---------|-----------|-----------|
| омыватель для авто | омыватель для машины | омыватель для автомобиля |
| омыватель стекла | омыватель лобового стекла | омыватель для стекол |
| шампунь для авто | шампунь для машины | автомобильный шампунь |
| чернитель шин | чернитель для шин | чернитель резины |

**✅ Использовать только:** реально разные словоформы с разным корнем.

| Разные корни | Пример |
|--------------|--------|
| ✅ Разрешено | стеклоомыватель ≠ омыватель (разные слова) |
| ✅ Разрешено | антифриз ≠ незамерзайка (разные слова) |
| ❌ Запрещено | омыватель для авто ≈ омыватель для машины (один смысл) |

## Пример исправления

**Проблема:** "очиститель" 13 раз, тошнота 3.61

**Решение:**
1. Оставить 8-9 вхождений "очиститель" (целевой ключ)
2. Заменить 4-5 на: "состав", "средство", "продукт"

**Результат:** тошнота 3.61 → 3.32 ✅
