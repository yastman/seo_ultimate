## Параметры объёма (SEO 2025-2026)

| Параметр | Значение | Примечание |
|----------|----------|------------|
| **Общий объём** | 500-700 слов | НЕ больше! |
| **Intro** | 40-60 слов | Над товарами |
| **Body** | 300-450 слов | Под товарами |
| **FAQ** | 100-150 слов | 3-4 вопроса |

**Почему не 800-1200?**
- Helpful Content System бьёт по "стенам SEO-текста"
- Mobile First: 1000 слов = бесконечный скролл
- Размытие транзакционного интента

## Границы контента (Scope Control)

Контент категории отвечает на вопрос: **"Какой {товар} мне купить?"**

### НЕ отвечает на:

| Вопрос | Где должен быть ответ |
|--------|----------------------|
| "Как работать с {товар}?" | Статья в блоге / инструкция товара |
| "Какие ошибки избегать при работе?" | FAQ товара / видео |
| "Какие аксессуары нужны?" | Cross-sell на странице товара |
| "Как нанести правильно?" | Инструкция к товару |

### Тест секции

Перед написанием каждой H2:
> **"Пользователь после этой секции станет ближе к ПОКУПКЕ или к ИСПОЛЬЗОВАНИЮ?"**

- К покупке → оставляем
- К использованию → удаляем

### Примеры теста

| Секция | Тест | Решение |
|--------|------|---------|
| "Как выбрать по типу лака" | К покупке | ✅ Оставить |
| "Паста + круг: комбинации" | К использованию | ❌ Удалить |
| "Чего избегать при выборе" | К покупке | ✅ Оставить |
| "Ошибки при работе" | К использованию | ❌ Удалить |
| "Выбор по марке авто" | К покупке | ✅ Оставить |
| "Как правильно наносить" | К использованию | ❌ Удалить |

## Коммерческий vs Информационный

| ✅ Коммерческий | ❌ Информационный |
|-----------------|-------------------|
| "Если нужен X → выбирай Y" | "Как работает X" |
| Таблица сравнения типов | Пошаговая инструкция 5+ шагов |
| "На что смотреть на этикетке" | "История создания" |
| Сценарии: новичок/профи/бюджет | Теория и принципы |
| FAQ про выбор | FAQ про процессы |

**Правило:** Контент категории = помощь в выборе товара, НЕ образовательная статья.

## Паттерн "Портрет покупателя"

**Вместо "характеристики товара" → "кто я и что мне взять"**

| ❌ Таблица характеристик | ✅ Таблица сценариев |
|--------------------------|---------------------|
| Тип / Материал / Размер | Ваш уровень → Что выбрать → Почему |
| Брусок / Глина / 100г | Новичок → Автоскраб-варежка → не страшно уронить |

### Примеры портретов

- Новичок vs Профи/Детейлер
- Мягкий лак (японцы) vs Твёрдый лак (немцы)
- Бюджет vs Премиум
- DIY vs Сервис
- Ежедневный уход vs Глубокая обработка

### Пример таблицы

```markdown
## Как выбрать: быстрый подбор

| Ваш уровень / Задача | Что выбрать | Почему |
|---------------------|-------------|--------|
| **Новичок (первый раз)** | Автоскраб-варежка | Не страшно уронить, можно помыть |
| **Детейлер / Профи** | Глина (брусок) | Максимум эффективности перед полировкой |
| **Локальная очистка** | Глина (кусочек) | Точечное удаление битума/смолы |
```

## Паттерн "Если X → Y"

**Минимум 3 раза в тексте!**

```markdown
**Если нужен быстрый результат** → жидкое стекло: нанесение за час.

**Если важна долговечность** → нанокерамика: держится до года.

**Если бюджет ограничен** → гибридные составы: компромисс цена/качество.
```

### Типичные сценарии

| Категория | Сценарии "Если X → Y" |
|-----------|----------------------|
| Защита | быстрый результат / долговечность / бюджет |
| Шины | заводской вид / максимум блеска / частые мойки |
| Моющие | зимняя грязь / авто с керамикой / мягкий лак |
| Инструменты | новичок / профи / локальная задача |

## Структура таблиц

### Правило: Max 2-3 таблицы на текст

| ❌ 6 таблиц | ✅ 2-3 таблицы |
|-------------|----------------|
| Справочник, стена данных | Гид: быстро находит ответ |
| Читатель теряется | «Воздух» между блоками |

**Лишние таблицы → объединить или заменить на «Если X → Y»**

### Правило 3 колонок

| ❌ 4+ колонки (перегруз) | ✅ 3 колонки (buyer guide) |
|-------------------------|---------------------------|
| Тип / Материал / Размер / Цена | Задача → Тип → Почему |

### Правильная структура

```markdown
| Проблема/Задача | Что использовать | Почему именно это |
|-----------------|------------------|-------------------|
| Чёрные точки, смола | Антибитум | Растворители работают по нефтепродуктам |
| Рыжие точки, "сыпь" | Iron Remover | Химически растворяет металл |
```

### Выбор по типу лака (для полировки)

```markdown
**Если «японец» (мягкий лак — Toyota, Honda)** — поролон Open Cell: деликатный, не пропилит.

**Если «немец» (твёрдый лак — BMW, Audi)** — микрофибра: пробивает керамический лак.
```

## Антипаттерны

### ❌ Секция "Типы X" как справочник

```markdown
❌ ПЛОХО:
## Типы чернителей

### Водные
{описание водных...}

### Силиконовые
{описание силиконовых...}

### Гелевые
{описание гелевых...}
```

**Почему плохо:** Это справочник (каннибализация L3), не buyer guide.

### ✅ Вместо этого — таблица сценариев

```markdown
✅ ХОРОШО:
## Как выбрать

| Задача | Тип | Почему |
|--------|-----|--------|
| Частые мойки | Водный | Легко обновлять |
| Редкие выезды | Силикон | Дольше держится |
```

### ❌ Секция "Как применять" с 5+ шагами

```markdown
❌ ПЛОХО:
## Как наносить чернитель

1. Очистить шину
2. Высушить
3. Нанести на аппликатор
4. Втереть
5. Дать впитаться
6. Ждать перед поездкой
```

**Почему плохо:** How-to = информационный интент.

### ✅ Вместо этого — таблица "Что влияет на результат"

```markdown
✅ ХОРОШО (max 2-3 пункта):
## Что влияет на результат

| Фактор | Влияние |
|--------|---------|
| Чистота поверхности | Грязь → быстро отвалится |
| Количество | Много → разбрызгивание (sling) |
```

## Профтермины (E-E-A-T)

Брать профессиональные термины из `RESEARCH_DATA.md` (источник истины), НЕ из `entities` в `_clean.json`.

Упоминать 3-4 термина для маркеров экспертности.

**Формат:** RU-термин первым, англ. в скобках при первом упоминании.

> **Примечание:** Поле `entities` в `_clean.json` сгенерировано автоматически и не несёт SEO-пользы.

## Пример: Полировочные пасты

Реальный кейс как текст съехал в инструкцию и был исправлен.

### ❌ БЫЛО (съехало в инструкцию)

```markdown
## Полироль + круг: правильные комбинации   ← выбор КРУГА, не пасты

| Задача | Состав | Круг | Результат |
|--------|--------|------|-----------|

## Ошибки, которые портят результат   ← HOW-TO, не выбор

**Если паста скатывается в катышки** → перегрели лак...
**Если новый пад царапает** → сухой поролон...
```

**Почему плохо:**
- "Полироль + круг" — это про работу, не про выбор пасты
- "Ошибки при работе" — инструкция после покупки

### ✅ СТАЛО (buyer guide)

```markdown
## Выбор по типу лака   ← помогает выбрать ПАСТУ

| Тип лака | Марки авто | Какую пасту брать |
|----------|------------|-------------------|
| Мягкий | Toyota, Honda | DAT-технология |
| Твёрдый | BMW, Audi | SMAT-технология |

## Чего избегать при выборе   ← помогает НЕ купить не то

**Если планируется керамика** → только Silicone-free
**Если паста с филлерами** → временный эффект, не для коррекции
```

**Почему хорошо:**
- "Выбор по типу лака" — помогает выбрать правильную пасту
- "Чего избегать" — предупреждает об ошибках ВЫБОРА, не работы

## Пример: Химчистка салона (переспам + 4 таблицы)

Реальный кейс: термины копировались в каждую ячейку → переспам.

### ❌ БЫЛО (4 таблицы, переспам "очиститель")

```markdown
## Как выбрать: сценарии
| Задача | Что брать |
|--------|-----------|
| Ткань | Очиститель ткани |        ← "очиститель" x1
| Потолок | Очиститель потолка |    ← "очиститель" x2
| Сиденья | Очиститель сидений |    ← "очиститель" x3

## На что смотреть при выборе        ← таблица #2
| Параметр | Значение |
...

## Выбор по форме выпуска            ← таблица #3 (дубль смысла)
| Форма | Особенности |
| Пенный очиститель | ...           ← "очиститель" x4

## APC или специализированный        ← таблица #4
| Критерий | Очиститель ткани |      ← "очиститель" x5
```

**Результат:** "очиститель" 12+ раз, density 4.7%, тошнота 4.24 — БЛОКЕР.

### ✅ СТАЛО (2 таблицы, синонимы)

```markdown
## Как выбрать: сценарии
| Задача | Что брать |
|--------|-----------|
| Ткань | Универсальный состав |    ← синоним
| Потолок | Пенный продукт |        ← синоним
| Сиденья | Химия для интерьера |   ← синоним

## Как определиться с формой          ← ТЕКСТ вместо таблицы
**Если потолок** → пена (не жидкость)
**Если точечно** → спрей RTU
**Если регулярно** → концентрат

## APC или специализированный
| Критерий | APC | Химия для ткани |  ← синоним
```

**Почему лучше:**
- 2 таблицы вместо 4 (гид, не справочник)
- "Очиститель" заменён на: состав, продукт, химия, спрей
- Добавлены паттерны "Если → То" вместо третьей таблицы
