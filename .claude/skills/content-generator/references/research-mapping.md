## Блок 1 → Intro (30-60 слов)

**Что брать из research:**
- Определение продукта
- Какую проблему решает
- Главное правило применения

**Формула:** {Что это} + {Зачем нужно} + {Главное условие}

**Пример трансформации:**

```
Research Блок 1:
- Определение: составы на основе силикона/воды для защиты шин
- Проблема: шины выгорают из-за UV и вымывания антиозонантов
- Правило: наносить только на сухую чистую резину

↓

Intro:
"Чернитель резины — финишный состав для защиты боковины шин
от UV-старения и придания глубокого чёрного цвета. Шины
выгорают из-за вымывания антиозонантов. Главное правило —
наносить только на чистую сухую резину."
```

## Блок 2 → Таблица сравнения

**Что брать:** Оси классификации с плюсами/минусами

**Правило:** Одна таблица на каждую ось (не смешивать!)

```
Research Блок 2:
Ось 1 — Носитель: водная (7-14 дней) vs растворитель (3-4 недели)
Ось 2 — Эффект: матовый (OEM) vs глянцевый (wet look)

↓

### По типу основы

| Тип | Стойкость | Плюсы | Минусы |
|-----|-----------|-------|--------|
| Водная | 7-14 дней | Меньше пыли, безопаснее | Короче держится |
| На растворителе | 3-4 недели | Дольше держится | Может сушить резину |

### По эффекту

| Эффект | Вид | Когда выбирать |
|--------|-----|----------------|
| Матовый | OEM-вид | Заводской стиль |
| Глянцевый | Wet look | Шоу-кары |
```

## Блок 4 → How-to шаги

**Что брать:** Пошаговая инструкция с временем

**Формат:** Нумерованный список с жирным действием

```
Research Блок 4:
1. Очистить шину (tire cleaner + щётка)
2. Высушить полностью
3. Нанести тонким слоем на аппликатор
4. Втереть круговыми движениями
5. Дать впитаться 5-10 мин
6. Ждать 20-30 мин до поездки

↓

## Как применять

1. **Очистить шину** — tire cleaner + щётка, смыть
2. **Высушить** — полностью, можно феном
3. **Нанести на аппликатор** — тонким слоем (не на шину напрямую!)
4. **Втереть** — круговыми движениями по боковине
5. **Дать впитаться** — 5-10 минут
6. **Выждать перед поездкой** — минимум 20-30 минут
```

## Блок 5 → Частые ошибки

**Что брать:** Ошибка → последствие → решение

**Формат:**
```markdown
**Частые ошибки:**
- **{Ошибка}** — {последствие}. {Как правильно}.
```

**Пример:**
```
Research Блок 5:
| Ошибка | Последствие | Решение |
| Грязная шина | Отваливается через 1-2 дня | Очистить + высушить |
| Слишком много | Sling на кузов | Тонкие слои |
| На протектор | Скользко | Только боковина |

↓

**Частые ошибки:**
- **Нанесение на грязную шину** — отвалится через 1-2 дня. Сначала очистить и высушить.
- **Слишком много средства** — разбрызгается на кузов (sling). Наносить тонкими слоями.
- **Попадание на протектор** — снижает сцепление. Наносить только на боковину.
```

↓

## FAQ

### Матовый или глянцевый — что выбрать?

Матовый даёт заводской OEM-вид, глянцевый — эффект "мокрых шин".
Для повседневной езды лучше матовый, для шоу-каров — глянцевый.

### Сколько держится чернитель?

Зависит от основы: водный — 7-14 дней, на растворителе — 3-4 недели.
После каждой мойки рекомендуется обновлять.
