## Research Matrix by Category Type

### Автохимия (химические составы)

**Категории:** aktivnaya-pena, antibitum, antimoshka, ochistiteli-diskov, ochistiteli-shin, ochistiteli-stekol, obezzhirivateli, dlya-ruchnoy-moyki

**Обязательные блоки (8):** + pH/агрессивность, совместимость с покрытиями, температурные ограничения

| Параметр | Что искать | Пример |
|----------|-----------|--------|
| pH | Нейтральный (7), щелочной (10-14), кислотный (1-4) | pH 12 для сильных загрязнений |
| Совместимость | PPF, керамика, воск, хром, пластик | Щелочь снимает воск |
| Температура | Мин/макс для нанесения | Не на горячий кузов |
| Время выдержки | Мин-макс | 2-5 минут |
| Что снимает | Воск, силант, керамика | Щелочная пена смывает воск |

### Защитные покрытия

**Категории:** vosk-tverdyy, vosk-zhidkiy, nanokeramika, silanty, kvick-detaylery

**Обязательные блоки (8):** + стойкость, подготовка поверхности, условия нанесения

| Параметр | Что искать | Пример |
|----------|-----------|--------|
| Стойкость | Недели/месяцы, от чего зависит | 3-6 месяцев |
| Подготовка | Обезжиривание, полировка | Глина + IPA |
| Слои | Количество, межслойная сушка | 2 слоя, 1 час между |
| Температура | При нанесении | +15...+25°C |
| Влажность | При нанесении | <70% |

### Инструменты

**Категории:** shchetki-dlya-moyki, raspyliteli-penniki, gubki-varezhki, vedra

**Обязательные блоки (8):** + материалы, размеры/резьба, ресурс

| Параметр | Что искать | Пример |
|----------|-----------|--------|
| Материал | Нейлон, микрофибра, ПЭТ | Мягкий нейлон для кузова |
| Размер | Диаметр, длина, объём | 20 мм, 200 мл |
| Резьба | Тип соединения | 1/4" |
| Совместимость | АВД, помпа, компрессор | Для Karcher K-series |
| Ресурс | Количество моек/стирок | 50+ стирок |

### Расходники (микрофибры, аппликаторы)

**Категории:** mikrofibra, tryapki, gubki-dlya-naneseniya

**Обязательные блоки (8):** + плетение, GSM, назначение

| Параметр | Что искать | Пример |
|----------|-----------|--------|
| GSM | Плотность ткани | 300-400 GSM для сушки |
| Плетение | Вафельное, ворсовое, сатин | Вафельное для стёкол |
| Размер | Стандартные размеры | 40x40 см |
| Уход | Стирка, сушка | 60°C без кондиционера |
| Риски | Разводы, ворс, царапины | Новую перед использованием постирать |

### Полировка

**Категории:** polirovalnye-pasty, polirovalnye-krugi, polirovalnye-mashiny

**Обязательные блоки (8):** + абразивность, совместимость круг-паста

## Example: Активная пена

```markdown
## Research: Активная пена (aktivnaya-pena)

### 1. Преимущества (зачем покупать)
- Размягчает грязь без контакта — меньше царапин
- Экономит время — 70% грязи уходит без губки
- Инкапсулирует песок — безопасно для лака

### 2. Виды/типы
- **Щелочные (pH 10-14):** для сильных загрязнений, зимней грязи
- **Нейтральные (pH 7-9):** для авто с защитными покрытиями
- **Кислотные (pH 4-6):** для минеральных отложений (редко)

### 3. Критерии выбора
- Для грязного авто → щелочная
- Для авто с керамикой/воском → нейтральная
- Для мойки самообслуживания → концентрат

### 4. Применение
1. Нанести на СУХОЙ кузов снизу вверх
2. Выдержать 2-5 минут (не давать высохнуть)
3. Смыть АВД сверху вниз

### 5. Способы нанесения
- Пенокомплект (насадка на АВД)
- Пеногенератор (от компрессора)
- Помповый распылитель (для домашней мойки)

### 6. Предосторожности
- НЕЛЬЗЯ: на горячий кузов, давать высохнуть
- НЕЛЬЗЯ: щелочную на воск/керамику — смоет
- НЕЛЬЗЯ: дешёвую щелочь на хром — помутнеет

### 7. Расход
- 50-150 мл концентрата на авто
- Разведение: 1:3 — 1:10 (см. этикетку)

### 8. Стойкость
- Не применимо (смывное средство)

---
**Источник:** Web Search
**Дата обновления:** 2025-12-18
```
