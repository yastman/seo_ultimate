# SEO Research for Ultimate.net.ua

Сбор фактуры и ограничений, чтобы текст категории был "про товар", а не выдумка/вода.

---

## Промпт-шаблон для Web Search

```
{Название категории} для автомобиля — составь матрицу данных для интернет-магазина:

| Поле | Ответ |
|------|-------|
| Что это | определение в 1 предложение |
| Для чего | какие задачи/проблемы решает |
| Классификация | виды по составу/назначению (с pH если химия) |
| Расход | сколько нужно на 1 авто/операцию |
| Инструкция | 3-5 шагов применения |
| Совместимость | с какими материалами/покрытиями работает, что снимает |
| Ошибки | типичные ошибки, чего нельзя делать |
| Критерии выбора | на что смотреть при покупке |

Дополнительно:
- Лайфхаки из детейлинг-сообщества
- Меры предосторожности (PPE, условия)

Формат: таблица + буллеты. Кратко, факты, без брендов и цен.
```

---

## 8 обязательных блоков

| # | Блок | Что собрать | Поисковые запросы |
|---|------|-------------|-------------------|
| 1 | **Что это** | Определение, какую проблему решает | `"{продукт} зачем нужен"` |
| 2 | **Виды/типы** | По составу/эффекту/формату | `"{продукт} виды"`, `"разница {тип1} {тип2}"` |
| 3 | **Критерии выбора** | На что смотреть, сценарии | `"как выбрать {продукт}"` |
| 4 | **Применение** | Шаги, время выдержки, смыв | `"как использовать {продукт}"` |
| 5 | **Способы нанесения** | Чем наносить, оборудование | `"чем наносить {продукт}"` |
| 6 | **Ошибки** | Что нельзя, на чём опасно | `"{продукт} ошибки"` |
| 7 | **Расход** | На одну обработку, разведение | `"{продукт} расход"` |
| 8 | **Совместимость** | Покрытия, материалы, что снимает | `"{продукт} совместим с"` |

## Матрица по типу категории

### Автохимия (составы/средства)

- pH/агрессивность + на что влияет
- Что снимает/ослабляет (воск, керамика, силант)
- Температурные ограничения
- Совместимость с покрытиями (PPF, тонировка, матовый лак)
- Типовые "почему не работает"

### Инструменты/аксессуары

- Совместимость: размер, резьба, тип крепления
- Материалы и свойства (микрофибра/нейлон/щетина)
- Ресурс и уход: как мыть, когда менять
- Риски: где может поцарапать

### Расходники (микрофибры, губки)

- Тип плетения/назначение
- Риск разводов/царапин
- Сколько нужно на задачу

## Формат RESEARCH_DATA.md

```markdown
# Research: {Category Name} ({slug})

## 1. Базовая матрица

| Данные | Результат |
|--------|-----------|
| **Что это** | {определение} |
| **Для чего** | {назначение} |
| **Классификация** | {виды} |
| **Расход** | {X мл на авто} |
| **Инструкция** | 1. ... 2. ... 3. ... |
| **Совместимость** | {материалы, что снимает} |
| **Ошибки** | {типичные ошибки} |
| **Критерии выбора** | {на что смотреть} |

## 2. Углубление (E-E-A-T)

- **Расход:** {детали}
- **Оборудование:** {что нужно}
- **Safety:** {ограничения, PPE}

## 3. Лайфхаки

- **Факт:** {интересный факт}
- **Лайфхак:** {совет от профи}

## 4. Меры предосторожности

- **Никогда** не {ошибка}
- **PPE:** {защита}

---
**Источник:** Web Search
**Дата:** {YYYY-MM-DD}
```

## Правила качества

1. **Факты > маркетинг** — только проверяемые данные
2. **Диапазоны** — если параметры отличаются по товарам
3. **"На этикетке"** — для точных пропорций
4. **Без брендов/цен** — только механика и выбор

---

## Workflow

1. Прочитать `categories/{slug}/data/{slug}_clean.json` — ключевые слова
2. Прочитать `categories/{slug}/meta/{slug}_meta.json` — типы, объёмы
3. Использовать промпт-шаблон для WebSearch
4. Заполнить RESEARCH_DATA.md по формату выше
5. Проверить: все 8 блоков заполнены
