## Common Mistakes

| Wrong | Right | Why |
|-------|-------|-----|
| `0.5–5л` | `0.5, 1, 5л` | Диапазон включает несуществующие объёмы |
| `безфосфатная` | `кислотная` | Реальный тип товара |
| `густая пена` | `щелочная, нейтральная` | Маркетинг vs факты |
| `Купить X` в H1 | `X для авто` | Commercial только в Title |
| Title без "Купить" | Title с "Купить" | Commercial модификатор обязателен |
| Эмодзи в Description | БЕЗ эмодзи | Правило 2025 |
| омыватели в ochistiteli-stekol | Отдельная категория | Разные товары |
| антидождь = очиститель | антидождь = защита | Разное назначение |
