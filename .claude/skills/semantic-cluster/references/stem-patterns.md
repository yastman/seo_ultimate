# Типові патерни дублів (Stem Patterns)

## Українська мова

### Суфіксальні варіанти

| Патерн | Приклад | Canonical |
|--------|---------|-----------|
| -а/-ка (diminutive) | машина → машинка | машина |
| -ння/-ка (action/tool) | полірування → поліровка | полірування |
| -ння/-ття | миття → мийка | миття |
| -ач/-ник | очищувач → очисник | очищувач |

### Число

| Патерн | Приклад | Canonical |
|--------|---------|-----------|
| singular/plural | засіб → засоби | засіб (якщо volume =) |
| collective | шампунь → шампуні | шампунь |

### Комерційні модифікатори (завжди дублі)

| Модифікатор | Приклад |
|-------------|---------|
| купити X | купити полірувальну машину |
| X ціна | полірувальна машина ціна |
| X Україна | полірувальна машина Україна |
| X недорого | полірувальна машина недорого |
| X інтернет-магазин | полірувальна машина інтернет-магазин |

**Canonical** — базова форма без модифікатора.

| -ние/-ка (action/tool) | полирование → полировка | полировка |

### Число

| Патерн | Приклад | Canonical |
|--------|---------|-----------|
| singular/plural | средство → средства | средство |

### Комерційні модифікатори (завжди дублі)

| Модифікатор | Приклад |

## Автохімія — специфічні патерни

### Детейлінг

| Група | Варіанти | Canonical |
|-------|----------|-----------|
| Полірування | машина, машинка, полірувальник | машина |
| Миття | мийка, миття, мийний засіб | миття |
| Скло | скло, стекло (RU) | скло (UK) |
| Гума | гума, резина (RU) | гума (UK) |

### Засоби

| Група | Варіанти | Canonical |
|-------|----------|-----------|
| Воск | воск, віск, вакса | воск |
| Шампунь | шампунь, автошампунь, мийний засіб | шампунь |
| Очищувач | очищувач, очисник, засіб для очищення | очищувач |

## Як визначити canonical

1. **Volume** — вища частотність = canonical
2. **Довжина** — коротша форма = canonical (при рівному volume)
3. **Нейтральність** — формальна форма > розмовна (машина > машинка)
4. **Українськість** — UK форма > RU форма для UK контенту (скло > стекло)
