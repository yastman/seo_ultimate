# Adaptation Rules: RU → UK (Automotive Detailing)

Детальные правила семантической адаптации контента для украинской версии.

---

## Core Principle

**Адаптация ≠ Перевод**

Задача — передать смысл и поисковый интент, а не слова.

| Аспект | Перевод | Адаптация |
|--------|---------|-----------|
| Фокус | Слова | Смысл + SEO |
| Ключи | Переводим | Берём из UK ядра |
| Структура | Может меняться | Сохраняем |
| Числа | Переводим | Оставляем (если релевантны) |

## Латиница (без перевода)

Оставляем как есть:

- **Химия:** pH, PPF, APC, HF, VOC, SiO2, TiO2
- **Методы:** pre-wash, wet look, OEM
- **Эффекты:** sling, coating, hydrophobic

**Правило:** латиница допустима ТОЛЬКО в body text как пояснение в скобках.

```markdown
# Правильно
гідрофобність (hydrophobic effect)

# Неправильно
## Hydrophobic захист
```

## Checklist адаптации

### BLOCKER (обязательно)

- [ ] резина → гума
- [ ] H1 из UK meta (не перевод RU H1)
- [ ] Primary keyword в intro
- [ ] Структура = RU (без изменений)

### WARNING (важно)

- [ ] Мінімум 1 H2 з secondary keyword
- [ ] Латиниця тільки в дужках
- [ ] Коммерційні — тільки в meta

### INFO (желательно)

- [ ] FAQ адаптований (не дослівний переклад)
- [ ] Сценарії "Якщо X → Y"
- [ ] Профтерміни з entities
