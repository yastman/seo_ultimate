# Archive

**[← Назад в корень](../README.md)**

Здесь хранятся устаревшие документы, планы и отчеты.
Файлы здесь не используются в активном пайплайне, но сохранены для истории.

## Содержимое

* `docs/`: Старые версии документации (Content Guides v1-v19).
* `reports/`: Старые отчеты (аудиты, coverage reports).
* `scripts_legacy/`: Устаревшие скрипты, которые были заменены новыми версиями в `/scripts`.
* `legacy_data/`: Старые данные.

## Политика архивации

1. Если файл не используется более 3 месяцев — переместить сюда.
2. Если скрипт заменен новым — старый переместить в `scripts_legacy`.
3. Отработанные планы (Task Plans) перемещать сюда после завершения.
