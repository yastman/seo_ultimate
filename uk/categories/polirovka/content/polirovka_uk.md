# Полірування авто

Полірування авто — процес корекції лакофарбового покриття для видалення подряпин, голограм та відновлення заводського блиску. Засоби для полірування авто включають пасти, круги, машинки та допоміжну хімію.

## Як обрати засоби для полірування авто

| Категорія | Призначення | Для кого |
|-----------|-------------|----------|
| Полірувальні пасти | Абразивна корекція лаку | Видалення дефектів |
| Полірувальні круги | Носій для пасти | Підсилення/зменшення зрізу |
| Полірувальна машинка | Механізація процесу | Швидкість і якість |

**Сценарії вибору:**
- **Якщо перше полірування автомобіля** → стартовий набір: машинка + 2–3 круги + 2 пасти
- **Якщо досвідчений детейлер** → розширення арсеналу під конкретні завдання
- **Якщо тільки підтримка** → фінішна паста + м'який круг достатні

### Що потрібно для полірування кузова

| Елемент | Мінімум | Оптимум |
|---------|---------|---------|
| Пасти | 1 (one-step) | 2–3 (система) |
| Круги | 2 | 4–6 різної жорсткості |
| Машинка | Ексцентрикова | + роторна для важких випадків |

## Особливості полірування автомобіля

Полірування машини виконується поетапно: спочатку груба паста видаляє основні дефекти, потім середня прибирає сліди від першого етапу, фінішна надає глибокий блиск. Кожен етап — чистий круг.

**Якщо один круг на все** → промивайте його щіткою кожні 2–3 панелі для видалення відпрацьованої пасти.

## Вибір полірувальної машинки

| Тип | Особливості | Для кого |
|-----|-------------|----------|
| Ексцентрикова (DA) | Безпечна, менше ризику пропалу | Початківці та універсальне використання |
| Роторна | Максимальний зріз, потребує досвіду | Важкі дефекти, досвідчені |
| Акумуляторна | Мобільність без дроту | Професіонали, виїзна робота |

## FAQ

### З чого почати полірування авто новачку?

Почніть з ексцентрикової машинки 15–21 мм ходу. Оберіть one-step пасту середньої абразивності та поролоновий круг середньої жорсткості. Потренуйтесь на непомітній ділянці.

### Чим полірувальні круги відрізняються між собою?

Круги відрізняються матеріалом (поролон, мікрофібра, вовна) і жорсткістю. Жорсткіші — для абразивних паст, м'які — для фінішу. Колір зазвичай позначає жорсткість, але у кожного виробника своя система.

### Як часто можна полірувати машину?

**Якщо все зроблено правильно** → раз на 1–2 роки для корекції. Кожне полірування знімає мікрошар лаку, тому зловживати не варто.

---

**Підсумок:**
- Якщо новачок: ексцентрикова машинка + one-step система
- Якщо важкі дефекти: роторна машинка + триступенева система паст
