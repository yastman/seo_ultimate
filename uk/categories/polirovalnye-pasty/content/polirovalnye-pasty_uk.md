# Полірувальна паста

Полірувальна паста для авто — абразивний склад для видалення подряпин, голограм та окислень з лакофарбового покриття. Правильно підібрана паста для полірування відновлює глибину кольору без зайвого зняття лаку.

## Як обрати полірувальну пасту для авто

| Тип | Абразивність | Для кого |
|-----|--------------|----------|
| Важка (heavy cut) | Висока | Глибокі подряпини, риски від P1500 |
| Середня (medium cut) | Середня | Голограми, сліди від щіток |
| Фінішна (finish) | Мінімальна | Надання блиску, видалення розводів |

**Сценарії вибору:**
- **Якщо на лаку глибокі риски** → полірувальна паста від подряпин (heavy cut)
- **Якщо тільки голограми після полірування** → середня або фінішна паста
- **Якщо м'який японський лак** → починайте з меншої абразивності

### На що дивитися на етикетці

| Маркер | Значення |
|--------|----------|
| «Cut» + цифра | Ступінь абразивності (1–10) |
| «Compound» | Абразивна паста (не фінішна) |
| «Polish» | Зазвичай менш абразивна |
| «One-step» | Універсальна для одного проходу |

## Особливості роботи з полірувальною пастою для автомобіля

Паста для полірування кузова працює у зв'язці з кругом — жорсткий круг підсилює абразивність, м'який — зменшує. Температура поверхні критична: перегрів вище 60°C може призвести до пропалу лаку. Працюйте секціями 40×40 см.

**Якщо паста підсихає на поверхні** → додайте пару крапель води або спеціального лубриканту.

## Що впливає на результат полірування

| Фактор | Вплив |
|--------|-------|
| Твердість лаку | М'які (японці) потребують меншої абразивності |
| Тип круга | Поролон/мікрофібра/вовна дають різний зріз |
| Оберти машинки | Нижчі = безпечніше, вищі = агресивніше |

## FAQ

### Чим полірувальна паста відрізняється від воску?

Паста для полірування — абразивний склад для корекції дефектів, вона знімає мікрошар лаку. Віск — захисне покриття без абразиву. Спочатку полірування (якщо потрібно), потім захист воском.

### Скільки пасти потрібно на одне авто?

**Якщо середній седан** → 50–100 грам на повне полірування одним етапом. Якщо триступеневе полірування — відповідно більше.

### Чи можна полірувати вручну?

Так, але ефективність значно нижча. Набір паст для полірування авто вручну підходить для локальних дефектів або як тимчасовий захід.

- Якщо тільки голограми: фінішна полірувальна паста достатня
