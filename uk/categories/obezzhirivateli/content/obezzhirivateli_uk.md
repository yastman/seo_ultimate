# Знежирювач для авто

Знежирювач для авто видаляє масляні плями, сліди пальців та залишки полірувальних паст з поверхні кузова. Якщо ви плануєте полірувати авто або наносити захисне покриття — знежирення є обов'язковим етапом підготовки.

## Як обрати знежирювач для авто

| Тип засобу | Особливості | Застосування |
|------------|-------------|--------------|
| На спиртовій основі (IPA) | Швидко випаровується, безпечний | Перед воском, поліруванням |
| На водній основі | М'яка дія, без запаху | Регулярне очищення |
| Силіконовий ремувер | Видаляє силікони та віск | Перед нанесенням кераміки |
| Універсальний | Комбінована формула | Загальна підготовка |

**Сценарії вибору:**

- **Якщо готуєтесь до полірування** → IPA-based знежирювач для повного видалення олій
- **Якщо наноситимете кераміку** → силіконовий ремувер обов'язковий
- **Якщо просто очищаєте кузов** → водна основа для делікатної роботи
- **Якщо працюєте в закритому приміщенні** → без запаху, на водній основі

## Знежирювач для кузова — на що звернути увагу

| Параметр | Що означає | Рекомендація |
|----------|------------|--------------|
| Концентрація IPA | Сила знежирення | 15–30% для полірування, 50%+ для кераміки |
| Форма випуску | Готовий vs концентрат | Концентрат економніший |
| Безпечність для ЛКП | Агресивність | Якісні засоби не пошкоджують лак |
| Швидкість випаровування | Час роботи | Швидке — менше розводів |

## Знежирювач перед поліруванням — навіщо потрібен

Полірувальні пасти містять олії та силікони, що надають блиску після обробки. Якщо ви хочете оцінити реальний стан лаку або нанести захисне покриття — ці залишки потрібно видалити.

Знежирювач кузова перед поліруванням показує справжній результат роботи. Без нього блиск може бути оманливим — від олій у пасті, а не від корекції лаку.

## Особливості використання

| Етап | Рекомендація |
|------|--------------|
| Перед поліруванням | Видаляє старі захисні шари |
| Між проходами пасти | Показує реальний результат |
| Перед воском/керамікою | Забезпечує адгезію покриття |
| Після полірування | Фінальна перевірка чистоти |

## FAQ

### Чи можна використовувати спирт замість знежирювача?

Ізопропіловий спирт (IPA) часто є основою знежирювачів. Якщо ви розводите чистий IPA — 20–30% для полірування достатньо. Але готові засоби зручніші та містять добавки проти розводів.

### Як часто потрібно знежирювати кузов?

Перед кожним нанесенням захисного покриття та під час полірування між проходами пасти. Для звичайного миття знежирення не потрібне.

### Чи пошкодить знежирювач пластик або гуму?

Якісний автомобільний знежирювач безпечний для всіх поверхонь. Але якщо засіб потрапив на чорний пластик — краще протерти вологою ганчіркою, щоб уникнути білих плям.

### Концентрат чи готовий розчин обрати?

Якщо ви часто поліруєте авто — концентрат економніший. Для разового використання готовий розчин зручніший.

---

**Підсумок:**
- Якщо готуєтесь до полірування — IPA-знежирювач
- Якщо наносите кераміку — силіконовий ремувер
- Якщо потрібен універсальний варіант — комбінований засіб
