# Рідкий віск для авто

Рідкий віск для авто — захисне покриття у текучій формі, що легко наноситься та швидко розполіровується. На відміну від твердого, цей варіант зручніший у використанні, хоча поступається глибиною блиску. Правильно підібраний засіб забезпечує гідрофобний ефект та захист на 4–8 тижнів.

## Як обрати рідкий віск для авто

| Тип | Особливості | Для кого |
|-----|-------------|----------|
| Карнауба-формула | Натуральний, теплий блиск | Цінителі естетики |
| Спрей-віск | Миттєве нанесення | Швидкий догляд |
| Концентрат воску | Розводиться водою | Мийки, великі об'єми |

**Сценарії вибору:**
- **Якщо хочете карнаубу без клопоту з пастою** → засіб на основі карнауби
- **Якщо максимальна швидкість** → спрей-віск після миття
- **Якщо для безконтактного миття** → віск для безконтактного миття (третя фаза)

## Особливості застосування рідкого воску

Засіб наноситься на мокру або суху поверхню залежно від інструкції. Спрей-віск працює на мокрому кузові — нанесли, розподілили, змили або витерли.

**Якщо з'являються плями** → нанесли забагато або дали висохнути на сонці.

## Рідкий чи твердий віск

| Параметр | Рідкий віск | Твердий |
|----------|-------------|---------|
| Зручність | Вища | Нижча |
| Глибина блиску | Середня | Максимальна |
| Час нанесення | 15–20 хв | 40–60 хв |
| Стійкість | 4–8 тижнів | 6–12 тижнів |

## FAQ

### Чим спрей-віск відрізняється від класичного воску?

Спрей — готовий до застосування без розполірування. Класичний потребує нанесення та розполірування. **Якщо час обмежений** → спрей. **Якщо важлива стійкість** → класичний формат.

### Чи можна використовувати як восковий шампунь?

Деякі воски поєднуються з миттям (2-в-1). Концентрат воску для мийок — окрема категорія. Не плутайте: шампунь з воском миє + захищає, чистий віск — тільки захист.

### Скільки тримається захист?

При якісній карнауба-формулі → 4–6 тижнів. Спрей-віск — 1–2 тижні. Для довшого захисту розгляньте синтетичний тип.

- Для балансу: карнауба у текучій формі для блиску та простоти
