# Силант

Силант для авто — полімерний захисний склад на основі синтетичних смол. На відміну від воску, цей засіб створює більш стійке покриття і тримається 3–6 місяців. Правильно підібраний силант для кузова забезпечує гідрофоб та захист від UV без жирного блиску.

## Як обрати силант для авто

| Тип | Особливості | Для кого |
|-----|-------------|----------|
| Спрей-формат | Легке нанесення | Швидкий догляд, регулярна підтримка |
| Рідкий | Класичне нанесення | Універсальне застосування |
| Пастоподібний | Максимальна концентрація | Довготривалий захист |

**Сценарії вибору:**
- **Якщо перший досвід із захисними покриттями** → спрей-варіант для простого старту
- **Якщо потрібен баланс стійкості та зручності** → рідкий формат
- **Якщо важлива максимальна довговічність** → пастоподібний полімерний склад

### На що дивитися на етикетці

| Маркер | Значення |
|--------|----------|
| «Sealant» | Полімерний захист (не віск) |
| «6-month protection» | Заявлена стійкість |
| «UV protection» | Захист від вигорання |
| «Hydrophobic» | Водовідштовхування |

## Особливості застосування силанту для кузова

Засіб наноситься на чисту, знежирену поверхню. Деякі склади потребують полімеризації 12–24 години — уникайте вологи у цей період.

**Якщо покриття залишає розводи** → зменшіть кількість продукту або відполіруйте чистою мікрофіброю раніше.

## Що впливає на стійкість захисного покриття

| Фактор | Вплив |
|--------|-------|
| Підготовка поверхні | Знежирення = краща адгезія |
| Час полімеризації | Повне затвердіння = максимальна стійкість |
| Агресивне миття | Лужна піна знижує довговічність |

## FAQ

### Чим силант відрізняється від воску?

Це синтетичний герметик на основі полімерів. Він стійкіший за натуральний віск (3–6 місяців проти 4–8 тижнів), дає менш теплий блиск, але краще захищає від хімії та UV.

### Чи можна наносити на кераміку?

**Якщо кераміка вже на кузові** → спеціалізований засіб або бустер для кераміки. Звичайний полімерний склад може не закріпитися на SiO2 шарі.

### Скільки засобу потрібно на одне авто?

**Якщо спрей-формат** → 30–50 мл достатньо для седана. Рідкий або пастоподібний — ще менше через вищу концентрацію.

- Якщо простота важливіша: спрей-формат для швидкого захисту
- Якщо максимальна стійкість: рідкий або пастоподібний силант для кузова
