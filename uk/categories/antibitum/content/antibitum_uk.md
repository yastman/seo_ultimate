# Антибітум

Антибітум розчиняє дорожній бітум, деревну смолу та гудрон без пошкодження лакофарбового покриття. Очищувач битума також справляється зі слідами скотчу та наклейок. Після поїздки свіжоукладеним асфальтом помітили чорні бризки на порогах — не тріть сухою мікрофіброю, розмажете бітумні плями по кузову. Розм'якшіть засобом від битума, знімайте безворсовою серветкою.

## Як обрати антибітум для авто

| Тип                      | Швидкість | Основа    | Застосування                  | Обмеження                    |
| ------------------------ | --------- | --------- | ----------------------------- | ---------------------------- |
| Сольвентний (нафтовий)   | Швидко    | Гас       | Металевий кузов               | Небезпечний для пластику, фар |
| Цитрусовий (терпеновий)  | Повільно  | D-лімонен | Метал, пластик, фари, гума    | Універсальний, безпечний     |

**Сольвентні склади** працюють агресивно — розчинник справляється з бітумними плямами за хвилини. Мінус: залишає білясті розводи на пластикових бамперах та помутніння фар із полікарбонату.

**Цитрусові формули** діють делікатніше, пахнуть апельсином. Підходять для будь-яких поверхонь: метал, пластик, фари, захисні плівки. Засновані на терпенах із шкірки цитрусових — екологічні, біорозкладні.

**Сценарії покупки:**

- **Бітумні плями на фарах або пластиковому бампері** — цитрусовий спрей запобігає помутнінню полікарбонату
- **Пороги вкриті товстим шаром застарілого гудрону** — сольвентний видаляч битума справляється швидше
- **Кузов автомобіля захищений поліуретановою плівкою** — тільки цитрусові, нафтові розчинники викликають пожовтіння

## На що звертати увагу при виборі очищувача битума

| Маркер                          | Значення                          | Рекомендація             |
| ------------------------------- | --------------------------------- | ------------------------ |
| "Без нафтових розчинників"      | Терпенова основа (цитрус)         | Безпечний для пластику   |
| "Test Spot Required"            | Обов'язковий тест на прихованій зоні | Для плівок, старого ЛФП |
| "Потребує подальшого захисту"   | Знімає віск у місці контакту      | Знадобиться силант       |

**Тест на непомітній ділянці (Test Spot)** — детейлінг-стандарт перед обробкою кузова. Капніть засіб від бітумних плям на приховану зону (під дверний ущільнювач), перевірте реакцію лаку або плівки через кілька хвилин.

**Метод вологої мікрофібри** — професійний підхід для вертикальних панелей. Просочіть безворсову серветку антибітумом для авто, прикладіть до забруднення на дверях — склад залишиться на місці, подіє глибше розпилення.

**Важливо:** Антибітум використовуйте ДО глини — якщо пустити глину першою, вона розмаже жирний бітум по лаку та забʼється. Після обробки обов'язково змийте залишки шампунем: сольвенти залишають масляну плівку, яка притягує новий пил.

## Що впливає на результат видалення бітуну

| Фактор            | Вплив                                        |
| ----------------- | -------------------------------------------- |
| Вік плями         | Свіжий бітум знімається повністю, старий залишає жовтий ореол у структурі лаку |
| Температура       | Гаряча поверхня прискорює випаровування розчинника, ефективність падає |
| Захист на кузові  | Засіб знімає віск та силант у зоні контакту, потрібне відновлення |

**Відновлення захисного шару** — обов'язковий етап після видалення бітуну. Оброблена зона втрачає віск або силант, розчинник знімає не тільки забруднення, але й захисне покриття. Нанесіть швидкий захисний склад (спрей-герметик або детейлер), інакше лак окислиться.

**Літній сезон** — пік бітумних забруднень кузова автомобіля. Свіжоукладений асфальт стає липким при спеці, розбризкується колесами на пороги та арки. Тримайте видаляч битума у багажнику — свіжі плями від смоли знімаються швидше.

## Як видалити битум з кузова: типові помилки

Уникайте поширених помилок при роботі з антибітумом:

- **Нанесення на брудний кузов** — дрібні частинки піску під час витирання залишають подряпини на лаку
- **Робота на гарячому ЛФП** — розчинник випаровується швидше, ніж встигає розчинити забруднення
- **Використання дешевих розчинників** — агресивна хімія пошкоджує пластикові елементи та гумові ущільнювачі
- **Висихання засобу на поверхні** — залишає масляні плями, які важко змити

**Правильна послідовність:** спочатку вимийте автомобіль безконтактною мийкою, висушіть кузов, потім наносьте антибітум на окремі забруднення. Після видалення бітумних плям — повторне миття з шампунем для нейтралізації залишків розчинника.

## Чим відмити бітумні плями: порівняння методів

| Метод              | Ефективність | Безпека для ЛФП | Коли використовувати           |
| ------------------ | ------------ | --------------- | ------------------------------ |
| Антибітум-спрей    | Висока       | Безпечно        | Локальні плями на кузові       |
| Гас чистий         | Середня      | Ризиковано      | Не рекомендовано для авто      |
| Уайт-спіріт        | Середня      | Ризиковано      | Тільки для металевих деталей   |
| Цитрусовий засіб   | Висока       | Максимально безпечно | Пластик, фари, плівки       |

Професійний антибітум для авто містить емульгатори, які дозволяють легко змити залишки водою. Побутові розчинники (гас, уайт-спіріт) не мають цієї властивості — після них залишається жирна плівка, яка потребує додаткового очищення.

## FAQ

### Чи видалить антибітум деревну смолу?

Розм'якшує, але повільніше ніж бітум — формула заточена під нафтопродукти. Для старих закам'янілих бруньок на кузові використовуйте ізопропіловий спирт (IPA) або спеціалізовані видалячі смоли з ензимами.

### Чому після обробки залишилась жовта пляма?

Бітумні масла проникли в структуру лакофарбового покриття — засіб прибрав верхній шар, але пігмент залишився всередині. Прибирається абразивним поліруванням з полірувальною машинкою. Видаляйте бітум одразу після поїздки, не чекайте поки в'їсться в лак.

### Чи безпечний антибітум для лаку автомобіля?

Так, професійні засоби від битума безпечні для заводського лакофарбового покриття. Головне — не допускати висихання на поверхні та змивати залишки шампунем. Для старого або пошкодженого лаку рекомендовано попередній тест на прихованій ділянці.

### Чи можна застосовувати на захисних плівках?

Тільки цитрусові формули безпечні для поліуретанових плівок (PPF). Сольвентні засоби від битума провокують пожовтіння або відшарування клейового шару. Тестуйте на краю плівки перед повною обробкою кузова.

### Який обʼєм знадобиться?

Орієнтуйтесь на площу забруднення та частоту поїздок. Для разової обробки достатньо компактного флакона 0.5л, для літнього сезону або парку з кількох машин беріть каністри більшого обʼєму 5л — вигідніше за ціною за мілілітр, зручніше для заправки розпилювачів.

**Підсумок покупки:**

- **Працюєте біля пластику або фар:** Цитрусовий видаляч битума — універсальний, безпечний варіант
- **Потрібна швидкість на металевих панелях:** Сольвентний очищувач битума — розчиняє гудрон за хвилини
- **Кузов оклеєний захисною плівкою:** Тільки терпенові склади без нафтових розчинників, уникнете пошкодження PPF
