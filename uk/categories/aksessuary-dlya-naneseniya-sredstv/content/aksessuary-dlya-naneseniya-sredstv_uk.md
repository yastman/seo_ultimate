# Губка для полірування автомобіля

Рівномірне нанесення воску, силанту чи кераміки неможливе без правильного аплікатора. Губка для полірування авто забезпечує контрольований шар засобу та економить витрату — надлишок просто не потрапляє на поверхню.

## Як обрати губку для полірування

| Тип | Матеріал | Для чого |
|-----|----------|----------|
| Поролоновий | Дрібнопористий поролон | Тверді воски, пасти |
| Велюровий | Мікрофібра велюр | Рідкі силанти, квік-детейлери |
| Комбінований | Поролон + мікрофібра | Універсальне застосування |
| Ганчірки | Безворсова тканина | Фінішне полірування |

### На що дивитися при виборі

| Характеристика | Що означає | Рекомендація |
|----------------|------------|--------------|
| Розмір | Площа нанесення | 10-12 см для кузова, 5-7 см для деталей |
| Жорсткість | Контроль тиску | М'який для лаків, щільніший для пластику |
| Шви | Ризик подряпин | Без швів або зі швами всередину |

## Сценарії вибору губки для полірування авто

**Якщо наносите твердий віск** — класичний поролоновий спонж. Круговими рухами розподіляєте пасту тонким шаром.

**Якщо працюєте з рідким силантом** — велюровий або мікрофібровий аплікатор. Вбирає надлишок, запобігає потьокам.

**Якщо обробляєте шини чи пластик** — окремий спонж (не той, що для кузова). Дрібнопористий поролон або спеціальна подушечка для дресингів.

### Поради щодо догляду

Аплікатори потрібно мити після кожного використання. Залишки воску та силанту твердіють і перетворюють м'який спонж на абразив. Сушити природним шляхом, не викручувати.

## FAQ

### Скільки аплікаторів потрібно?

Мінімум три: для кузова, для шин/пластику, для скла. Якщо використовуєте різні типи захисту (віск, силант, кераміка) — краще окремий аплікатор під кожен.

### Чим відрізняється від полірувальної серветки?

Аплікатор — для нанесення засобу. Серветка — для розполіровки та зняття надлишків. Це різні етапи процесу, тому використовуються разом.

### Як зрозуміти, що аплікатор зношений?

Поверхня стає жорсткою, нерівномірною, губка втрачає форму. Такий інструмент може залишати розводи та подряпини. Краще замінити на новий.

---

**Підсумок:** Якщо працюєте з твердими восками — поролонова губка для полірування. Для рідких силантів та кераміки — велюрові аплікатори. Тримайте окремий набір для кузова та для шин.
