# Шерстяний круг для полірування

Шерстяний круг для полірування використовується на першому етапі корекції лаку — коли потрібно видалити глибокі подряпини та оксидацію. Якщо вам потрібен агресивний зріз дефектів — хутряний круг для полірування та wool pad справляються з цим швидше за поролоновий диск полірувальний.

## Як обрати хутряний круг для полірування

| Тип хутра | Особливості | Рівень зрізу |
|-----------|-------------|--------------|
| Натуральна шерсть | М'яка, менше нагрівається | Середній–високий |
| Синтетичне хутро | Агресивніше, довговічніше | Високий |
| Змішаний ворс | Баланс властивостей | Середній |
| Короткий ворс | Точніший контроль | Середній |

**Сценарії вибору:**

- **Якщо працюєте з м'яким японським лаком** → натуральна шерсть, менше ризику пропалити
- **Якщо видаляєте глибокі подряпини** → синтетика з агресивним зрізом
- **Якщо потрібен універсальний варіант** → змішаний ворс
- **Якщо важливий контроль температури** → короткий натуральний ворс

## Шерстяний круг для полірування — на що звернути увагу

| Параметр | Що означає | Рекомендація |
|----------|------------|--------------|
| Діаметр | Розмір робочої поверхні | 125мм — деталі, 150–180мм — панелі |
| Тип кріплення | Шерстяний круг на полірувальну машинку | Липучка (Velcro) — стандарт |
| Довжина ворсу | Агресивність та температура | Довший — холодніше, коротший — точніше |
| Щільність | Рівномірність роботи | Висока щільність — стабільний результат |

## Вовняний круг — особливості роботи

Wool pad з натуральної шерсті вимагає правильної техніки. На відміну від поролону, хутро може залишати голограми — колові подряпини, що видно на сонці. Тому після шерстяного етапу зазвичай проходять м'яким поролоновим падом.

Перед першим використанням новий хутряний круг рекомендують «розбити» — прогнати на низьких обертах з невеликою кількістю пасти. Це розпушує ворс і покращує роботу.

## Що впливає на результат полірування

| Фактор | Вплив |
|--------|-------|
| Тиск на машинку | Занадто сильний — перегрів, слабкий — немає ефекту |
| Швидкість обертів | Високі — агресивніше, низькі — безпечніше |
| Кількість пасти | Мало — сухе тертя, багато — розбризкування |
| Стан паду | Забруднений — подряпини, чистий — якісний зріз |

## FAQ

### Чим хутряний круг відрізняється від поролонового?

Шерстяний пад працює агресивніше — він швидше видаляє дефекти, але може залишати голограми. Поролон м'якший, підходить для фінішного етапу. Якщо у вас глибокі подряпини — почніть з хутра, потім перейдіть на поролон.

### Як доглядати за вовняним кругом?

Після кожного використання видаляйте залишки пасти спеціальною щіткою або стисненим повітрям. Періодично промивайте теплою водою з м'яким засобом. Якщо ворс злипся — пад потребує заміни.

### Натуральна чи синтетична шерсть краще?

Натуральна шерсть працює делікатніше та менше нагріває лак — підходить для м'яких покриттів. Синтетичне хутро агресивніше та довговічніше — для твердих європейських лаків.

### Який діаметр обрати?

Для ексцентрикової машинки з ходом 15–21мм — пади 125–150мм. Для роторної — 150–180мм. Маленькі накладки 75мм — для важкодоступних місць.

---

**Підсумок:**
- Якщо м'який лак — натуральний вовняний круг
- Якщо твердий лак з глибокими дефектами — синтетичне хутро
