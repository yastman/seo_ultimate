# Хімія для миття авто

Правильно підібрана хімія для миття авто — основа якісного догляду за кузовом. Від вибору засобу залежить, наскільки ефективно видаляється бруд та чи збережеться захисне покриття.

## Що входить до хімії для миття

| Підкатегорія | Призначення | Приклади |
|--------------|-------------|----------|
| Активна піна | Безконтактне миття | Лужні, нейтральні піни |
| Автошампуні | Контактне миття | Нейтральні, керамічні |
| Очищувачі | Видалення стійких забруднень | Від бітуму, комах, іржі |
| Знежирювачі | Підготовка до захисту | APC, антисилікон |

## Як обрати хімію для миття

**Якщо безконтактне очищення** — активна піна. Лужна для важких забруднень, нейтральна для регулярного використання та авто з захистом.

**Якщо контактне очищення** — автошампунь. pH-нейтральний зберігає воскові та керамічні покриття, знежирювальний — для підготовки до полірування.

**Якщо стійкі забруднення** — спеціалізовані очищувачі. Бітум, смола, комахи потребують відповідної хімії для миття авто.

### Професійна хімія vs побутова

| Критерій | Професійна | Побутова |
|----------|------------|----------|
| Концентрація | 1:50 – 1:200 | Готова до використання |
| Ефективність | Вища | Достатня для легких забруднень |
| Ціна за літр | Нижча (з розведенням) | Вища |
| Для кого | Автомийки, детейлінг | Домашній догляд |

## Поради щодо вибору

- **Для авто з керамікою** — тільки pH-нейтральні засоби
- **Для щотижневого догляду** — м'які шампуні, нейтральна піна
- **Перед поліруванням** — знежирювальна автохімія для миття

## FAQ

### З чого почати набір хімії?

Мінімальний комплект: активна піна + нейтральний шампунь + очищувач від бітуму. Цього достатньо для 90% задач домашнього догляду.

### Чи можна мити лужною піною авто з воском?

Лужна піна поступово змиває захисні покриття. Якщо хочете зберегти віск чи кераміку — використовуйте нейтральну хімію для миття авто.

### Яка різниця між безконтактним та ручним очищенням?

Безконтактний метод — піна розчиняє бруд без дотику до кузова. Ручний метод — фізичне видалення залишків щіткою чи варежкою. Найкращий результат дає комбінація обох підходів.

---

**Підсумок:** Якщо доглядаєте за авто з захисним покриттям — нейтральна хімія для миття авто. Для глибокого очищення — активна піна та спеціалізовані очищувачі. Професійна автохімія економніша у довгостроковій перспективі.
