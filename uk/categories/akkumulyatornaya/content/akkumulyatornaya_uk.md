# Акумуляторні полірувальні машинки

Акумуляторна полірувальна машинка дає вам свободу руху без прив'язки до розетки. Це особливо важливо для виїзного детейлінгу та роботи в місцях без електрики. Якщо ви обираєте полірувальну машинку на акумуляторі, зверніть увагу на ємність батареї, час роботи та тип приводу — нижче ми допоможемо вам розібратися.

## Як обрати акумуляторну полірувальну машинку

| Параметр | Що означає | На що вам звернути увагу |
|----------|------------|--------------------------|
| Ємність (Ah) | Скільки енергії зберігає | Якщо працюєте довго — обирайте від 5 Ah |
| Напруга | V — потужність | 18V — стандарт, 20V — для інтенсивної роботи |
| Тип приводу | Ексцентрик чи ротор | Якщо ви новачок — ексцентрикова безпечніша |
| Хід ексцентрика | Діаметр орбіти (мм) | 12-15 мм — універсальний вибір |

**Сценарії вибору:**

- **Якщо займаєтесь виїзним детейлінгом** → акумуляторний полірувальник стане основним інструментом, обирайте з двома батареями в комплекті.
- **Якщо працюєте в гаражі з розеткою** → бездротовий інструмент буде доповненням до мережевого для важкодоступних місць.
- **Якщо ви новачок** → починайте з ексцентрикової моделі на акумуляторі — вона безпечніша і не залишає голограм.

## Бездротова полірувальна машинка: переваги та обмеження

Бездротова полірувальна машинка має очевидну перевагу — мобільність. Ви можете полірувати авто в будь-якому місці: на паркінгу, у дворі клієнта, на виставці.

| Перевага | Обмеження |
|----------|-----------|
| Свобода руху без дроту | Час роботи обмежений зарядом |
| Зручність у важкодоступних місцях | Потужність знижується при розряді |
| Компактність для транспортування | Потрібні запасні батареї для довгих сесій |

**Якщо плануєте працювати більше 2 годин** → обов'язково мати другий акумулятор або зарядний пристрій у авто.

## Бездротовий полірувальник для авто: на що звернути увагу

| Характеристика | Рекомендація для вас |
|----------------|---------------------|
| Літій-іонний акумулятор | Стандарт — легкий, без ефекту пам'яті |
| Час роботи від заряду | 40-60 хв на одній батареї — норма |
| Вага з батареєю | До 2.5 кг — комфортно для тривалої роботи |
| Сумісність батарей | Якщо у вас є інструменти того ж бренду — перевірте сумісність |

## FAQ

### Скільки працює акумуляторна полірувальна машинка від одного заряду?
Залежить від ємності та інтенсивності роботи. Зазвичай 40-60 хвилин на 5 Ah. Якщо вам потрібно більше — тримайте запасний акумулятор.

### Чи можна повністю замінити мережеву на акумуляторну?
Для виїзної роботи — так. Для стаціонарного детейлінгу краще мати обидва варіанти: мережеву для довгих сесій, бездротову для мобільності.

### Яку ємність акумулятора обрати?
Якщо працюєте з одним авто за раз — достатньо 4-5 Ah. Якщо кілька авто підряд — обирайте 6+ Ah або тримайте два блоки живлення.

### Чи знижується потужність при розряді?
Так, це особливість акумуляторних інструментів. При 20-30% заряду ви відчуєте зниження оборотів. Плануйте заміну батареї заздалегідь.

---

**Підсумок — що вам обрати:**
- **Для виїзного детейлінгу** → акумуляторна полірувальна машинка ексцентрикового типу з двома батареями.
- **Для гаражної роботи** → полірувальна машинка на акумуляторі як доповнення до мережевого інструменту.
- **Для новачка** → ексцентрик 12-15 мм ходу, уникайте роторних до набуття досвіду.
