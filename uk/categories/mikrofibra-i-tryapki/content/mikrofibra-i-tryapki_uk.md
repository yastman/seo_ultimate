# Мікрофібра для авто

Якісна мікрофібра для авто — основа безпечного догляду без подряпин. Від вибору серветки залежить, чи залишаться розводи після сушіння та чи пошкодиться лак при поліруванні.

## Як обрати мікрофібру для авто

| Тип | GSM | Призначення | Особливості |
|-----|-----|-------------|-------------|
| Коротковорсова | 200-350 | Полірування, скло | Без ворсинок на поверхні |
| Довговорсова | 400-600 | Сушіння, протирання | Вбирає більше вологи |
| Вафельна | 300-400 | Сушіння, скло | Швидке вбирання |
| Твіст | 700-1200 | Сушіння великих поверхонь | Максимальне вбирання |

### Сценарії вибору мікрофібри

**Якщо для сушіння кузова** — вафельна або твіст-мікрофібра з високим GSM. Вбирає воду без розтирання, зменшує ризик подряпин.

**Якщо для полірування** — коротковорсова 300-400 GSM. Ганчірка для авто без розводів має бути без жорстких швів та етикеток.

**Якщо для скла** — гладка безворсова тканина. Не залишає ворсинок, забезпечує ідеальну прозорість.

## На що дивитися при виборі мікрофібри

| Характеристика | Що означає | Рекомендація |
|----------------|------------|--------------|
| GSM (г/м²) | Щільність | Вище = вбирає більше |
| Склад | 70/30 або 80/20 | Більше поліаміду = м'якіше |
| Шви | Ризик подряпин | Без швів або зі швами всередину |
| Розмір | Площа роботи | 40×40 см універсальний |

## Правила догляду

Прайте мікрофібру окремо від іншого одягу. Не використовуйте кондиціонер — він забиває волокна та погіршує вбирання. Сушіть при низькій температурі.

**Ознаки заміни:** жорсткість, втрата вбирання, розлізання волокон, стійкі плями від масел.

## FAQ

### Скільки серветок потрібно для догляду?

Мінімальний набір: 2-3 для сушіння, 2-3 для полірування, 1-2 для скла, 1-2 для інтер'єру. Краще мати запас — брудну мікрофібру не можна використовувати до прання.

### Яка мікрофібра для сушіння авто найкраща?

Твіст-структура з GSM 700+ для великих поверхонь. Вафельна для деталей та швидкого збору води. Обидва типи ганчірок для сушіння працюють добре.

### Чи можна прати з іншими речами?

Не рекомендовано. Інші тканини залишають ворс, який потім переноситься на авто. Пральний порошок без фосфатів, температура до 60°C.

---

**Підсумок:** Для сушіння — висока GSM мікрофібра для авто (твіст або вафля). Для полірування — коротковорсова без швів. Тримайте окремий набір для кузова, скла та інтер'єру.
