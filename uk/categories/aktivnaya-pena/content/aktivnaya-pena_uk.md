# Піна для миття авто

Активна піна для миття автомобіля розм'якшує бруд без контакту з кузовом — безконтактна мийка захищає лак від мікроподряпин. Якщо ваше авто має керамічне покриття або віск — обирайте pH-нейтральну піну, вона збереже захисний шар. Для зимового бруду підійде лужна піна для миття авто, для водяного каменю — кислотна формула.

## Як обрати активну піну для миття автомобіля

| Тип | pH | Миюча сила | Коли брати |
|-----|----|------------|------------|
| Лужний | 10-12 | Висока | Зимовий бруд, реагенти, перша мийка. Змиває віск та силант |
| Нейтральний | 7-9 | Середня | Регулярний догляд, авто з керамікою/воском/PPF |
| Кислотний | 2-4 | Специфічна | Водяний камінь, мінеральні відкладення, вапно. Вимагає обережності з хромом |
| З воском (Wash & Wax) | 7-9 | Середня + захист | Миття і легкий захист в одному флаконі. Не замінює повноцінний віск |

**Сценарії вибору:**

- **Авто з керамічним покриттям або воском** — нейтральний склад збереже гідрофобний шар, лужний при регулярному застосуванні його знімає
- **Пороги вкриті зимовим брудом та реагентами** — лужний концентрат розчиняє сіль та масляну плівку за кілька хвилин
- **Білуваті розводи від жорсткої води** — кислотна формула розчиняє мінеральні відкладення, але тестуйте на непомітній ділянці
- **Хочете миття та захист за один етап** — продукт з воском залишає легкий гідрофобний шар після змивання
- **Побутова мінімийка (тиск нижче 100 бар)** — піна для мінімийки з підвищеною липкістю, рідка консистенція при слабкому тиску — це норма

## Активна піна для безконтактної мийки: на що дивитися

| Маркер на етикетці | Що означає | Рекомендація |
|--------------------|------------|--------------|
| "pH нейтральний" / "Safe for coatings" | Безпечний для кераміки та воску | Регулярний догляд |
| "Alkaline" / "Лужний" | Агресивне очищення | Сильні забруднення, не частіше разу на місяць |
| "Acidic" / "Кислотний" | Розчиняє мінеральні відкладення | Водяний камінь, вапно |
| "Wash & Wax" / "З воском" | Миє і залишає захисний шар | Швидкий догляд без окремого воску |
| "High foam" / "Snow foam" | Густий склад, довше тримається | Для АВТ від 100 бар |

**Тест на прихованій ділянці (Test Spot)** — детейлінг-стандарт для авто із захисними покриттями. Нанесіть продукт на непомітну зону (під дверний ущільнювач), перевірте реакцію через 5 хвилин.

**Dwell time (час витримки)** — скільки склад працює на кузові. Оптимум 5-10 хвилин: хімія розм'якшує бруд, але не встигає висохнути. Якщо висохла до змивання — залишаться білуваті розводи.

## Шампунь для безконтактної мийки: що впливає на результат

| Фактор | Вплив |
|--------|-------|
| Тиск АВТ | Нижче 100 бар — рідка консистенція (норма для побутових мінімийок), 120-150 бар — густий шар |
| Температура кузова | Гарячий метал прискорює висихання — охолодіть водою перед нанесенням |
| Розведення | Міцніше (1:10) — агресивніше, слабше (1:50) — делікатніше та економніше |
| Напрямок нанесення | Знизу вгору — гравітація допомагає складу довше триматися на вертикальних панелях |

**Витрата концентрату активної піни** — 200-400 мл на легковий автомобіль при розведенні 1:20. Літрової пляшки вистачає на 3-5 повних мийок. Якщо потрібна економія — концентрат піни для авто при розведенні 1:50 дає до 50 мийок з однієї каністри.

## Піна для автомобіля: типові помилки при виборі

**Лужна формула для щотижневої мийки** — ваш віск і силант змиються за кілька процедур. Для регулярного догляду беріть тільки нейтральний pH.

**Кислотний склад на хром без тесту** — агресивна хімія може пошкодити хромовані елементи. Обов'язково тестуйте на непомітній ділянці.

**Готовий розчин (RTU) для піногенератора** — без збивання тиском не створює потрібну текстуру, гроші на вітер.

**Залишити піну до повного висихання** — утворюються білуваті плями та розводи, які складно видалити. Змивайте, поки склад ще "пухнастий".

**Нанесення на гарячий кузов** — піна для миття машини швидко стікає і не встигає працювати. Охолодіть поверхню водою або мийте в тіні.

## Сумісність піни для безконтактної мийки з покриттями

| Покриття | Лужна піна | Нейтральна піна | Примітки |
|----------|------------|-----------------|----------|
| Незахищений лак | Обережно (1:20+, до 10 хв) | Безпечно | Лужна вимагає правильного розведення |
| Кераміка | Обережно | Ідеально | Нейтральна не руйнує покриття |
| Віск/силант | При 1:20 безпечно, при 1:5 змиває | Безпечно | Лужна може змити при концентрації |
| PPF (захисна плівка) | Обережно | Рекомендовано | Нейтральна краще для плівок |

## FAQ

### Який продукт обрати для авто з керамікою?

pH-нейтральний без фосфатів. Лужний допустимий при сильному забрудненні — змивайте за 5-10 хвилин, не допускайте висихання на кузові.

### Чим шампунь для безконтактної мийки автомобілів відрізняється від ручного?

Різні етапи. Піна для автомийки (активна піна) — перший етап (pre-wash): наноситься через АВТ, працює 5-10 хвилин, видаляє пухкий бруд без губки. Ручний шампунь — другий етап після ополіскування.

### Чи варто брати продукт з воском?

Для швидкого догляду — так. Wash & Wax економить час: миєте і отримуєте легку гідрофобність за один прохід. Але повноцінний віск або силант це не замінює.

### Чому безконтактна піна швидко стікає і не тримається?

Тиск нижче 100 бар, гарячий кузов, слабке розведення або неякісний піногенератор. Перевірте АВТ (оптимум 120-140 бар), охолодіть поверхню, збільшіть концентрацію.

### Як часто можна використовувати лужну піну?

Раз на місяць максимум. Щотижнево — тільки нейтральну піну для авто. Лужна при частому застосуванні знімає захисні покриття.

**Підсумок — що вам обирати:**

- **Регулярна мийка, авто із захистом** — нейтральний концентрат
- **Зимовий бруд, реагенти** — лужна формула
- **Водяний камінь, мінеральні відкладення** — кислотний склад
- **Миття + легкий захист** — продукт з воском (Wash & Wax)
- **Побутова мінімийка** — професійна піна для миття авто з підвищеною липкістю (High foam)
