# Губки та рукавички

Мочалка для автомобіля — базовий інструмент ручного миття, який визначає безпечність процесу для лакофарбового покриття. Якщо ви миєте машину самостійно, правильний вибір між поролоном, мікрофіброю та шерстю збереже лак від подряпин. Мочалка для автомобіля у вигляді рукавички зручніша за класичну губку і краще утримує піну.

## Як обрати мочалку для автомобіля

| Матеріал | Особливості | Для кого |
|----------|-------------|----------|
| Поролон | Великі пори, швидко набирає воду | Регулярне миття, новачки |
| Мікрофібра | М'яка, утримує бруд у волокнах | Делікатний догляд, темні авто |
| Шерсть (wash mitt) | Максимальна м'якість, преміум | Детейлери, цінний лак |

**Сценарії вибору:**

- **Якщо миєте авто раз на тиждень** → поролонова губка з великими порами — дешева та практична
- **Якщо темне авто і важливо уникнути подряпин** → мійна рукавичка з мікрофібри
- **Якщо працюєте з преміум автомобілями** → шерстяна рукавичка (wash mitt) — найбезпечніший варіант

### Губка для авто чи рукавичка — що краще

Класична поролонова губка добре набирає воду та піну, але вимагає частого полоскання. Бруд осідає на поверхні і може дряпати лак.

Рукавичка для миття авто зручніше сидить на руці, а її волокна захоплюють бруд всередину. Після кожної панелі достатньо прополоскати у відрі з чистою водою.

## На що звернути увагу при виборі мочалки для автомобіля

| Характеристика | Що означає | Рекомендація |
|----------------|------------|--------------|
| Великі пори | Утримує більше піни | Для ефективного миття |
| Двостороння | Різні текстури | Універсальне застосування |
| Петля/кріплення | Фіксація на руці | Зручність використання |
| Chenille/синель | Довгий ворс | Делікатне очищення |

## Особливості використання

Будь-яка губка потребує попереднього замочування у мильному розчині. Суха поверхня може накопичувати статику і притягувати пил.

Використовуйте метод двох відер: одне з шампунем, інше з чистою водою для полоскання. Це запобігає перенесенню бруду назад на лак.

## FAQ

### Як часто міняти губку?

Поролонову — кожні 2-3 місяці при активному використанні. Мікрофіброву рукавичку — раз на рік при правильному догляді. Шерстяну wash mitt — може служити роками.

### Чи можна мити двигун тією ж губкою?

Ні, для моторного відсіку використовуйте окрему щітку або дешеву губку. Жир та пил з двигуна забруднять інструмент для кузова.

### Яка різниця між поролоном та мікрофіброю?

Поролон утримує піну на поверхні, мікрофібра — захоплює бруд у волокна. Для темного лаку мікрофібра безпечніша, для швидкого миття світлого авто — поролон практичніший.

### Як доглядати за рукавичкою?

Після миття ретельно прополощіть у чистій воді та висушіть на повітрі. Мікрофібру та шерсть можна прати в машинці без кондиціонера.

---

**Підсумок:**
- Якщо бюджетний варіант — поролонова мочалка для автомобіля з великими порами
- Якщо темний лак — мікрофіброва мийна рукавичка
- Якщо преміум авто — шерстяний wash mitt
