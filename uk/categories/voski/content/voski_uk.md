# Віск для авто

Віск для авто — захисне покриття, що надає блиск і водовідштовхування кузову. Воскові засоби бувають натуральними, синтетичними і гібридними. Правильно підібраний віск автомобільний захищає ЛКП від UV, бруду та легких подряпин.

## Як обрати віск для авто

| Тип | Форма | Для кого |
|-----|-------|----------|
| Твердий | Паста в банці | Максимальний блиск, шоу-кари |
| Рідкий | Рідина у флаконі | Зручність нанесення |
| Спрей | Розпилювач | Швидкий догляд між миттям |

**Сценарії вибору:**
- **При акценті на глибокий блиск** → твердий натуральний карнауба
- **Для балансу зручності та якості** → рідкий формат
- **Для швидкого догляду між повним миттям** → спрей-варіант

### Типи за складом

| Склад | Особливості |
|-------|-------------|
| Натуральний | Карнауба, бджолиний — теплий блиск |
| Синтетичний | Полімери — довговічність |
| Гібридний | Поєднання — баланс властивостей |

## Особливості використання воску для авто

Захисний засіб наноситься на чисту суху поверхню. Більшість продуктів потребують затвердіння до матового стану перед розполіруванням.

**Якщо залишає плями** → нанесли на гарячий кузов або під прямим сонцем.

## Від чого залежить результат віск-захисту

| Фактор | Вплив |
|--------|-------|
| Підготовка | Чистий кузов = краща адгезія |
| Температура | У тіні, 15–25°C |
| Інструмент | Аплікатор + чиста мікрофібра |

## FAQ

### Чим відрізняється від силанту?

Натуральний карнауба дає тепліший блиск, але тримається 4–8 тижнів. Синтетичний засіб та силанти — 3–6 місяців. При пріоритеті естетики обирайте карнаубу, при практичному підході — синтетику.

### Як часто наносити?

**Якщо натуральний карнауба** → кожні 4–6 тижнів. Синтетичний — раз на 2–3 місяці. Спрей можна після кожного миття для підтримки.

### Чи можна наносити на скло?

Не рекомендується. Для скла є спеціальні засоби. У разі випадкового потрапляння — видаліть засобом для скла, поки не засох.
