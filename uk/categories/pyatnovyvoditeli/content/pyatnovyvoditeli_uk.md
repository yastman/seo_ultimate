# Плямовивідник

Плямовивідник для авто — спеціалізований засіб для видалення локальних забруднень з тканинних та велюрових поверхонь салону. На відміну від загальних очищувачів, цей засіб працює точково і справляється з застарілими та стійкими плямами.

## Як обрати плямовивідник для авто

| Тип | Принцип дії | Для кого |
|-----|-------------|----------|
| Ензимний | Розщеплює органіку | Білкові плями (кров, молоко, їжа) |
| Окислювальний | Знебарвлення | Кава, чай, сік |
| Розчинний | Розчиняє жири | Олія, мастило, помада |

**Сценарії вибору:**
- **Якщо пляма від їжі чи напоїв** → ензимний засіб (він «з'їдає» органіку)
- **Якщо пляма від олії чи мастила** → розчинний тип на основі спирту або ацетону
- **Якщо невідоме походження** → універсальний spot remover для салону

### На що дивитися на етикетці

| Маркер | Значення |
|--------|----------|
| «Enzymatic» | Ензимний склад для органіки |
| «Oxygen-based» | Кисневий для знебарвлення |
| «Safe for fabrics» | Безпечний для тканин |
| «Pre-spray» | Наносити перед основним чищенням |

## Особливості використання плямовивідника для салону авто

Плямовивідник для сидінь авто наноситься точково на забруднення, не на всю поверхню. Час витримки залежить від типу плями: свіжі — 2–3 хвилини, застарілі — до 10–15 хвилин. Після обробки промокніть мікрофіброю або витягніть екстрактором.

**Якщо пляма не зникла з першого разу** → повторіть обробку або спробуйте засіб іншого типу.

## Що впливає на результат

| Фактор | Вплив |
|--------|-------|
| Свіжість плями | Свіжі видаляються легше |
| Тип тканини | Синтетика чиститься краще за натуральні |
| Температура | Тепла вода посилює дію ензимів |

## FAQ

### Чим цей засіб відрізняється від очищувача салону?

Spot remover — це концентрований засіб для локальних забруднень. Очищувач салону — для загального чищення великих площ. **Якщо є конкретна пляма** → спочатку точкова обробка, потім загальне чищення.

### Чи безпечний для шкіри та алькантари?

Більшість таких засобів розроблені для тканин. Для шкіри та алькантари використовуйте спеціалізовані засоби — універсальні можуть залишити сліди або пошкодити покриття.

### Як уникнути розводів після обробки?

Після нанесення та витримки витягніть засіб разом із забрудненням — мікрофіброю або екстрактором. Не залишайте засіб висихати на поверхні.

---

**Підсумок:**
- Якщо органічні плями: ензимний засіб для видалення
- Якщо жирові забруднення: розчинний засіб на спиртовій основі
