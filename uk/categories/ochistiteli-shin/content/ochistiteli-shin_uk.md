# Очищувач шин

Очищувач шин — спеціалізована лужна хімія для глибокого очищення гуми. Звичайний автошампунь змиває тільки пил, але безсилий проти **антиозонантів (рудого нальоту)**, старого силікону та дорожньої плівки. Засіб для чистки шин створює «фундамент» — ідеально чисту гуму для стійкого чорніння.

**Головне правило:** Якщо нанести чорнитель на брудну шину, він не вбереться і розлетиться точками по кузову (Sling) на першій же поїздці.

## Який формат обрати

### Готові спреї (Ready-to-Use)

Класичний формат «купив — нанів».

- **Для кого:** Для ентузіастів та догляду за особистим авто
- **Плюси:** Безпечна заводська пропорція, не потрібно нічого змішувати
- **Витрата:** залежить від ступеня забруднення і розміру шин

### Концентрати (Pro Line)

Потужна основа, що вимагає розведення водою (1:3 – 1:10).

- **Для кого:** Для тих, хто миє часто, або для сім'ї з кількома машинами
- **Плюси:** Економія (платите тільки за активну речовину) + гнучкість (1:3 для старих шин, 1:10 для профілактики)
- **Витрата:** залежить від пропорції розведення
- **Вимога:** Потрібна порожня пляшка з тригером

### Склади «2-в-1» (Wheel & Tire)

Універсальні засоби для дисків і гуми.

- **Для кого:** Для експрес-миття
- **Плюси:** Один склад на все колесо — економить час
- **Мінуси:** Слабші за спеціалізовані проти складного нагару на дисках

## Чек-лист покупця: що докупити одразу

Очищувач гуми — це частина системи. Сама по собі банка працювати не буде.

**Жорстка щітка (Must Have)** — на відміну від дисків, гуму не можна подряпати. М'які пензлики марні — тільки жорсткий нейлон вибиває бруд із пор.

**Аплікатор чорнителя** — поролоновий для низького профілю (економить чорнитель), пензель для позашляхових шин (AT/MT) з високим рельєфом.

**Рукавички** — pH 12–13 сушить шкіру рук. Працюйте в нітрилових рукавичках.

**Чорнитель (Dressing)** — очищувач знежирює і сушить гуму. Залишати її «голою» не можна — посіріє і потріскається. Чорнитель зволожує і захищає.

## Тест білої піни — як зрозуміти, що шина чиста

Золотий стандарт детейлінгу:

- **Піна коричнева/рuda** → хімія витягує бруд із пор, потрібно змити і повторити
- **Піна біла** → шина ідеально чиста, можна сушити і наносити чорнитель

На занедбаних шинах це 2–3 проходи засобом для очистки гуми.

## Сумісність із дисками

Більшість складів мають лужну основу (pH 12–13).

**Безпечно:** Для лакованих та пофарбованих дисків (при дотриманні інструкції).

**Обережно: полірований алюміній та анодування!** Луг (Sodium Hydroxide) з'їдає оксидну плівку за хвилини. Диск стає білуватим/мутним. **Відновити не можна** — тільки перефарбування. Змивайте хімію з дисків негайно!

## ⚠️ Критичні помилки

**Робота без щітки** — хімія розм'якшує бруд, але не видаляє його.
**Наслідок:** Без механічної проробки пори залишаються забитими, чорнитель не вбереться.
**Рішення:** Завжди працюйте жорсткою щіткою.

**Залишити гуму «голою»** — очищувач знежирює і сушить.
**Наслідок:** Без чорнителя шина посіріє і почне тріскатися через 2–3 тижні.
**Рішення:** Після очищення обов'язково нанесіть дресинг.

**Потрапляння на полірованi/анодованi диски** — луг (pH 12–13) з'їдає оксидну плівку за хвилини.
**Наслідок:** Диск стає білуватим, відновити не можна.
**Рішення:** Змивайте хімію з дисків негайно.

**Робота на сонці** — склад висихає швидше, ніж встигаєте змити.
**Наслідок:** Залишаються білі розводи, які складно прибрати.
**Рішення:** Працюйте в тіні, змивайте за 2–3 хвилини.

## Підсумок вибору

- **Для дому:** Готовий очищувач шин у спреї — 4–6 машин із пляшки
- **На потоці:** Концентрат 1–5 л, розводиться 1:3–1:10 — на 30–150 миттів
- **Інструмент:** Жорстка щітка — без неї хімія не працює
- **Захист рук:** Рукавички обов'язкові
- **Після очищення:** Чорнитель — щоб гума не сохла

## FAQ: Відповіді експерта

### Чому чорнитель «злітає» через день?

Причина — погане очищення. Пори забиті → дресинг лежить плівкою зверху → злітає на першій кочці. Рішення: тест білої піни + повне сушіння + тонкий шар.

### Як відмити рудий наліт із шин у домашніх умовах?

Рудий наліт — це антиозонанти (Tire Blooming). Нанесіть очищувач шин, потріть жорсткою щіткою, змийте. Повторюйте, поки піна не стане білою. На занедбаних шинах — 2–3 проходи.

### Чи можна використовувати для гумових килимків?

Так! Очищувач шин ідеально підходить для глибокого чищення килимків — повертає насичений чорний колір і прибирає в'їлий бруд.

### Чи можна мити цим диски?

**Ні!** Очищувач шин занадто агресивний (pH 12–13) для лаку та анодування. Для дисків потрібен спеціалізований очищувач з підходящим pH.
