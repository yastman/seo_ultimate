# Очищувач гуми

Очищувач гуми — спеціалізований засіб для чищення бокової частини покришок перед нанесенням чорнителя або захисного дресингу. Засіб для догляду за гумою визначає стійкість фінішного покриття — без глибокого очищення дресинг тримається лише кілька днів.

## Як обрати очищувач шин

| Тип | pH | Для кого |
|-----|-----|----------|
| Лужний препарат | 10–12 | Глибоке очищення застарілих нашарувань |
| Нейтральний | 7–8 | Регулярне обслуговування, делікатні покришки |
| Кислотний | 2–4 | Видалення мінеральних відкладень |

**Сценарії вибору:**
- **Якщо покришки давно не чистились** → лужна хімія (розчиняє старий силікон і бруд)
- **Якщо регулярне миття** → нейтральний склад без агресивного впливу
- **Якщо є білий наліт** → кислотний засіб для видалення солей

### На що дивитися на етикетці

| Маркер | Значення |
|--------|----------|
| «Tire cleaner» | Саме для покришок, не універсальний APC |
| «Alkaline» / «pH 11+» | Лужний склад для складних забруднень |
| «Safe for rubber» | Не пересушує матеріал |
| «Ready to use» | Готовий розчин без розведення |

## Засіб для догляду за шинами — особливості

Лужна хімія працює за принципом емульгування — розчиняє органічні речовини (старий дресинг, дорожню плівку, бітум). Час витримки залежить від ступеня забруднення: легке — 2–3 хвилини, застаріле — до 5 хвилин.

Відновлювач гуми повертає чорний колір посірілим покришкам. Поліроль для гуми створює глянсовий або матовий фініш залежно від типу.

**Якщо очищення перед чорнінням** → важливо повністю змити залишки, інакше дресинг не зчепиться з поверхнею.

## Що впливає на результат

| Фактор | Вплив |
|--------|-------|
| Ступінь забруднення | Чим більше шарів — тим довша витримка |
| Температура | Холодний матеріал працює повільніше |
| Тип щітки | Жорстка щетина для текстурованих покришок |

## FAQ

### Чим очищувач відрізняється від APC?

Спеціалізований препарат має склад, оптимізований для роботи з каучуком — не пересушує поверхню і краще розчиняє характерні забруднення (старий силікон, антиозонантні воски). APC може справлятися з легким брудом, але для глибокого очищення перед чорнінням потрібен саме tire cleaner.

### Як часто потрібно чистити покришки?

**Якщо регулярно наносите дресинг** → очищення перед кожним нанесенням. Якщо просто миєте авто — раз на 2–4 тижні достатньо для підтримки вигляду.

### Чи можна використовувати на мотоциклетних покришках?

Так, але тільки на бокові частини. Протектор контактних поверхонь обробляти не рекомендується — залишки засобу можуть вплинути на зчеплення.

---

**Підсумок:**
- Якщо покришки з товстим шаром старого дресингу: обирайте лужний препарат
- Якщо регулярний догляд: нейтральний засіб збереже матеріал в належному стані
