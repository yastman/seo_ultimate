# Очищувач шин

Очищувач шин — спеціалізований засіб для чищення бокової частини гуми перед нанесенням чорнителя або захисного дресингу. Якість підготовки визначає стійкість фінішного покриття — без глибокого очищення дресинг тримається лише кілька днів.

## Як обрати очищувач шин

| Тип | pH | Для кого |
|-----|-----|----------|
| Лужний очищувач | 10–12 | Глибоке очищення застарілих нашарувань |
| Нейтральний | 7–8 | Регулярне обслуговування, делікатні гуми |
| Кислотний | 2–4 | Видалення мінеральних відкладень |

**Сценарії вибору:**
- **Якщо гума давно не чистилась** → лужна хімія (вона розчиняє старий силікон і бруд)
- **Якщо регулярне миття** → нейтральний склад без агресивного впливу
- **Якщо є білий наліт** → кислотний засіб для видалення солей

### На що дивитися на етикетці

| Маркер | Значення |
|--------|----------|
| «Tire cleaner» | Саме для шин, не універсальний APC |
| «Alkaline» / «pH 11+» | Лужний склад для складних забруднень |
| «Safe for rubber» | Не пересушує гуму |
| «Ready to use» | Готовий розчин без розведення |

## Особливості використання очищувача гуми

Лужна хімія для шин працює за принципом емульгування — розчиняє органічні речовини (старий дресинг, дорожню плівку, бітум). Час витримки залежить від ступеня забруднення: легке — 2–3 хвилини, застаріле — до 5 хвилин.

**Якщо очищення перед чорнінням** → важливо повністю змити залишки, інакше дресинг не зчепиться з поверхнею.

## Що впливає на результат

| Фактор | Вплив |
|--------|-------|
| Ступінь забруднення | Чим більше шарів — тим довша витримка |
| Температура | Холодна гума працює повільніше |
| Тип щітки | Жорстка щетина для текстурованих шин |

## FAQ

### Чим очищувач шин відрізняється від APC?

Спеціалізований очищувач гуми має склад, оптимізований для роботи з каучуком — не пересушує поверхню і краще розчиняє характерні для шин забруднення (старий силікон, антиозонантні воски). APC може справлятися з легким брудом, але для глибокого очищення перед чорнінням потрібен саме tire cleaner.

### Як часто потрібно чистити шини?

**Якщо ви регулярно наносите дресинг** → очищення перед кожним нанесенням. Якщо просто миєте авто — раз на 2–4 тижні достатньо для підтримки вигляду.

### Чи можна використовувати на мотоциклетних шинах?

Так, але тільки на бокові частини. Протектор контактних поверхонь обробляти не рекомендується — залишки засобу можуть вплинути на зчеплення.

---

**Підсумок:**
- Якщо шини з товстим шаром старого дресингу: обирайте лужний очищувач
- Якщо регулярний догляд: нейтральний засіб збереже гуму в належному стані
