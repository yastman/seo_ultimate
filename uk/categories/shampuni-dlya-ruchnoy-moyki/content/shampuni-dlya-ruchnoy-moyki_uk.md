# Автошампунь для ручного миття

Автошампунь для ручного миття — концентрований засіб для контактної обробки кузова губкою чи рукавичкою. Правильний шампунь для ручного миття авто забезпечує лубрикацію (ковзання), мінімізуючи ризик мікроподряпин під час протирання.

## Як обрати шампунь для ручного миття авто

| Тип | Особливості | Для кого |
|-----|-------------|----------|
| Нейтральний шампунь | pH 6–8, безпечний | Авто із захисним покриттям |
| Керамічний шампунь | Додає SiO2 шар | Підтримка керамічних покриттів |
| Воскований шампунь | Миття + легкий захист | Швидкий догляд без окремого воску |

**Сценарії вибору:**
- **Якщо авто з керамікою або PPF** → автомобільний шампунь для ручного миття нейтрального pH
- **Якщо хочете підсилити гідрофоб** → керамічний шампунь з SiO2
- **Якщо немає часу на окремий віск** → воскований варіант 2-в-1

### На що дивитися на етикетці

| Маркер | Значення |
|--------|----------|
| «pH-neutral» | Безпечний для захисних покриттів |
| «High lubricity» | Високе ковзання = менше подряпин |
| «Wax» / «Gloss» | Має восковий або глянцевий ефект |
| «Dilution 1:500» | Концентрація розведення |

## Особливості використання автошампуню для ручного миття

Шампунь для миття авто вручну використовується у методі двох відер: одне з мильним розчином, друге — з чистою водою для полоскання рукавички. Пропорція розведення: концентрати зазвичай 1:200–1:500, точне значення — на етикетці.

**Якщо шампунь занадто піниться** → зменшіть концентрацію; надмір піни ускладнює змивання.

## Місце у двофазному митті

| Фаза | Засіб | Призначення |
|------|-------|-------------|
| Перша | Активна піна | Безконтактне видалення основного бруду |
| Друга | Автошампунь | Контактне домивання залишків |

**Якщо пропустити першу фазу** → весь бруд тертиметься по лаку під час ручного миття, ризик подряпин зростає.

## FAQ

### Чим автошампунь відрізняється від активної піни?

Активна піна — для безконтактного попереднього миття (перша фаза). Шампунь для ручного миття авто — для контактної доробки (друга фаза). Вони доповнюють один одного: піна знімає основний бруд, шампунь — те, що залишилося.

### Чи можна використовувати побутовий миючий засіб?

Не рекомендується. Побутова хімія (Fairy, AOS) знежирює занадто агресивно — змиває захисні покриття і пересушує гуму. Нано шампунь або нейтральний шампунь для авто розроблені спеціально для ЛКП.

### Скільки шампуню витрачається на одне миття?

**Якщо концентрат 1:400** → 30–50 мл на 15–20 л води у відрі достатньо для одного легкового авто.

---

**Підсумок: як обрати автошампунь для ручного миття**
- Якщо авто з керамічним покриттям: нейтральний або керамічний шампунь
- Якщо швидкий догляд: воскований автошампунь для ручного миття та легкого захисту одночасно
