## 1. Типология наборов для авто

### 6 основных типов

| Тип | Целевая аудитория | Типичный состав | Стойкость | Когда выбирать |
|-----|----------|----------|----------|----------|
| **Набор для мойки (Exterior)** | Автолюбители, еженедельная мойка | Шампунь, губка, микрофибра, ведро, wax/sealant | 5-7 компонентов | Регулярный уход; дома; недорого начать |
| **Набор для химчистки салона (Interior)** | Владельцы с грязным салоном | Очистители (ткань/кожа/пластик), щетки, кисти, микрофибра | 6-8 компонентов | Профилактика; пятна; запахи в салоне |

| **Подарочный (Gift Box)** | Подарок автомобилистам | Красивая упаковка; ассортимент средств; аксессуары | 5-8 компонентов | День рождения; 23 февраля; Рождество |

## 2. Что должно входить в базовые наборы

### 2.1 Набор для мойки автомобиля (Exterior)

#### Минимальный состав (4-5 компонентов)

| Компонент | Назначение | Критерии выбора | Источник |
|-----------|-----------|--------|----------|
| **pH-нейтральный шампунь** | Основная мойка без повреждения лака | pH 7.0 ± 0.5; не использовать dish soap | [1,4,14] |
| **Микрофибровая губка или варежка** | Мягкое нанесение шампуня | Двойное плетение; 300-400 GSM (не менее) | [2,4,11] |
| **Микрофибра для сушки** | Удаление воды без полос | 300-400 GSM; отдельная от мойки; вафельное плетение | [2,11] |
| **Ведро (2 шт. рекомендуется)** | Два ведра: с шампунем, с полосканием | С grit guard на дне (защита от грязи) | [1,4] |
| **Воск или защита** | Завершающая защита после мойки | Paste, liquid или spray; совместимо с краской | [4,5] |

#### Оптимальный состав (7-10 компонентов)

| Компонент | Назначение | Источник |
|-----------|-----------|----------|
| Шампунь (pH-neutral) | Основная мойка | [1,4,14] |
| Микрофибра губка / varjeshka | Нанесение шампуня | [1,11] |
| Микрофибра для сушки (2-3 шт) | Сушка без полос | [2,11] |
| Ведро с grit guard (2 шт) | Система двухведёрной мойки | [1,4] |
| Очиститель для дисков | Специально для колёс | [5,11] |
| Щётка для колёс | Жёсткая для колёс; мягкая для дисков | [7,11] |
| Воск/силант | Защита после мойки | [4,5] |
| Быстрый детейлер (Quick Detailer) | Дополнительный блеск между мойками | [4,6] |
| Очиститель стекла | Кристальная чистота стекол | [11] |
| Полотенце для дохлопов | Финальная сушка труднодоступных мест | [1] |

**Критерий качества микрофибры** [2]: Минимум 300 GSM (граммов на кв. метр); двойное плетение; никаких размягчителей при стирке

### 2.2 Набор для химчистки салона (Interior)

#### Состав по типам поверхностей

| Тип салона | Очиститель | Щётки/Кисти | Кондиционер | Микрофибра | Примечание |
|----------|-----------|-----------|-----------|-----------|----------|
| **Тканевый салон** | Универсальный очиститель ткани | Щётки для ткани (мягкие) | Fabric protector | 300-400 GSM | Безводная опция для спешки |
| **Кожаный салон** | pH-neutral кожаный очиститель | Специальные щётки для кожи | Кожаный кондиционер | 300 GSM short-pile | Glycerine-based; не сушит |
| **Пластиковые детали** | All-Purpose Cleaner (APC) | Жёсткие щётки для пластика | Plastic restorer/дressing | Микрофибра | Protects from UV fading |
| **Комбо (ткань+кожа+пластик)** | 3 специализированных очистителя | Набор 5-7 щёток разных | 2 кондиционера (ткань+кожа) | 5-7 микрофибр | Most complete |

#### Минимальный интерьерный кит (5-7 позиций)

1. **Универсальный очиститель** (All-Purpose Cleaner / APC)
2. **Щётки для деталей** (3-5 разных размеров)
3. **Микрофибра для салона** (3-5 шт; отдельно от наружной)
4. **Vacuum (пылесос)**
5. **Защита (Protectant)**
6. **Освежитель воздуха** (опционально)
7. **Кисти малые** (для вентиляции, кнопок)

**Источники** [3,7,12,15]

### 2.3 Стартовый набор детейлера (Beginner Pro)

#### Обязательные компоненты (15-20 позиций)

**Инструменты (Power tools):**
- Pressure washer (1500-2000 PSI) или foam cannon
- Air compressor + blower (для сушки)
- Wet-Dry vacuum (интерьер)

**Химия (Exterior):**
- pH-neutral car shampoo
- Wheel & tire cleaner
- Clay bar kit
- Polishing compound (если нужно)
- Wax / Sealant / Ceramic
- Quick detailer
- Glass cleaner
- Trim dressing

**Инструменты (Manual):**
- 2-3 wash mitts
- 3-5 microfiber towels (дойка)
- 2-3 drying towels
- Glass towel (waffle weave)
- Tire brush (жёсткая)
- Wheel brush (мягкая)
- Detail brushes (3-5 разных)
- Grit guards (2 шт для buckets)

**Интерьер:**
- Interior cleaner (universal APC)
- Leather cleaner (если есть кожа)
- Upholstery brush set
- Microfiber cloths для интерьера
- Vacuum with attachments

**Стоимость**: $200-400 USD (начальный уровень) [7,10,11]

| Формат | Преимущества | Недостатки | Когда выбирать |

## 7. Частые ошибки при выборе кита

| Ошибка | Последствие | Как избежать | Источник |
|--------|-------------|------------------|----------|
| **Набор с дублирующимися средствами** | Переплата; излишество | Проверить состав; убедиться в разнообразии | [4,6] |
| **Несовместимые средства в наборе** | Может снижать эффект; конфликт формул | Выбирать monobrend или проверенные комбо | [6] |
| **Переплата за ненужные компоненты** | Не использует; деньги впустую | Выбрать минимальный кит; расширять потом | [4] |
| **Недооценка значения микрофибры** | Царапины на краске; плохой результат | Выбрать 300+ GSM; отдельную для каждого назначения | [2,11] |
| **Выбор кита не по цели** | Например, подарок вместо рабочего | Четко определить цель использования | [5,13] |
| **Liquid wax вместо paste (для новичка)** | Менее эффективно; короче держится | Для новичка даже liquid OK; главное — правильная техника | [4,5] |
| **Игнорирование pH-нейтрального шампуня** | Повреждение лака; потускнение | ВСЕГДА выбирать pH-neutral (7.0 ± 0.5) | [1,4,14] |

## 8. Преимущества готового кита vs отдельных покупок

| Критерий | Готовый кит | Отдельные покупки | Вывод |
|----------|----------|----------|--------|
| **Цена** | Часто 10-20% дешевле (bundling) | Базовая цена | Кит выгоднее ✅ |
| **Сбалансированность** | Продумано производителем | Нужно разбираться | Кит лучше для новичка ✅ |
| **Удобство** | Всё в одной коробке | Нужно искать отдельно | Кит удобнее ✅ |
| **Гибкость** | Ограничена составом | Полная кастомизация | Отдельно гибче ✅ |
| **Экономия на аксессуарах** | Включены щётки, микрофибра | Нужно добавить | Кит экономит ✅ |

**Вывод** [4,5,6,13]: Для новичков кит выгоднее; для опытных — лучше собрать самому

## 9. Сценарии использования и рекомендации

### Сценарий 1: "Купил новую машину — хочу ухаживать сам"

**Рекомендация**: Combo kit (exterior + basic interior)  
**Состав**: 10-15 компонентов; включить: shampoo, wax, interior cleaner, towels  
**Цена**: 800-1500 грн  
**Где**: Amazon, local retailers, detailing shops  
**Источник**: [5,13]

### Сценарий 2: "Подарить полезное другу-автомобилисту"

**Рекомендация**: Gift box набор (красивая упаковка)  
**Состав**: Разнообразные средства + микрофибра + инструкция  
**Цена**: 1000-3000 грн  
**Критерий**: Красивая упаковка > функциональность  
**Источник**: [5]

### Сценарий 3: "Нужна химчистка салона — купить кит или отдельные средства?"

**Рекомендация**: Interior specialist kit (если нужно регулярно)  
**Если одноразово**: Отдельные средства (cleaner + brush + microfiber)  
**Состав кита**: Cleaner (ткань/кожа/пластик) + brushes + microfiber + protectant  
**Цена**: 600-1200 грн кит; 200-300 грн отдельные  
**Источник**: [3,12]

### Сценарий 4: "Начинаю бизнес детейлинга"

**Рекомендация**: Beginner Pro kit + power tools  
**Инвестиция**: $300-500 (первоначальный) + $500-1500 (power tools)  
**Состав**: Full exterior + interior + tools + chemicals  
**Источники обучения**: YouTube каналы, Reddit threads  
**Источник**: [7,10,11]

### Сценарий 5: "Есть мойка высокого давления — какой кит взять?"

**Рекомендация**: Химия к pressure washer + foam cannon combo  
**Дополнить**: Shampoo concentrate; wheel cleaner; tire shine; drying aids  
**Цена**: 400-800 грн (just chemicals)  
**Не забыть**: Grit guards для buckets!  
**Источник**: [1,7]

## 11. Критический вывод: Что выбрать?

### Простой чек-лист для покупателя

**Задайте себе 3 вопроса:**

1. **Цель?**
   - Регулярный уход → Exterior/Combo kit
   - Подарок → Gift box
   - Бизнес → Beginner Pro
   - Салон → Interior kit

2. **Опыт?**
   - Новичок → Ready-made кит (максимум 10-15 компонентов)
   - Опытный → Можете собрать сами или выбрать специализированный
   - Профи → Custom builder или professional kits

3. **Бюджет?**
   - До 800 грн → Бюджетный exterior
   - 800-2000 грн → Средний combo
