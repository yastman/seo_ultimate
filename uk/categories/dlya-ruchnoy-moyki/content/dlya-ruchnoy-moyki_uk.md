# Автошампунь для ручного миття

Автошампунь для ручного миття — це не просто «мило для машини». Його головне завдання — **лубрикація** (ковзання). Шампунь створює маслянисту прошарок між губкою та лаком, завдяки якій піщинки ковзають, а не дряпають. Без лубрикації ручне миття = мережа подряпин на сонці. **Обов'язково зб'їйте бруд безконтактною піною перед ручним миттям!** Ручне миття по піску — смерть для лаку.

## Як обрати

| Тип | pH | Призначення | Особливості |
|-----|-----|-------------|-------------|
| Нейтральний (Pure) | 7 | Регулярне миття авто в кераміці | Не залишає плівки, не заважає захистам |
| З добавками (Wax/SiO2) | 7–8 | Блиск і тимчасовий захист | «Лінива» мийка — помив і захистив |
| Кислотний (Acidic) | 4–6 | Реанімація кераміки (Descaling) | Видаляє водяний камінь, повертає гідрофоб |
| Знежирюючий (Strip Wash) | 9–10 | Підготовка до полірування | Змиває старі воски та силанти |

**Сценарії вибору:**

- **Є кераміка або віск** → нейтральний автомобільний шампунь для ручного миття (pH 7). Не чіпає захист.
- **Хочете захист одразу** → шампунь для ручного миття з воском або SiO2. Гідрофоб на 1–2 тижні.
- **Кераміка перестала відштовхувати воду** → кислотний шампунь для авто (pH 4–6). Розчиняє мінеральні відкладення в порах кераміки.
- **Перед поліруванням або новим покриттям** → Strip Wash (pH 9–10). Видаляє все, що заважає адгезії.
- **Сильний бруд після траси** → шампунь з посиленими ПАР + обов'язково Pre-Wash піною.

## Метод двох відер

Двовідерне миття — стандарт детейлінгу, який запобігає подряпинам. Забудьте про одне відро на всю машину.

**Алгоритм:**

1. **Відро з шампунем** → занурюєте рукавицю.
2. **Миєте одну панель** (капот, двері — не всю машину одразу).
3. **Відро з чистою водою** → полощете рукавицю, терете об решітку (Grit Guard).
4. **Знову в шампунь** → повторюєте.

**Grit Guard (решітка):** Бруд осідає під решіткою завдяки гасінню водовороту. Піна тут ні при чому — працює саме фізика: решітка не дає бруду піднятися назад.

**Рухи:** Завжди **прямі поздовжні** (вздовж кузова), НЕ кругові. Кругові подряпини (swirl marks) найпомітніші на сонці.

**Інструмент:** Забудьте про щільні жовті губки з супермаркету. Використовуйте **мікрофіброву рукавицю** або великопористу губку. Вони втягують пісок усередину себе, а не труть ним по кузову.

## Як застосовувати

1. **Pre-Wash обов'язковий!** Нанесіть безконтактну піну, витримайте 3–5 хвилин, змийте. Це прибере 80% абразиву.
2. Розведіть шампунь для миття кузова авто у відрі згідно з інструкцією виробника (дозування залежить від концентрації).
3. Мийте зверху вниз — дах, капот, двері, пороги останніми (там найбільше бруду).
4. Використовуйте метод двох відер із решітками.
5. Змийте залишки і висушіть вафельною або плюшевою мікрофіброю.

## ⚠️ Критичні помилки

**Миття без Pre-Wash** — пісок на кузові + рукавиця = подряпини.
**Наслідок:** Мережа swirl marks, видима на сонці.
**Рішення:** Безконтактна піна обов'язкова перед кожним ручним миттям!

**Одне відро** — бруд із рукавиці повертається на кузов.
**Наслідок:** Губка знову несе абразив по лаку.
**Рішення:** Двовідерний метод — стандарт. Мінімум — полощіть рукавицю в окремому відрі.

**Кругові рухи** — створюють swirl marks, які видно під будь-яким кутом.
**Наслідок:** «Голограми» і подряпини по всьому кузову.
**Рішення:** Тільки прямі поздовжні рухи!

**Висихання шампуню** — на сонці хімія залишає плями.
**Наслідок:** Білуваті розводи, що важко відмити.
**Рішення:** Працюйте в тіні, змивайте швидко.

**Засіб для посуду (Fairy)** — категорично ні.
**Наслідок:** Містить солі та агресивні жировидаляючі компоненти: **висушує ґумки та пластик** (вони біліють), миттєво вбиває воски та кераміку.
**Рішення:** Тільки автомобільний шампунь — він створений з урахуванням лубрикації та pH-балансу.

## Підсумок вибору

- **Щоденний догляд:** Нейтральний автошампунь для ручного миття (pH 7) — безпечний для будь-яких покриттів
- **Хочете захист:** З воском або SiO2 — гідрофоб на 1–2 тижні після кожного миття
- **Кераміка втратила гідрофоб:** Кислотний автошампунь (pH 4–6) — раз на 1–2 місяці для «реанімації»
- **Перед поліруванням:** Strip Wash — повне видалення старих захистів

## FAQ: Відповіді експерта

### Чи можна мити машину засобом для посуду?

Ні. Fairy та аналоги містять агресивні жировидаляючі компоненти та солі. Вони **висушують гумові ущільнювачі** (стають білуватими і тріскаються), вбивають пластик і миттєво змивають будь-який захист — віск, силант, навіть послаблюють кераміку. Автомобільний шампунь для ручного миття створений з урахуванням лубрикації та pH-балансу.

### Чому кераміка перестала відштовхувати воду?

Пори кераміки забиваються мінеральними відкладеннями (водяний камінь). Нейтральний шампунь їх не бере. Використовуйте кислотний шампунь для авто (pH 4–6) раз на місяць — він розчиняє відкладення і повертає гідрофобні властивості.

### У чому різниця між губкою та рукавицею?

Щільна губка з супермаркету гонить пісок по поверхні. Мікрофіброва рукавиця з довгим ворсом **затягує абразив усередину волокон**, не даючи йому контактувати з лаком. Результат — менше подряпин після кожного миття.

### Скільки шампуню лити у відро?

Дозування залежить від концентрації — дивіться інструкцію виробника. Більше — не означає чистіше. Занадто багато шампуню залишає плівку на кузові та ускладнює сушіння.
