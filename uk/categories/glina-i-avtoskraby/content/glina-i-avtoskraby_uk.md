# Глина та автоскраби

Глина для авто видаляє вкраплення, які не змиваються звичайним шампунем: металевий пил, смолу, бітум, пилок дерев. Якщо провести долонею по вимитому кузову і відчути шорсткість — вам потрібна автомобільна глина або автоскраб. Після такого очищення поверхня стає ідеально гладкою, а захисні покриття тримаються значно довше.

## Як обрати глину для авто

| Тип | Абразивність | Для чого | Для кого |
|-----|--------------|----------|----------|
| Синя глина (Fine) | Низька | Легкі забруднення, регулярний догляд | Новачки, свіжий лак |
| Жовта/біла (Medium) | Середня | Помірні вкраплення | Авто 2-5 років |
| Автоскраб | Низька-середня | Швидке очищення без розминання | Професіонали, потокова робота |

**Сценарії вибору:**

- **Якщо авто нове або регулярно доглядаєте** → синя глина для авто — м'яка, безпечна для тонкого лаку
- **Якщо багато бітуму та промислового нальоту** → жовта або біла глина з більшою абразивністю
- **Якщо працюєте з потоком машин** → автоскраб — не потребує розминання, служить на 50-100 авто

### Глина для кузова чи автоскраб — що обрати

Класичний брусок дешевший і дає повний контроль над процесом. Але його потрібно постійно розминати, щоб чиста сторона контактувала з лаком. Якщо маса впала на підлогу — викидайте.

Автоскраб (clay bar у вигляді рукавички або пада) довговічніший: один скраб замінює 10-20 брусків глини. Він зручніший для великих панелей, але коштує дорожче.

## Глина для авто — на що звернути увагу

| Маркер | Що означає | Рекомендація |
|--------|------------|--------------|
| Fine / Light | Низька абразивність | Для регулярного догляду |
| Medium | Середня абразивність | Для запущених авто |
| Heavy | Висока абразивність | Тільки перед полірування |
| Reusable | Багаторазовий | Автоскраби з можливістю промивання |

## Особливості використання

Абразивна маса працює тільки з лубрикантом — спеціальним спреєм або розчином шампуню у воді. Без змащування брусок залипає і залишає сліди.

Рухи мають бути лінійними: вперед-назад або вліво-вправо. Кругові рухи при потраплянні піщинки створять "завитки", які складно виполірувати.

## FAQ

### Скільки бруска потрібно на одне авто?

Шматка 100 г вистачає на 3-5 повних очищень. Якщо працюєте регулярно — обирайте автоскраб, який служить на 50-100 машин.

### Чи можна використовувати clay bar на склі?

Так, детейлінг-брусок ефективно прибирає мікро-наліт з лобового скла. Після цього двірники працюють тихіше і без пропусків.

### Як перевірити чи потрібне очищення?

Проведіть рукою по вимитому кузову через тонкий пластиковий пакет — він підсилить тактильні відчуття. Якщо відчуваєте шорсткість — потрібна глина для авто.

### Що робити якщо брусок впав на підлогу?

Викинути. Пісок та дрібні частинки перетворять його на наждак, який пошкодить лак.

---

**Підсумок:**
- Якщо регулярний догляд — синя глина для авто
- Якщо запущений кузов із вкрапленнями — жовта або біла глина
- Якщо потокова робота — автоскраб для швидкості та економії
