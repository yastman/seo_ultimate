# Очищувач дисків

Очищувач дисків — спеціалізована хімія для коліс, яка розчиняє гальмівний пил, нагар та бітум без механічного впливу. Засоби для чистки дисків працюють за рахунок хімічної реакції з металевими частинками. Головне правило: кислота — це потужно, але небезпечно. Якщо у вас дорогі диски або карбон-кераміка — тільки нейтральні склади. Якщо старі литі — можна кислоту, але без HF і з рясним змиванням.

## Види хімії для дисків

| Тип складу | pH | Призначення | Обмеження |
|------------|-----|-------------|-----------|
| Нейтральний | 6–8 | Регулярне миття, легкий пил | Немає |
| Iron Remover | 6–8 | Гальмівний пил, метал | Немає |
| Лужний | 10–12 | Жир, масло, дорожня плівка | **Заборонено для керамічних гальм, анодування** |
| Кислотний | 1–3 | Застарілий нагар | **Заборонено для хрому, полір. алюмінію, PCCB, анодування** |

## Чек-лист покупця: що потрібно крім банки

### Інструменти — різні для дисків і гуми

**Для чорного глянцю (Black Gloss):** Жорсткі йоржики (типу EZ Detail) дряпають глянець миттєво — «павутина» з'являється після першого миття. Рішення: тільки Wheel Woolies (вовняні «палички») або мікрофіброва рукавиця.

**Для звичайних литих:** Щітки середньої жорсткості, йоржики для спиць і болтів.

**Для гуми:** Тут навпаки — потрібна жорстка щітка. М'яка просто гладить бруд, не вибиваючи його з пор шини.

### Хімія «2-в-1» — лужні склади

Tire & Wheel Cleaner (луг pH 10–12) — універсальний варіант для тих, хто не хоче дві окремі банки.

- **Плюс:** Ідеально миє гуму (витягує коричневу піну/антиозонанти) і підходить для регулярного миття дисків
- **Мінус:** Якщо диски занедбані (жовтий нагар роками), «2-в-1» не впорається — потрібна кислота

### Фінішний захист — економія на майбутнє

Нанесення кераміки для дисків (Rim Coat) або силанту після глибокого очищення дозволяє в майбутньому мити диски просто водою і безконтактним шампунем. Гальмівний пил не в'їдається, змивається струменем води. **Це заощадить бюджет на хімії в наступному сезоні.**

## Як обрати засіб для чистки дисків

**Литі диски, регулярний догляд** → нейтральний засіб для дисків або Iron Remover.

**В'їлий гальмівний пил (руді точки)** → очищувач колісних дисків з індикатором.

**Товстий шар нагару** → кислотна хімія для чистки дисків + рясне змивання.

**Хромовані диски** → тільки нейтральні засоби для миття дисків.

**Полірований алюміній** → нейтральна рідина для чистки дисків.

**Карбон-кераміка (PCCB), анодовані диски, Center Lock** → **ТІЛЬКИ** нейтральний шампунь. Кислота і луг знищать покриття.

**Литі з лакуванням** → засіб для миття литих дисків без кислоти.

## Як застосовувати

1. **Охолодіть диски!** Температура має бути нижче +30°C. Перевіряйте не шину, а гальмівний диск (обережно рукою на відстані).
2. Змийте основний бруд водою.
3. Нанесіть засіб для миття дисків, починаючи з внутрішньої сторони.
4. Витримайте 2–5 хвилин, не даючи висохнути.
5. Пройдіться щіткою по спицях і болтах.
6. **Рясно пролийте водою** супорти, болти і ніпелі — хімія не повинна залишитися в прихованих порожнинах.

**Pro tip:** Нанесення через піннік або піногенератор подовжує цикл роботи складу — піна довше тримається на поверхні і не стікає, хімія встигає розчинити більше нагару.

## ⚠️ Критичні помилки

**Нанесення на гарячі диски** — хімія для дисків висихає миттєво, залишаючи незмивні розводи.
**Наслідок:** Супорти Brembo можуть поблідшати від хімічного опіку — яскравий лак стане матовим.
**Рішення:** Охолодіть диски до температури нижче +30°C.

**Кислота на хромі або PCCB** — незворотні матові плями.
**Наслідок:** Карбон-керамічні гальма коштують сотні тисяч — їх вбиває будь-яка агресивна хімія.
**Рішення:** Для дорогих дисків — тільки нейтральні склади (pH 6–8).

**Кислота на дисках зі сколами** — якщо є «бордюрка», кислота проникає ПІД лак через скол і піднімає покриття зсередини.
**Наслідок:** Лак почне злазити павутинкою за сезон.
**Рішення:** Перед агресивною хімією огляньте диски на пошкодження.

**Потрапляння лугу на гуму** — викликає Tire Blooming (порудіння шин).
**Наслідок:** Луг вимиває антиозонанти з гуми.
**Рішення:** Змивайте склад з шин негайно.

**Flash Rust після миття** — якщо помили диски Iron Remover і залишили машину мокрою на ніч, гальма можуть «прикипіти».
**Наслідок:** Вібрація та биття в кермо.
**Рішення:** Після миття проїдьте 100 метрів і пару разів різко загальмуйте.

**Використання засобів з HF** — плавикова кислота небезпечна для життя.
**Наслідок:** Хімічні опіки, небезпека для здоров'я.
**Рішення:** Перевіряйте склад перед покупкою — уникайте HF.

## Підсумок вибору

- **Регулярний догляд:** Нейтральний Iron Remover — безпечний для будь-якого покриття
- **Товстий нагар:** Кислотний очищувач дисків 1 л (без HF!) + рясне змивання водою
- **Хром, полірування, PCCB:** Тільки нейтральна рідина для чистки дисків
- **На потоці:** Концентрат — дешевше за літр, але потрібна тара і чиста вода для розведення
- **Інструмент:** Для чорного глянцю — тільки мікрофібра/шерсть; для гуми — жорстка щітка
- **Захист:** Кераміка або силант для дисків — щоб мити рідше і економити на хімії

## FAQ: Відповіді експерта

### Чому Iron Remover змінює колір і пахне тухлими яйцями?

Фіолетовий колір — реакція на іони заліза (гальмівний пил). Запах сірки — норма для якісного складу. Працюйте надворі.

### Гальмівний пил в'ївся намертво. Iron Remover не допомагає?

Iron Remover прибирає тільки поверхневий пил. Запечений нагар (жовті кути спиць) знімається тільки кислотою — локально на плями, потерли 30 сек, рясно змили.

### Очищувач зіпсував ковпачки (Center Caps). Чому?

Ковпачки часто з анодованого алюмінію — луг/кислота роз'їдає покриття. Знімайте перед агресивною хімією або використовуйте pH-нейтральний шампунь.

### Чи можна мити диски взимку одразу після траси?

Ні! Гарячі диски + крижана вода = деформація гальмівного диска (биття в кермо). Дайте охолонути 5–10 хвилин.
