# Полірувальна машинка для авто

Полірувальна машинка для авто замінює ручну роботу при видаленні подряпин, голограм та матовості лаку. Роторна полірувальна машинка дає максимальну абразивність — для глибоких дефектів, ексцентрикова безпечніша для новачків і підходить для фінішного полірування. Правильний вибір залежить від завдання: корекція чи підтримка блиску.

## Типи полірувальних машинок для авто

| Тип | Принцип роботи | Для кого | Коли обирати |
|-----|----------------|----------|--------------|
| Роторна | Диск обертається навколо осі | Професіонали | Глибокі подряпини, корекція лаку |
| Ексцентрикова (DA) | Диск обертається + коливається | Новачки та професіонали | Фінішне полірування, безпечна робота |
| Орбітальна | Тільки коливання, без обертання | Для нанесення восків | Розполіровка складів, не корекція |

**Роторна vs ексцентрикова:**

- **Роторна** — агресивна, швидко знімає лак, ризик перегріву та пропалів. Потребує досвіду
- **Ексцентрикова** — м'якша робота, складніше пошкодити лак, але повільніша при корекції

## Характеристики полірувальної машинки для авто

| Параметр | Що означає | Рекомендація |
|----------|------------|--------------|
| Хід ексцентрика | Амплітуда коливань (мм) | 8-21 мм. Більший хід — швидша робота |
| Потужність | Ватт | 500-1200 Вт для легкових авто |
| Оберти | RPM (обертів за хвилину) | Регулювання 1000-6000 RPM |
| Діаметр диска | 75-180 мм | 125 мм — універсальний, 75 мм — важкодоступні місця |
| Вага | Кг | 1.5-3 кг — комфорт при тривалій роботі |

**Оптимальний вибір для початківця** — ексцентрикова полірувальна машинка з ходом 12-15 мм, регулюванням обертів, діаметром 125 мм.

## Що потрібно крім машинки

| Витратник | Призначення | Кількість |
|-----------|-------------|-----------|
| Полірувальні круги | Різна жорсткість для різних етапів | Мінімум 3-4 шт різних |
| Полірувальні пасти | Абразивні та фінішні | 2-3 види |
| Мікрофібра | Видалення залишків пасти | 5-10 шт |
| Знежирювач | Підготовка та контроль результату | 1 флакон |
| Малярний скотч | Захист пластику та гуми | 1-2 рулони |

## FAQ

### Яка полірувальна машинка краща для новачка?

Ексцентрикова (Dual Action). Безпечніша — складніше пропалити лак через особливості руху диска. Роторна вимагає досвіду контролю температури та тиску.

### Чи можна полірувати звичайною болгаркою?

Технічно можливо з адаптером, але небезпечно. Болгарка не має регулювання обертів, занадто агресивна — високий ризик пропалів та пошкодження лаку.

### Скільки коштує базовий набір для полірування?

Полірувальна машинка для авто початкового рівня + набір кругів + пасти — орієнтовно від 3000-5000 грн. Професійне обладнання суттєво дорожче.
