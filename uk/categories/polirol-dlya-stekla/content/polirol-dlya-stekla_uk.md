# Поліроль для скла

Поліроль для скла авто — абразивний або хімічний засіб для видалення подряпин, водного каменю та помутніння з лобового та бокового скла. Правильно підібрана паста для скла відновлює оптичну прозорість без пошкодження поверхні.

## Як обрати поліроль для скла авто

| Тип | Абразивність | Для кого |
|-----|--------------|----------|
| Паста з оксидом церію | Середня | Глибокі подряпини, водний камінь |
| Паста з оксидом заліза | М'яка | Легке помутніння, фінішна обробка |
| Хімічний розчинник | Без абразиву | Мінеральні відкладення без шліфування |

**Сценарії вибору:**
- **Якщо подряпини відчуваються нігтем** → паста з оксидом церію (cerium oxide)
- **Якщо скло просто помутніло** → м'яка паста або хімічний очищувач
- **Якщо білий наліт від води** → спочатку хімія для видалення солей, потім полірування

## Особливості роботи з поліроллю для скла

Засіб для полірування лобового скла працює за принципом контрольованого зняття мікрошару — абразивні частинки оксиду церію вирівнюють поверхню на рівні подряпини. Робота виконується полірувальною машинкою з повстяним кругом на низьких обертах (1000–1500 об/хв).

**Якщо полірування вручну** → процес займає значно більше часу і підходить лише для локальних дефектів.

## Що впливає на результат

| Фактор | Вплив |
|--------|-------|
| Глибина подряпин | Глибокі (>0.1мм) можуть не піддаватися |
| Температура скла | Перегрів = ризик тріщини |
| Тип круга | Повсть або фетр для скла |

## FAQ

### Чи можна видалити глибокі подряпини поліроллю?

Поліроль для скла авто ефективна для поверхневих пошкоджень глибиною до 0.05–0.1 мм. Глибші подряпини, які відчуваються нігтем, можуть стати менш помітними, але повністю не зникнуть — у такому випадку розглядайте заміну скла.

### Чи підходить для полірування фар?

Ні, для фар потрібні окремі засоби — полікарбонат оптики має іншу твердість і вимагає інших абразивів. Паста для скла може пошкодити пластикові фари.

### Як часто можна полірувати лобове скло?

**Якщо скло в нормальному стані** → полірування раз на 2–3 роки для видалення накопичених мікроподряпин. Часте полірування зменшує товщину скла.

---

**Підсумок:**
- Якщо подряпини від щіток: паста з оксидом церію для машинного полірування
- Якщо водний камінь: спочатку хімічний розчинник, потім полірування
