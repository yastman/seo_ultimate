# Автошампунь

Автошампунь — базовий засіб для контактного миття кузова. Якісний шампунь для авто видаляє дорожній бруд, пил та органіку, не пошкоджуючи лакофарбове покриття та зберігаючи захисні шари воску чи кераміки.

## Як обрати автошампунь для миття

| Тип | pH | Особливості | Для кого |
|-----|-----|-------------|----------|
| Нейтральний | 6-8 | Безпечний для захисту | Авто з воском, керамікою |
| Кислотний | 3-5 | Видаляє мінеральні нальоти | Підготовка до полірування |
| Лужний | 9-11 | Сильне знежирення | Перед нанесенням захисту |
| Керамічний | 6-8 | Додає гідрофобність | Підтримка кераміки |

### Сценарії вибору

**Якщо авто з керамічним покриттям** — тільки pH-нейтральний або спеціалізований керамічний шампунь для авто. Кислотні та лужні засоби руйнують захист.

**Якщо миєте перед поліруванням** — знежирювальний склад. Видалить старий віск та забезпечить чисту базу для роботи.

**Якщо регулярне обслуговування** — нейтральний концентрат. Економічний, безпечний, підходить для частого застосування.

## Автошампунь: концентрат vs готовий

| Формат | Переваги | Недоліки |
|--------|----------|----------|
| Концентрат | Економія, гнучкість розведення | Потрібно готувати розчин |
| Готовий | Зручність, стабільна концентрація | Дорожче за літр миття |

Для автомийок та професійного використання — концентрати 1:100 або 1:200. Для домашнього догляду — готові розчини або концентрати 1:50.

## На що дивитися на етикетці

| Маркування | Що означає |
|------------|------------|
| pH-neutral | Безпечний для всіх покриттів |
| Wax safe | Не змиває захист |
| High foam | Багато піни для ручного миття |
| Low foam | Для апаратів високого тиску |

## FAQ

### Який шампунь для автомийки обрати?

Для мійок високого тиску — концентрат з низьким піноутворенням (Low foam). Шампунь для мійок високого тиску працює через хімічну активність, а не через піну.

### Чи можна мити посудним засобом?

Технічно так, але не рекомендовано. Побутові засоби занадто агресивні — змивають захисні покриття та можуть залишати плівку. Автомобільний шампунь розроблений саме для ЛФП.

### Скільки шампуню потрібно на одне миття?

Залежить від концентрації. Зазвичай 30-50 мл концентрату на відро (15-20 л). Для машини середнього розміру достатньо одного відра розчину.

---

**Підсумок:** Якщо авто з захисним покриттям — pH-нейтральний автошампунь. Для підготовки до полірування — знежирювальний. Концентрати економніші, готові — зручніші.
