# UK Localization

**[← Назад в корень](../README.md)**

Папка для украинских версий контента.
Структура зеркальна основной папке `categories/`.

## Структура

`uk/categories/{slug}/`

* `content/{slug}_uk.md`: Текст на украинском языке.
* `meta/{slug}_meta.json`: Мета-теги на украинском.
* `data/{slug}_clean.json`: (Опционально) Ключи на украинском, если проводился отдельный сбор семантики. В большинстве случаев используется маппинг с RU ключей.

## Процесс создания

1. Завершается работа над RU версией (Status: Content Ready).
2. Запускается скрипт `scripts/uk_seed_from_ru.py` (или через команду `/uk-content-init`), который создает структуру папок.
3. Генерируется контент (перевод + адаптация) и сохраняется в `_uk.md`.
4. Проверяется валидатором `scripts/validate_uk.py`.
