# Полировка авто

Полировка автомобиля восстанавливает блеск и убирает дефекты лакокрасочного покрытия. Профессиональные средства для полировки авто работают предсказуемо, сохраняют толщину лака и дают стабильный результат. Правильный выбор оборудования и расходников экономит время, защищает покрытие от пробития до грунта.

## Оборудование для полировки кузова

Основа системы — полировальная машинка. Ротационные дают максимальную коррекцию, требуют опыта и контроля давления. Эксцентриковые (dual action) прощают ошибки, безопаснее для новичков. Аккумуляторные модели удобны для локальной обработки, труднодоступных зон, не требуют розетки.

**Если первый опыт полировки машины** → эксцентриковая машинка с ходом 8-12 мм.
**Если глубокая коррекция лака** → ротационная с регулировкой оборотов.
**Если работа без розетки** → аккумуляторная модель от 18V.

## Расходные материалы: круги и пасты

Полировальные круги различаются по жёсткости, типу материала. Жёсткие снимают дефекты, мягкие выводят блеск. Пасты подбираются под задачу: абразивные составы для коррекции царапин, финишные для доводки блеска, универсальные для одноэтапной обработки.

**Если средние царапины на кузове** → жёсткий круг + абразивная паста.
**Если блеск без дефектов** → мягкий круг + финишная полировка.
**Если чёрный цвет авто** → финишный состав против голограмм.

## Вспомогательные инструменты

Толщиномер ЛКП (PTG) измеряет остаточную толщину лака — защита от пробития до грунта. Наборы деконтаминации (глина, удалитель металлических вкраплений) убирают загрязнения перед обработкой, критически важны для результата. Аппликаторы, микрофибра обеспечивают чистое нанесение, удаление остатков без царапин.

## Бытовой vs профессиональный подход

| Бытовой | Профессиональный |
|---------|------------------|
| Обработка без измерения толщины | Толщиномер перед работой |
| Один пад на весь кузов | Разные насадки под задачу |
| Работа на солнце | Работа в тени, комнатная температура |
| Пропуск деконтаминации | Удалитель металла → глина → коррекция |
| Один состав | Система: абразив + финиш |

Профессиональный подход требует больше времени, гарантирует безопасность покрытия, предсказуемый результат.

## FAQ

### Что нужно для полировки кузова?

Базовый набор: полировальная машинка, насадки разной жёсткости, пасты для коррекции и финиша. Профессиональный результат требует толщиномера, набора деконтаминации — минимизирует риск пробить лак.

### Какие пасты и круги выбрать?

Зависит от состояния ЛКП, опыта. Новичкам подходят универсальные составы с мягкими падами — безопасно, хороший результат. Глубокая коррекция требует системы: абразивная паста, затем финишная полировка.

### Чем отличается ротационная машинка от эксцентриковой?

Ротационная вращается по кругу, даёт максимальную коррекцию, может оставить голограммы при неправильной технике. Эксцентриковая (dual action) совмещает вращение с колебаниями — безопаснее для новичков, прощает ошибки давления.

### Можно ли полировать авто без опыта?

Да, если использовать эксцентриковую машинку, мягкие круги, универсальные пасты. Главное — работать на холодной поверхности, избегать рёбер и кромок, не давить на инструмент. Толщиномер покажет запас лака перед началом работы.

---

**Итог — с чего начать:**
- **Диагностика** → толщиномер для оценки запаса лака
- **Выбор машинки** → эксцентриковая для первого опыта, ротационная для профи
- **Подбор расходников** → круги и пасты под задачу, тип краски
