# Research: Очистители стекол (ochistiteli-stekol)

## 1. Базовая матрица поиска

| Данные | Результат |
| :--- | :--- |
| **Что это** | Составы для удаления жирового налета, следов насекомых и табачного конденсата. |
| **Для чего** | Обеспечение прозрачности без разводов и бликов. Подготовка перед "антидождем". |
| **Классификация** | **Спиртовые:** быстрые. **Без спирта:** для тонировки и экранов. **Пенные:** анти-стекание. |
| **Расход** | **10-15 мл** готового состава на все стекла седана. |
| **Инструкция** | 1. Нанести на фибру. 2. Растереть по поверхности. 3. Располировать сухой стороной. |
| **Совместимость** | Безопасны для ЛКП. Составы с аммиаком опасны для тонировки! |
| **Ошибки** | Грязные тряпки, работа на солнце, использование аммиака на тонировке. |
| **Критерии выбора** | Отсутствие аммиака, скорость испарения, антистатический эффект. |

## 2. Углубление для детейлинга (E-E-A-T)

* **Расход:** Очень малый. 12-15 мл готового состава на все стекла седана. Флакона 500 мл хватает на 30-40 чисток.
* **Оборудование:** Вафельное полотенце (Glass Towel) или коротковорсная микрофибра, Вторая сухая микрофибра для финиша.
* **Safety / Ограничения:** Для тонированных стекол использовать только "Tint Safe" составы без аммиака. Не распылять прямо на приборную панель.
* **Эффективность:** Удаляет "дымку" от пластика и сигарет, которую трудно убрать обычной водой.

## 3. Полезные факты и лайфхаки

* **Факт:** Грязное стекло изнутри — это часто результат "испарений" (Outgassing) пластика приборной панели на солнце. Это жирный налет, который требует качественных ПАВ или спирта.
* **Лайфхак (Two-Towel Method):** Используйте две фибры. Одной (влажной от состава) моем поверхность, а **второй (абсолютно сухой)** СРАЗУ ЖЕ располировываем. Если ждать, пока состав высохнет сам — разводы неизбежны.
* **Лайфхак (Directional Trick):** Чтобы легко определить, с какой стороны стекла остались разводы, вытирайте внешнюю сторону — **горизонтально**, а внутреннюю — **вертикально**. Если видишь полосу — по ее направлению сразу понятно, где нужно дотереть.
* **Лайфхак:** Опустите стекло на 1-2 см. Протрите открывшуюся верхнюю кромку (edge), где всегда скапливается полоска грязи, которую не видно в закрытом состоянии.
* **Гайд (микро):** Всегда распыляйте очиститель на фибру, а не на само стекло (особенно изнутри), чтобы химия не попала на приборную панель или обивку.

## 4. Меры предосторожности

* **Тонировка:** Никогда не используйте очистители с аммиаком (Ammonia) на тонированных стеклах. Аммиак разрушает структуру пленки и клей.
* **Экраны:** Не используйте обычные спиртовые очистители для iPad/Tesla экранов. Используйте только специальные деликатные составы для электроники.
