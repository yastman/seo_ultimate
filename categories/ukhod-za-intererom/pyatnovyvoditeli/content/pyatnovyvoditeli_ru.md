# Пятновыводитель

Пятновыводитель для автомобиля — специализированный состав для удаления пятен с обивки салона. Выбор определяется типом загрязнения. Белковые следы (кровь, молоко) требуют ферментов и холодной воды, жировые пятна — щелочных составов, универсальные энзимные продукты подходят для большинства органических загрязнений. Неправильный механизм закрепит пятно вместо удаления.

## Как выбрать пятновыводитель для салона авто: по типу пятна

| Пятно | Что выбирать | Почему именно это |
|-------|--------------|-------------------|
| Кровь, молоко, рвота | Энзимный (protease) | Ферменты расщепляют белок, а не "сваривают" его |
| Кофе, чай, вино | Энзимный + кислотный | Кислота нейтрализует танины, ферменты убирают след |
| Масло, жир, еда | Щелочной (pH 8-10) | Саванификация превращает жир в растворимое мыло |
| Старые въевшиеся | Концентрат с энзимами | Длительная выдержка проникает глубже в волокна |

**Если вы столкнулись с кровью или молоком** — используйте только холодную воду. Горячая сваривает белок и делает удаление невозможным.

**Если пятно жировое** — допустима тёплая вода, она ускоряет растворение масел и улучшает результат.

**Если тип загрязнения неизвестен** — берите ферментный состав: он безопасен для большинства органических следов и не рискует закрепить пятно.

## На что смотреть при выборе

| Параметр | Что означает | Кому подходит |
|----------|--------------|---------------|
| Механизм действия | Ферментный разлагает органику; ПАВ-щелочной растворяет жир | Энзимы — для универсального применения, щелочь — для жира |
| pH состава | Нейтральный (6-8) безопасен для велюра и кожи; щелочной (8-10) только для прочной синтетики | Нейтральный — если не уверены в материале |
| Форма выпуска | Спрей готовый к применению удобен для точечной работы; концентрат для глубокой химчистки экстрактором | Спрей — новичкам, концентрат — профи |
| Время выдержки (dwell time) | Энзимный: 15-30 мин для свежих, 30-60 мин для застарелых; щелочной: 5-15 мин | Чем старше пятно — тем дольше держите состав |

**Перед первым применением** — тестируйте состав на скрытом участке (под сиденьем, край обивки). Дайте высохнуть и проверьте реакцию ткани через 24 часа.

**Важно: промокайте, не трите.** Трение заталкивает загрязнение глубже в волокна и расширяет пятно. Прикладывайте салфетку с легким давлением, меняя её по мере впитывания.

## По типу материала обивки

| Материал | Что вам подходит | Чего избегать |
|----------|------------------|---------------|
| Ткань (полиэстер) | Ферментный или щелочной с обязательным смыванием экстрактором | Оставлять без полоскания — ПАВ притянет грязь |
| Велюр | Только нейтральный pH, ферментный очиститель | Тереть щёткой — потеряете ворс |
| Ковролин | Ферментный с работой экстрактором | Растворители отклеивают клеевую основу |
| Кожа | Специализированный для кожи (нейтральный pH 6-7) | Щелочной сушит и вызывает растрескивание |

**Если у вас велюр или алькантара** — выбирайте только мягкие составы с нейтральным pH. Агрессивная щелочь разрушает деликатные волокна.

**Если кожа** — после очистки обязательно наносите кондиционер. Без него материал пересыхает и теряет эластичность.

## Что такое энзимный очиститель

**Энзимный очиститель** — ферменты (протеаза, липаза, амилаза), которые расщепляют органические молекулы: белки, жиры, крахмалы. В отличие от бытовых средств, не маскируют запах, а удаляют его источник на молекулярном уровне. Профессионалы используют энзимные составы для удаления запахов домашних животных, рвоты, пролитого молока — там, где бытовая химия только смывает видимый след.

## FAQ

### Какой пятновыводитель для сидений авто лучше выбрать?

Зависит от типа загрязнения. Для органических следов (кофе, кровь, еда) берите ферментный очиститель с протеазой и липазой. Для жировых — щелочной ПАВ-состав. Если не уверены в природе пятна, универсальный вариант — ферментный: он безопаснее для всех типов обивки.

### Можно ли использовать один состав для всех типов обивки?

Универсальный пятновыводитель для авто с нейтральным pH подходит для ткани, велюра и кожи. Но для сложных загрязнений на прочной синтетике (застарелый жир, масло) эффективнее специализированный щелочной состав. Для деликатных материалов рискуйте только нейтральными средствами.

### Какой пятновыводитель взять для застарелых пятен?

Энзимный концентрат с увеличенным временем выдержки. Спреи готовые к применению рассчитаны на свежие следы и поверхностную работу. Для въевшихся загрязнений нужен концентрат и обработка экстрактором — только так состав проникает в глубину волокон.

### Нужно ли смывать пятновыводитель после использования?

Да, особенно если используете щелочной или ПАВ-состав. Остатки средства притягивают грязь из воздуха, и через пару недель загрязнение возвращается (эффект впитывания). Ферментные составы менее критичны, но профессионалы всё равно рекомендуют финальное полоскание экстрактором.

---

**Итог — что вам покупать:**
- **Кровь, молоко, органика** → энзимный пятновыводитель
- **Жир, масло, еда** → щелочной ПАВ-очиститель
- **Велюр, кожа, деликатные материалы** → только нейтральный pH
- **Глубокая химчистка, застарелые пятна** → концентрат для экстрактора
