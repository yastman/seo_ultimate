# Пятновыводители

Пятновыводитель для авто удаляет загрязнения с обивки салона — ткани, велюра, ковролина. Главный критерий выбора — тип загрязнения: белковые требуют энзимов, жировые — щелочных составов. Неправильный выбор закрепляет след навсегда.

## Как выбрать: по типу пятна

| Пятно | Что брать | Почему |
|-------|-----------|--------|
| Кровь, молоко, рвота | Энзимный (protease) | Ферменты разлагают белок, а не закрепляют |
| Кофе, чай, вино | Энзимный + кислотный | Кислота нейтрализует танины |
| Масло, жир, еда | Щелочной (pH 8-10) | Саванификация превращает жир в мыло |
| Старые въевшиеся | Энзимный концентрат | Время выдержки 30-60 минут вместо 15 |

**Если загрязнение белковое (кровь, молоко)** — только холодная вода. Горячая «сваривает» белок и закрепляет его в волокнах.

**Если след жировой** — допустима тёплая вода, она ускоряет растворение масел.

**Если тип загрязнения неизвестен** — ферментный состав безопаснее, работает на большинстве органики.

## На что смотреть при выборе

| Параметр | Что означает |
|----------|--------------|
| Тип механизма | Ферментный — для органики и запаха; ПАВ-щелочной — для жира |
| pH состава | Нейтральный (6-8) безопасен для велюра и кожи; щелочной (8-10) только для синтетики |
| Форма выпуска | Спрей RTU — для свежих загрязнений; концентрат — для глубокой химчистки |
| Время выдержки (dwell time) | 15-20 минут для свежих, 30-60 минут для застарелых |

## Выбор по материалу обивки

| Материал | Что подходит | Чего избегать |
|----------|--------------|---------------|
| Ткань (полиэстер) | Энзимный или щелочной с полосканием | Оставлять без смывания |
| Велюр | Только нейтральный pH, ферментный | Тереть — потеря ворса |
| Ковролин | Ферментный, работа экстрактором | Растворители — отклеивают основу |
| Кожа | Специальный для кожи (нейтральный pH) | Щелочной — сушит и трескает |

**Если велюр или алькантара** — только мягкие составы. Щелочь разрушает деликатные волокна.

**Если кожа** — после очистки обязательно кондиционирование, иначе пересыхает.

## Чего избегать при выборе

**Бытовой очиститель** — риск высокого pH. Бытовые средства часто щелочные (pH 10+), повреждают ткань салона.

**Растворитель для жвачки/битума** — растворяет клей ковролина. Безопаснее масло эвкалипта или заморозка CO₂.

**Без смывания** — остаток ПАВ притягивает грязь. Через неделю загрязнение «вернётся» (wicking effect).

**Без теста на скрытом участке** — агрессивный состав может обесцветить краску на дешёвой коже или пластике. Проверять под сиденьем, ждать 24 часа.

## FAQ

### Какой пятновыводитель для авто лучше?

Зависит от типа загрязнения. Для органических (кофе, кровь, еда) — энзимный очиститель с протеазой и липазой. Для жировых — щелочной ПАВ. Универсальный вариант — ферментный, безопаснее для всех типов обивки.

### Пятновыводитель для ткани или кожи — в чём разница?

Для ткани допустим щелочной pH (8-10) с обязательным смыванием. Для кожи — только нейтральный pH (6-7), иначе материал сохнет и трескается. После кожи нужен кондиционер.

### Какой пятновыводитель взять для застарелых загрязнений?

Энзимный концентрат с увеличенным временем выдержки (30-60 минут). Спрей RTU рассчитан на свежие следы (15-20 минут). Для въевшихся — только концентрат и работа экстрактором.

---

**Итог — что покупать:**
- **Кровь, молоко, органика** → энзимный пятновыводитель
- **Жир, масло** → щелочной ПАВ-очиститель
- **Велюр, кожа** → только нейтральный pH
- **Глубокая химчистка** → концентрат для экстрактора
