## Промпт для исследования (11 блоков)

### Блок 1: Почему выбор аксессуара критичен

1. Как неправильная губка создает царапины (swirl marks) на лаке.
2. Принцип "удержания грязи" (dirt trapping).

### Блок 2: Виды и типы (3 независимые оси)

> ⚠️ **Важно:** Форма и материал — разные параметры. Микрофибровая варежка и микрофибровая губка — это разные инструменты.

**Ось 1 — Форма / конструкция:**

| Тип           | Описание                 | Плюсы                      | Минусы          |
| ------------- | ------------------------ | -------------------------- | --------------- |
| Губка         | Квадратная/прямоугольная | Удобно держать             | Меньше площади  |
| Варежка       | Надевается на руку       | Контроль, чувствительность | Намокает рука   |
| Рукавица      | Большая площадь          | Быстрее мойка              | Тяжелее мокрая  |
| Двухсторонняя | Разные материалы         | Универсальность            | Сложнее в уходе |

**Ось 2 — Материал:**

| Тип                    | Описание              | Когда выбирать      |
| ---------------------- | --------------------- | ------------------- |
| Микрофибра             | Синтетические волокна | Универсальная мойка |
| Шенилл (Chenille)      | "Макароны"            | Грязные авто        |
| Шерсть (Lambswool)     | Натуральная           | Деликатная мойка    |
| Крупнопористый поролон | Ячеистая структура    | Ополаскивание       |

**Ось 3 — Назначение:**

| Тип             | Описание           | Когда выбирать      |
| --------------- | ------------------ | ------------------- |
| Для кузова      | Мягкая, безопасная | Основная мойка      |
| Для дисков/арок | Жёсткая            | Стойкие загрязнения |
| Для стёкол      | Без ворса          | Финальная протирка  |
| Универсальная   | Средняя жёсткость  | Нет отдельных       |

### Блок 3: Как выбрать (Buyer's Guide)

1. **Безопасность:** Почему "кирпич" из поролона за 20 грн — это смерть для лака (песок не уходит в поры)?
2. **Удобство:** Рукавица vs Варежка с пальцами — что удобнее, чтобы не уронить на землю?
3. **Грязь:** Шенилл ("макароны") или Микрофибра — что лучше вытягивает песок при мойке джипа?
4. **Натуралка:** Стоит ли переплачивать за овечью шерсть (Lambswool) или за ней слишком сложно ухаживать (гниет)?
5. **Колеса:** Почему для дисков нужна отдельная варежка и почему нельзя брать старую кузовную?
