# Губки и варежки

Губка для мойки авто — основной инструмент контактной мойки. Неправильный выбор приводит к царапинам: дешёвый поролон перетаскивает грязь по лаку вместо того, чтобы её удерживать. Микрофибра или шерсть захватывают частицы в волокна и защищают покрытие.

## Материал: что безопасно для лака

| Материал              | Захват грязи                 | Риск царапин | Срок службы    |
| --------------------- | ---------------------------- | ------------ | -------------- |
| Микрофибра (chenille) | Удерживает в волокнах        | Низкий       | 300–500 стирок |
| Шерсть (lambswool)    | Удерживает, мягче всего      | Минимальный  | 150–250 стирок |
| Поролон               | Не удерживает, перетаскивает | Высокий      | Короткий       |

**Важно:** Поролоновая мочалка для авто подходит только для колёс и арок — на кузове оставляет микроцарапины.

### Подбор под тип лакокрасочного покрытия

| Тип ЛКП                                       | Рекомендация                       | Почему                                                               |
| :-------------------------------------------- | :--------------------------------- | :------------------------------------------------------------------- |
| **Мягкий лак** (японцы: Toyota, Mazda, Honda) | **Натуральная шерсть** (lambswool) | Самый мягкий ворс, скользит без трения. Минимальный риск «паутинки». |
| **Твёрдый лак** (немцы: BMW, Audi, Mercedes)  | **Микрофибра** (chenille)          | «Червячки» отлично удерживают песок. Износостойкая, служит годами.   |
| **Диски, пороги, арки**                       | **Отдельная губка**                | Никогда не используйте варежку для кузова на колёсах!                |

**Если лак чувствительный** → только шерсть.
**Если важна долговечность инструмента** → микрофибровая варежка.

## Форма: губка, варежка, рукавица

| Форма                  | Особенность                 | Когда выбирать            |
| ---------------------- | --------------------------- | ------------------------- |
| Губка                  | Компактная, удобно держать  | Небольшие участки, детали |
| Варежка (с пальцами)   | Контроль и чувствительность | Детальная работа, изгибы  |
| Рукавица (без пальцев) | Большая площадь покрытия    | Быстрая обработка панелей |

**Рекомендация:** Для регулярной мойки — рукавица из микрофибры. Для деликатного лака — шерстяная перчатка для мойки авто.

## Варежка или губка: что лучше для мытья машины

| Критерий               | Варежка               | Губка                |
| ---------------------- | --------------------- | -------------------- |
| Площадь за один проход | Больше                | Меньше               |
| Контроль давления      | Выше (рука внутри)    | Ниже                 |
| Удержание воды         | Достаточное           | Высокое (поролон)    |
| Безопасность           | Выше (при микрофибре) | Зависит от материала |

Рука в варежке чувствует поверхность, регулирует нажим.

## Специализация: кузов и колёса отдельно

| Зона         | Инструмент                   | Почему                           |
| ------------ | ---------------------------- | -------------------------------- |
| Кузов        | Мягкая микрофибра или шерсть | Безопасность лака                |
| Диски и арки | Отдельная жёсткая губка      | Тормозная пыль и битум абразивны |

**Правило:** Одна мочалка — одна зона. Частицы с дисков царапают лак.

## Что влияет на результат

| Фактор                 | Влияние                            | Рекомендация                   |
| ---------------------- | ---------------------------------- | ------------------------------ |
| Одно ведро             | Грязь возвращается на кузов        | Два ведра + решётка Grit Guard |
| Падение на землю       | Песчинки поцарапают весь кузов     | Варежку заменить сразу         |
| Кондиционер при стирке | Плёнка — ткань перестаёт впитывать | Стирка без кондиционера        |
| Одна варежка на всё    | Грязь с дисков на кузов            | Раздельные инструменты         |

## Как ухаживать за варежкой

| Параметр           | Рекомендация                      |
| ------------------ | --------------------------------- |
| Температура стирки | 30–40°C (холодная или тёплая)     |
| Моющее средство    | Мягкое, без кондиционера          |
| Сушка              | На воздухе или низкий нагрев      |
| Хранение           | Сухое место, отдельно от колёсных |

**Нельзя:** Кондиционер для белья оставляет плёнку — ткань перестаёт впитывать.

## Когда менять мочалку

| Признак            | Что означает        |
| ------------------ | ------------------- |
| Жёсткая на ощупь   | Повреждение волокон |
| Видимые разрывы    | Износ структуры     |
| Запах после стирки | Плесень внутри      |
| 300+ стирок        | Естественный износ  |

При падении на землю — заменить сразу. Одна песчинка в волокнах поцарапает весь кузов.

## FAQ

### Какая губка лучше для мойки авто?

Микрофибровая chenille — баланс цены, безопасности, срока службы. Шерсть мягче, дороже, требует деликатного ухода.

### Чем мыть машину без царапин?

Метод двух вёдер, мягкая микрофибра/шерсть, отдельный инструмент для колёс. Избегать поролона и круговых движений.

### Сколько варежек нужно?

Минимум две: кузов + диски. Оптимально три-четыре — смена по загрязнению, просушка между мойками.

### Можно ли стирать в машинке?

Микрофибру — деликатный режим без кондиционера. Шерсть — вручную или щадящий цикл.

---

**Итог:**

-   Для кузова — микрофибра или шерсть, не поролон
-   Для колёс — отдельная жёсткая губка
-   Метод двух вёдер защищает от царапин
-   Без кондиционера при стирке, полная сушка перед хранением
