## Промпт для исследования (11 блоков)

### Блок 1: Что такое микрофибра?

1. Почему микрофибра заменила натуральную замшу (chamois) в детейлинге?
2. Состав 80/20 vs 70/30 (полиэстер/полиамид): как это влияет на мягкость и впитываемость?

### Блок 2: Виды и типы (3 независимые оси)

> ⚠️ **Важно:** Плотность (GSM) и плетение — разные параметры. Twisted Loop может быть и 600, и 1200 GSM.

**Ось 1 — Плотность (GSM):**

| Диапазон     | Описание          | Плюсы                      | Минусы               |
| ------------ | ----------------- | -------------------------- | -------------------- |
| 200-300 GSM  | Низкая плотность  | Быстро сохнет, дёшево      | Мало впитывает       |
| 350-500 GSM  | Средняя плотность | Универсальность            | Компромисс           |
| 600-1400 GSM | Высокая плотность | Максимальная впитываемость | Долго сохнет, дорого |

**Ось 2 — Тип плетения:**

| Тип                                | Описание            | Когда выбирать              |
| ---------------------------------- | ------------------- | --------------------------- |
| Twisted Loop (крученая петля)      | Скрученные петли    | Сушка кузова                |
| Waffle Weave (вафельное)           | Вафельная текстура  | Стёкла, финишная протирка   |
| Pearl / Short Pile (короткий ворс) | Гладкая поверхность | Располировка паст, керамики |
| Suede (замша)                      | Имитация замши      | Нанесение керамики          |

**Ось 3 — Тип обработки края:**

| Тип                    | Описание                    | Когда выбирать        |
| ---------------------- | --------------------------- | --------------------- |
| Оверлок (обметка)      | Классический край           | Бюджетные задачи      |
| Edgeless (безкраевые)  | Без швов по краям           | Для ЛКП               |
| Ultrasonic / Laser Cut | Ультразвуковой/лазерный рез | Премиум, безопасность |

### Блок 3: Как выбрать (Buyer's Guide)

1. **Сушка:** Twisted Loop против Waffle — что реально сушит в одно касание и не царапает?
2. **Керамика:** Почему для растирания керамики нужна "замшевая" (Suede) фибра, а не пушистая?
3. **Края:** Edgeless (без краев) — это необходимость или маркетинг? Царапает ли оверлок?
4. **Стекла:** Почему вафельное полотенце лучше для стекол, чем обычное пушистое?
5. **Цена/Качество:** Стирать или Выкидывать — какие фибры "одноразовые" (для битума/двигателя)?

### Блок 4: Безопасность (Края и швы)

1. Почему оверлок может царапать лак?
2. Технология Ultrasonic Cut (ультразвуковой рез) и Laser Cut.

### Блок 5: Цветная кодировка в детейлинге

1. Зачем использовать фибры разных цветов для разных задач? (Предотвращение перекрестного загрязнения).

### Блок 6: Правильный уход (Стирка)

1. Почему нельзя использовать обычный порошок и кондиционер?
2. Температурный режим стирки.
3. Как восстановить мягкость фибры после сушки?

### Блок 7: Ошибки

1. Использование одной фибры для всего (колеса → кузов → салон).
2. Сушка фибры на горячей батарее (плавление волокон).
3. Использование старой "деревянной" фибры на ЛКП.
