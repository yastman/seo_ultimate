## Промпт для исследования (11 блоков)

### Блок 1: Зачем нужны аппликаторы

1. Роль аппликатора в равномерном распределении состава.
2. Почему нельзя наносить составы пальцами или тряпкой.

### Блок 2: Виды и типы (3 независимые оси)

> ⚠️ **Важно:** Материал и форма — разные параметры. Поролоновый аппликатор может быть и круглым, и прямоугольным.

**Ось 1 — Материал:**

| Тип              | Описание             | Плюсы                 | Минусы               |
| ---------------- | -------------------- | --------------------- | -------------------- |
| Поролоновый      | Для восков, силантов | Универсальность, цена | Впитывает состав     |
| Микрофибровый    | Для кондиционеров    | Экономит состав       | Может оставлять ворс |
| Замшевый (suede) | Для керамики         | Точное нанесение      | Одноразовый          |
| Велюровый        | Для финиша           | Мягкость              | Дороже               |

**Ось 2 — Форма:**

| Тип                | Описание                 | Когда выбирать    |
| ------------------ | ------------------------ | ----------------- |
| Круглый            | Универсальный            | Воск, силанты     |
| Прямоугольный блок | Для больших поверхностей | Кузов, керамика   |
| С ручкой           | Эргономичный             | Длительная работа |
| Пальчиковый        | Для мелких деталей       | Молдинги, зазоры  |

**Ось 3 — Назначение по составу:**

| Тип               | Описание                 | Когда выбирать    |

| Для керамики      | Точное нанесение         | Ceramic coating   |
| Для кондиционеров | Мягкий, не царапает      | Кожа, пластик     |
| Универсальный     | Для всего                | Если не критично  |

### Блок 3: Как выбрать (Buyer's Guide)

1. **Экономия:** Почему микрофибровый аппликатор "съедает" меньше дорогой керамики, чем поролоновый?
2. **Форма:** Что удобнее для воска — традиционный "блинчик" или "кирпичик" с ручкой?
3. **Мелкие детали:** Нужен ли пальчиковый аппликатор для решетки радиатора или можно справиться обычным?
4. **Одноразовость:** Правда ли, что после нанесения кварца аппликатор превращается в наждачку?
5. **Текстура:** Зачем некоторые аппликаторы делают "вафельными"?

## Стратегия поиска (Deep Research)

1. **Техническая база (US/EU):** Искать физику процесса, химические реакции и Safety Data Sheets на англоязычных ресурсах:
   - Autogeek Online
   - Detailing World (UK)
   - Reddit r/AutoDetailing
   - Сайты производителей и экспертные блоги

2. **Локализация (UA):** Использовать локальные источники только для:
   - Проверки доступных моделей и ассортимента.
   - Специфики рынка и цен.

3. **Баланс источников:**
   - 70% информации — западные экспертные форумы/гайды.
   - 30% информации — локальные магазины и видео.
