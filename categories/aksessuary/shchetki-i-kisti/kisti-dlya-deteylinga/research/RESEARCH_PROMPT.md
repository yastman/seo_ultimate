## Контекст

Интернет-магазин автохимии Ultimate.net.ua. Категория "Кисти для детейлинга" включает детейлинг-пензли (кисти), щётки для химчистки салона и наборы кистей разных размеров. Основные типы товаров:

**Ассортимент магазина:**
- Наборы детейлинг-пензлей (3-5 шт) разных размеров
- Кисти с натуральным ворсом (для деликатных поверхностей)
- Кисти с синтетическим ворсом (устойчивы к химии)
- Кисти со смешанным ворсом (универсальные)
- Щётки для химчистки салона и чистки щелей
- Ультрамягкие кисточки для чистки без царапин

**Ключевые характеристики товаров:**
- Тип ворса: натуральный / синтетический / смешанный
- Длина щетины: стандартная / длинная
- Жёсткость: ультрамягкие / мягкие / средние
- Размеры в наборах: разные диаметры для разных зон
- Применение: салон / экстерьер / универсальные
