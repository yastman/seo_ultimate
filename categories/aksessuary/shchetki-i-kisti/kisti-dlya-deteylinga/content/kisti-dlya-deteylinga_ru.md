# Щетка для детейлинга

Щетка для детейлинга помогает вычистить труднодоступные зоны — вентиляцию, швы, кнопки, диски — без риска царапин. Кисть с мягким ворсом подходит для панели и кожи, кисть с жёстким ворсом — для колёс и решёток. Если вам нужна точечная работа с деталями, выбирайте кисточки для детейлинга с тонким ворсом, для больших поверхностей — модели с широкой основой.

## Типы инструментов для детейлинга салона

| Тип | Жёсткость щетины | Основное применение | Особенности |
|-----|-----------------|---------------------|-------------|
| **Салонные** | Мягкая / средняя | Панель, кнопки, швы, вентиляция | Деликатные волокна с антистатическим эффектом |
| **Для дисков** | Жёсткая | Колёса, арки, решётки радиатора | Длинная ручка, устойчивость к агрессивной химии |
| **Для кожи** | Мягкая | Сидения, руль, двери из кожи и алькантары | Короткие волокна для равномерного распределения |
| **Универсальные наборы** | Микс | Комплексная обработка салона и кузова | Набор кистей разных размеров для всех зон |

## Как выбрать набор щеток для детейлинга

**Если нужна обработка мелких деталей салона** → возьмите кисть для щелей с мягким ворсом небольшого диаметра. Она проникает в зазоры между кнопками и вентиляционными жалюзи, не царапая пластик.

**Если планируется химчистка сидений** → щетка для чистки салона автомобиля со средней жёсткостью и волокнами средней длины. Такая структура вспенивает очиститель в текстуре ткани, сохраняя основу невредимой.

**Если работаете с кожаной обивкой** → используйте щетку для кожи авто с мягкой натуральной щетиной. Жёсткие синтетические варианты могут оставить микроцарапины на финишном покрытии.

**Если чистите диски и арки** → берите щетку для дисков авто с длинной ручкой и жёстким ворсом. Длина обеспечивает доступ к внутренней стороне спиц, а плотная структура снимает налёт тормозной пыли.

**Если вы начинающий детейлер** → начните с набора щеток для детейлинга, где есть инструменты разного размера. Это экономичнее, чем покупать каждый аксессуар отдельно, и даёт понимание, какие форматы нужны чаще.

**Если наносите составы на труднодоступные зоны** → выбирайте модели с плотной короткой щетиной. Они равномерно распределяют воски, дрессинги и защитные составы по рельефным поверхностям.

### Материал щетины: на что обратить внимание

**Натуральное волокно** подходит для деликатных поверхностей — кожи, лакированных вставок, хромированных элементов. Оно мягче синтетики, но хуже переносит агрессивную химию и требует тщательной просушки после мойки.

**Синтетическая щетина** более износостойкая и химически инертная. Используется для дисков, пластика, резиновых уплотнителей. Такие инструменты можно промывать под напором воды без потери формы.

**Смешанный состав** (натуральное + синтетика) — компромисс между мягкостью и прочностью. Подходит для универсальных салонных моделей, где нужна и деликатность, и долговечность.

## Частые вопросы

**Можно ли одной щеткой чистить и салон, и диски?**
Нежелательно. Щетка для дисков авто накапливает частицы металла и тормозной пыли, которые могут поцарапать пластик салона. Держите отдельные инструменты для кузова и интерьера.

**Как часто менять детейлинг-аксессуары?**
Зависит от интенсивности использования. При регулярной мойке салонные модели держатся дольше, варианты для дисков изнашиваются быстрее из-за агрессивной химии. Меняйте, когда щетина начинает выпадать или терять форму.

**Чем кисти отличаются от щёток для детейлинга?**
Кисти (detail brush) имеют более длинный и мягкий ворс, собранный в пучок. Щётки — жёсткую щетину на широкой основе. Первые подходят для точечной работы, вторые — для больших плоскостей.

**Нужен ли отдельный инструмент для вентиляции?**
Желательно. Кисть для вентиляции имеет сложную геометрию, где накапливается пыль. Узкая модель проникает между рёбрами решётки, вычищая налёт без демонтажа панели.

**Можно ли мыть аксессуары в посудомоечной машине?**
Нет. Высокая температура и агрессивные моющие средства повреждают клей в обойме, распушают щетину. Мойте вручную в тёплой воде с нейтральным шампунем.
