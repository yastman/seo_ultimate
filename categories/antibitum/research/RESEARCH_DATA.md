# Research: Антибитум (antibitum)

## 1. Базовая матрица поиска

| Данные | Результат |
| :--- | :--- |
| **Что это** | Специализированный очиститель на основе органических растворителей или терпеновых масел (цитрусовых) для удаления нефтепродуктов. |
| **Для чего** | Удаление пятен битума, гудрона, смолы деревьев, следов скотча и масляных загрязнений. |
| **Классификация** | **По составу:** Спиртовые/сольвентные (агрессивные) и Терпеновые (безопасные, с запахом цитруса). |
| **Расход** | **100-250 мл** на полную очистку сильно загрязненного низа легкового авто. |
| **Инструкция** | 1. Вымыть и высушить кузов. 2. Распылить на пятна. 3. Подождать 1-5 минут. 4. Протереть микрофиброй без сильного нажима. 5. Тщательно смыть водой с шампунем. |
| **Совместимость** | Безопасен для ЛКП, стекол и дисков. Может постепенно смывать воски. Безопасен для керамики. |
| **Ошибки** | Нанесение на грязный кузов (риск царапин), работа на горячем ЛКП (быстрое испарение), использование дешевых растворителей (порча пластика). |
| **Критерии выбора** | Безопасность для пластика/резины, скорость работы, наличие эмульгаторов (для легкого смывания водой). |

## 2. Углубление для детейлинга (E-E-A-T)

* **Расход:** 250-500 мл на полную очистку сильно загрязненного легкового авто. Локально — буквально несколько пшиков.
* **Оборудование:** Микрофибра (лучше старая, которую не жалко выбросить), Распылитель (триггер).
* **Safety / Ограничения:** Не использовать на солнце. Не допускать высыхания. При работе с пластиком сначала проверить на незаметном месте.
* **Эффективность:** Растворяет битум в течение 1-2 минут. Требует деликатной механической помощи фиброй для застарелых пятен.

## 3. Полезные факты и лайфхаки

* **Факт:** Битум — это не просто грязь, это нефтепродукт, который при длительном контакте может "прокрасить" (желтить) светлый лак навсегда.
* **Лайфхак (Sequence):** Всегда используйте антибитум **ДО** глины. Если пустить глину первой, она просто "размажет" жирный битум по лаку и мгновенно забьется, придя в негодность.
* **Лайфхак:** Для удаления следов скотча или наклеек используйте терпеновые составы (цитрусовые) — они работают деликатнее и не пахнут агрессивной химией.
* **Лайфхак:** Если битум застарелый, накройте обработанное место салфеткой, смоченной в антибитуме — это предотвратит быстрое испарение и позволит составу глубже проникнуть в пятно.
* **Гайд (микро):** Всегда смывайте антибитум шампунем. Сольвенты оставляют маслянистую пленку, которая будет притягивать новую пыль.

## 4. Меры предосторожности

* **Пластик:** Избегайте попадания на текстурированный неокрашенный пластик — некоторые дешевые антибитумы могут оставить на нем белесые пятна.
* **Резина:** Длительный контакт с софтовыми растворителями может привести к разбуханию тонких резиновых уплотнителей.
* **Пожаробезопасность:** Составы крайне огнеопасны. Не используйте на солнце и рядом с источниками искр.
