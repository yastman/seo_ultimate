# Глина и автоскрабы

Очищающая глина для кузова автомобиля убирает въевшиеся загрязнения, которые не смывает обычная мойка: промышленный налёт, битум, смолу деревьев, металлическую пыль. Полимерная глина для авто готовит поверхность к нанесению защитных покрытий. Детейлинг-брусок (clay bar) даёт идеальную гладкость перед полировкой, автоскраб-рукавица — быстрый результат без риска испортить инструмент. Если вам важна скорость — выбирайте автоскраб, для глубокой подготовки под керамику — синюю глину для авто.

## Как выбрать глину для чистки авто

| Тип | Плюсы | Минусы | Для кого |
| --- | ----- | ------ | -------- |
| **Брусок (clay bar)** | Глубокая очистка, повторяет рельеф, экономный расход смазки | Если упал — выбросить, медленнее работать | Профи, подготовка к полировке |
| **Автоскраб-рукавица** | Быстро, можно промыть после падения, удобно держать | Плоская поверхность — меньше глубина очистки | Новички, регулярный уход |
| **Автоскраб-диск** | Работает с полировальной машинкой, ускоряет процесс | Нужен опыт, риск перегрева поверхности | Профессиональный детейлинг |

**Сценарии:**
- **Если готовите кузов под керамику** → детейлинг-брусок средней абразивности
- **Если нужен быстрый результат без рисков** → автоскраб-варежка
- **Если битум только на порогах** → кусочек бруска точечно

## На что смотреть на этикетке

| Маркер | Что означает | Кому подходит |
| ------ | ------------ | ------------- |
| **Fine (мягкая)** | Деликатная очистка | Регулярный уход, мягкий лак (японские авто, перекрас) |
| **Medium (средняя)** | Стандартные загрязнения | Подготовка к полировке, обычная эксплуатация |
| **Heavy (жёсткая)** | Сильный налёт, опыл краски | Промышленный район, после зимы — только с опытом |

**Важно:** Чем выше абразивность, тем больше риск микроцарапин. Если лак мягкий — берите только Fine.

## Лубрикант для глины — почему нельзя без него

Без смазки глина прилипает к лаку и царапает его вместо очистки. Профессиональный лубрикант создаёт скользящий слой и вымывает вытянутые частицы, не оставляя разводов.

| Что использовать | Подходит | Риск царапин |
| ---------------- | -------- | ------------ |
| Специальный спрей для глины | Оптимальное скольжение | Минимальный |
| Квик-детейлер | Хорошая альтернатива | Низкий |
| Разбавленный шампунь | Крайний случай | Средний (может оставить плёнку) |
| Вода | Не работает | Высокий |

**Правило:** Обильно — и на поверхность, и на инструмент. Лучше перелить, чем недолить.

## Синяя глина для авто или голубая — есть ли разница

Нет. Цвет — маркетинг. Синяя = голубая = blue clay. Глина для чистки авто одинаково эффективна независимо от оттенка. Производители красят глину для визуального отличия. Реальное значение имеет только маркировка абразивности: Fine/Medium/Heavy на упаковке.

## Что влияет на результат

| Фактор | Влияние | Рекомендация |
| ------ | ------- | ------------ |
| **Лубрикант** | Без него масса прилипает, царапает лак | Обильно на панель и брусок |
| **Движения** | Круговые оставляют паутинку | Прямые линии, лёгкое давление |
| **Падение** | Песчинки в бруске = царапины по всему кузову | Брусок выбросить, автоскраб промыть |
| **Температура** | На горячей панели смазка высыхает | Работать в тени, кузов прохладный |

## FAQ

### Автомобильная глина или автоскраб — что лучше для новичка?

Автоскраб безопаснее: если уронили — промыли водой и продолжили. Брусок после падения нужно выбросить, иначе песчинки внутри массы поцарапают весь кузов. Для первого опыта выбирайте варежку.

### Можно ли использовать на стекле и хроме?

Да, автомобильная глина безопасна для стекла, хрома, лака. Главное — достаточно лубриканта и мягкая абразивность (Fine). Убирает налёт с фар и хромированных решёток так же эффективно, как с кузова.

### Сколько хватает одного бруска?

Стандартный кусок — на обработку седана целиком. Большие бруски обычно делят на несколько частей и используют постепенно. Автоскраб-варежка служит несколько обработок при правильном хранении.

### Как часто нужно обрабатывать кузов глиной?

Зависит от условий: в городе достаточно раз в год (весной после реагентов), под деревьями или в промышленной зоне — несколько раз за сезон. Проверить просто: наденьте пакет на руку и проведите по панели — если ощущаете шероховатости, пора чистить.

---

**Итог — что вам покупать:**
- **Подготовка под полировку или керамику** — глина для полировки автомобиля средней абразивности (детейлинг-брусок)
- **Регулярный уход** — автоскраб-варежка для экономии времени
- **Глина для кузова** работает эффективно только с лубрикантом: специальный спрей оптимален, разбавленный шампунь допустим, вода без добавок — нет
