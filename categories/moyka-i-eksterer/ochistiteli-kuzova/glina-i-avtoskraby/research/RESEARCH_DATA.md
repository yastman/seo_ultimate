# Research: Глина и автоскрабы (glina-i-avtoskraby)

## 1. Базовая матрица поиска

| Данные | Результат |
| :--- | :--- |
| **Что это** | Масса (глина) или резинополимерный инструмент для глубокой очистки пор лака. |
| **Для чего** | Удаление въевшихся вкраплений металла, опыла краски, смолы, которые не смываются шампунем. |
| **Классификация** | **Глина:** Fine (мягкая), Medium, Heavy. **Автоскрабы:** пады, полотенца, перчатки. |
| **Расход** | **20-40 г** глины или 1 качественный автоскраб (на 50-100 машин). |
| **Инструкция** | 1. Вымыть и высушить авто. 2. Обильно нанести лубрикант. 3. Тереть продольными движениями. 4. Разминать глину. |
| **Совместимость** | Безопасны для ЛКП и стекол при наличии лубриканта. Удаляют старые воски. |
| **Ошибки** | Использование упавшей на пол глины (выбросить!), недостаточно лубриканта, круговые движения. |
| **Критерии выбора** | Степень загрязнения (абразивность), долговечность (скрабы служат дольше). |

## 2. Углубление для детейлинга (E-E-A-T)

*   **Расход:** Бруска 100 г хватает на 3-5 полных очисток авто. Автоскраб долговечен (до 50-100 машин при правильной очистке).
*   **Оборудование:** Лубрикант (специальный состав или раствор шампуня), Микрофибра для протирки.
*   **Safety / Ограничения:** Категорический запрет на использование без смазки (лубриканта). Не работать на горячем кузове.
*   **Эффективность:** Делает поверхность "гладкой как стекло". Подготавливает базу для идеальной адгезии защитных покрытий.

## 3. Полезные факты и лайфхаки

*   **Факт:** Глина не "срезает" загрязнения, она их "вытягивает" и запечатывает в себе. Поэтому ее нужно постоянно разминать.
*   **Лайфхак (Baggie Test):** Чтобы понять, нужна ли чистка глиной, наденьте обычный тонкий пластиковый пакет на руку и проведите по лаку после мойки. Пакет усилит тактильные ощущения, и вы почувствуете всю "шершавость" (песчинки), которую не слышно голой рукой.
*   **Лайфхак (Lubricant Secret):** Вместо дорогого лубриканта можно использовать воду с небольшим количеством pH-нейтрального шампуня без воска. Это обеспечивает отличное скольжение.
*   **Лайфхак:** Очистите глиной лобовое стекло. Это уберет весь микро-налет, и дворники начнут работать идеально тихо и без пропусков.
*   **Гайд (микро):** Совершайте только продольные/поперечные движения. Круговые движения при попадании песчинки создадут "завитки", которые сложно выполировывать.

## 4. Меры предосторожности

*   **Падение:** Если кусочек глины упал на землю — он сразу отправляется в мусор. Песок превратит его в наждак.
*   **Температура:** Не работайте на горячем ЛКП. Лубрикант высохнет, и глина начнет залипать, оставляя следы.
