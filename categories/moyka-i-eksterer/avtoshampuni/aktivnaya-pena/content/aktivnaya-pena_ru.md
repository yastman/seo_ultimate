# Активная пена

Активная пена для мойки автомобиля (Pre-wash) размягчает 80–90% загрязнений без контакта с кузовом. Для керамики — нейтральная формула (pH 7–9), для зимней грязи и реагентов — щелочная (pH 10–12).

## Как выбрать: сценарии

| Тип | pH | Назначение | Влияние на защиту |
|-----|-----|------------|-------------------|
| Щелочная (Active/Power) | 10–12 | Зимняя грязь, жир, реагенты | Смывает воск; безопасна для керамики |
| Нейтральная (Safe) | 7–9 | Регулярная обработка | 100% безопасна для всех покрытий |
| С добавками (SiO₂) | разный | Обработка + защита | Добавляет гидрофобный слой |

**Обычная грязь, пыль, насекомые** → щелочная пена (pH 10–12).

**Регулярный уход ухоженного авто** → нейтральная (pH 7–9).

**Машина с керамикой, воском, PPF** → нейтральная формула (pH 7–9).

**Хотите обработку и защиту сразу** → пена с SiO₂.

## На что смотреть

| Параметр | Значение для покупателя |
|----------|-------------------------|
| pH | Щелочной мощнее, смывает силанты. Нейтральный безопасен |
| Концентрация | Выше концентрация = экономичнее per wash |
| Густота | Плотная "шапка" дольше держится на кузове |
| Совместимость | Пенокомплект и пеногенератор требуют разное разведение |

## Выбор по оборудованию

| Оборудование | Разведение | Причина |
|--------------|------------|---------|
| Пенокомплект (foam lance) | 1:5–1:10 | Слабое давление, нужна густота |
| Пеногенератор | 1:20–1:50 | Высокое давление взбивает лучше |
| Ручной распылитель | Ready-to-use | Без давления, максимальная концентрация |

**Пена стекает как вода** → увеличьте концентрацию или замените таблетку в оборудовании.

**Используете Kärcher, Bosch, Nilfisk** → проверьте совместимость с моделью.

## Чего избегать

**Щелочная химия для еженедельного ухода** → смоете воск за 2-3 процедуры. Берите нейтральную.

**Слишком слабый концентрат** → придётся лить больше, экономия исчезнет.

**Игнорировать pH при керамике** → щелочь безопасна только при быстром смывании (3-5 минут).

## FAQ

### Пена стекает как вода. Что делать?

Проблема в оборудовании, не в химии. Дешёвые пенокомплекты имеют плохую пенную таблетку. Попробуйте концентрацию 1:5 вместо 1:10. Не помогло — замените таблетку. Тёплая вода взбивает лучше.

### Snow Foam или Pre-Wash (TFR)? В чём разница?

**Snow Foam** — густая шапка, обычно pH-нейтральная. Работает мягче, идеальна для летней пыли. **TFR (Traffic Film Remover)** — жидкая эмульсия, почти без пены, обычно щелочная. Эффективнее против зимней грязи и реагентов.

### Можно ли щелочную пену на машине с керамикой?

Да. Керамика выдерживает pH 11–12, если не давать пене высыхать. Используйте щелочь только при сильном загрязнении. Для регулярной мойки — нейтральные формулы pH 7–9.

---

**Итог — что покупать:**
- **Регулярная мойка** → нейтральная пена (pH 7–9)
- **Сильная грязь, зима** → щелочная (pH 10–12)
- **Керамика + защита** → нейтральная без воска
