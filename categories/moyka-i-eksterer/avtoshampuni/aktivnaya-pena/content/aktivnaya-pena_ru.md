# Пена для мойки авто

Пена для мойки авто — pre-wash концентрат для бесконтактной очистки. Густой слой размягчает 80-90% дорожной грязи за 5-10 минут dwell time без механического контакта с лаком. Критерии выбора активной пены: pH формулы, совместимость с защитным покрытием, липкость состава.

## Как выбрать: сценарии

| Состояние авто | Тип | Почему |
|----------------|-----|--------|
| Зимняя грязь, реагенты | Щелочная (pH 10-12) | Разлагает соль, масляную плёнку за 7-10 минут |
| Еженедельный уход | Нейтральная (pH 6-8) | Сохраняет воск, силант, керамику |
| Керамика, PPF, свежий воск | pH-нейтральная без фосфатов | Поддерживает гидрофобность покрытия |
| Лёгкая пыль | Любая, разведение 1:30-1:50 | Экономичный расход концентрата |

**Авто с керамическим покрытием** → нейтральная формула (pH 6-8). Щелочь при регулярном применении снижает гидрофобный эффект.

**Зимний период, дорожные реагенты** → щелочной концентрат (pH 10-12). Выдержка 7-10 минут, смывание до высыхания обязательно.

**Свежий воск или силант** → нейтральная химия сохраняет защитный слой. Щелочная при разведении 1:5 смывает воск за 2-3 процедуры.

## Параметры выбора

| Параметр | Значение |
|----------|----------|
| pH | 10-12 = агрессивная очистка, риск смывания воска. 6-8 = безопасность для покрытий |
| Разведение | Пеногенератор: 1:20-1:50. Пенокомплект минимойки: 1:5-1:10 |
| Липкость | Густая формула удерживается 5-10 минут на вертикальных панелях |
| Расход | 200-400 мл концентрата на легковой автомобиль |
| Давление АВД | Оптимум 120-150 бар. Минимум для пенообразования: 100 бар |

## Совместимость с оборудованием

**Минимойка 100-120 бар (Kärcher, Bosch, Nilfisk)** → нейтральный концентрат с повышенной липкостью. Разведение 1:5-1:10 в пенокомплекте. Давление ниже 100 бар = жидкий раствор вместо густой шапки.

**Профессиональный пеногенератор** → концентрат 1:20-1:50. Давление 120-150 бар обеспечивает густую пену, удерживающуюся 10 минут на вертикалях.

**Ручной распылитель** → готовый раствор (RTU). Концентрат без давления не вспенивается.

## Ошибки выбора

**Щелочная формула для еженедельной мойки** — воск/силант смывается за 2-3 процедуры. Регулярный уход требует pH 6-8.

**Разведение 1:100 и слабее** — моющая способность минимальна. Рабочий диапазон: 1:20-1:50 для пеногенератора.

**Концентрат без указания pH** — невозможно оценить агрессивность, совместимость с защитой кузова.

## FAQ

### Какую пену выбрать для минимойки?

Нейтральную (pH 6-8), совместимую с бытовыми АВД. Разведение 1:5-1:10 в пенокомплекте. Давление ниже 100 бар — ориентир липкость состава, не толщина слоя.

### Чем отличается от обычного шампуня?

Бесконтактная химия наносится пеногенератором, работает 5-10 минут (dwell time), удаляет рыхлую грязь без губки. Шампунь — контактный этап после pre-wash. Разные фазы мойки, не взаимозаменяемые продукты.

### Какой продукт подходит для керамики?

pH-нейтральный без фосфатов. Щелочной допустим при сильном загрязнении — смывание за 5 минут, высыхание на кузове недопустимо.

---

**Итог:**
- **Регулярная мойка** → нейтральный концентрат (pH 6-8)
- **Зимняя грязь, реагенты** → щелочная формула (pH 10-12)
- **Керамика, воск, PPF** → pH-нейтральная без фосфатов
- **Минимойка до 120 бар** → состав с повышенной липкостью
