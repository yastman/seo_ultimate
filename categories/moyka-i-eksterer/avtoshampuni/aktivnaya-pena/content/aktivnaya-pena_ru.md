# Пена для мойки авто

Пена для мойки авто — pre-wash концентрат для бесконтактной очистки кузова. Густой слой размягчает 80-90% дорожной грязи за 5-10 минут без механического контакта с лаком, снижая риск микроцарапин при последующей ручной мойке. Критерии выбора: pH формулы, совместимость с защитным покрытием, липкость состава для удержания на вертикальных панелях.

## Как выбрать: сценарии

| Состояние авто | Тип состава | Почему подходит |
|----------------|-------------|-----------------|
| Зимняя грязь, дорожные реагенты | Щелочной (pH 10-12) | Разлагает соль и масляную плёнку за 7-10 минут выдержки |
| Еженедельный уход | Нейтральный (pH 6-8) | Сохраняет воск, силант, керамическое покрытие |
| Авто с керамикой, PPF, свежим воском | pH-нейтральный без фосфатов | Поддерживает гидрофобность защитного слоя |
| Лёгкая пыль, птичий помёт | Любой, разведение 1:30-1:50 | Экономичный расход концентрата |

**Авто с керамическим покрытием** → выбирать нейтральную формулу (pH 6-8). Щелочные составы при регулярном применении постепенно снижают гидрофобный эффект керамики.

**Зимний период с дорожными реагентами** → щелочной концентрат справляется лучше. Время выдержки 7-10 минут, обязательно смывать до высыхания на кузове.

**Свежий воск или силант на кузове** → нейтральная химия сохраняет защитный слой дольше. Щелочной состав при разведении крепче 1:10 способен смыть воск за 2-3 процедуры.

## Параметры для сравнения

| Параметр | На что влияет |
|----------|---------------|
| pH | 10-12 = агрессивная очистка, риск повреждения воска. 6-8 = безопасность для любых покрытий |
| Разведение | Высокое давление: 1:20-1:50. Бытовая минимойка: 1:5-1:10. Ручной опрыскиватель IK Foam: 1:30 |
| Липкость | Густая формула удерживается 5-10 минут на вертикальных панелях, давая время для работы ПАВ |
| Расход | 200-400 мл концентрата на легковой автомобиль среднего размера |
| Давление АВД | Оптимум 120-150 бар для густой шапки. Минимум 100 бар для базового вспенивания |
| Температурный режим | Воздух 5-30°C, поверхность кузова не горячее 40°C. На горячем металле состав высыхает быстрее |

## Совместимость с оборудованием

**Бытовая минимойка 100-120 бар** → нейтральный концентрат с повышенной липкостью. Разведение 1:5-1:10 в комплекте для нанесения. При давлении ниже 100 бар получается жидкий раствор вместо густой шапки — это ограничение оборудования, а не продукта.

**Профессиональное оборудование высокого давления** → концентрат с разведением 1:20-1:50. Давление 120-150 бар обеспечивает густой слой, который удерживается до 10 минут на вертикальных поверхностях.

**Ручной помповый опрыскиватель (IK Foam)** → разведение около 1:30. Слой будет менее густым, но времени выдержки достаточно для размягчения лёгких загрязнений.

**Готовый раствор RTU** → для распылителей без давления. Концентрат без механического взбивания не даёт нужной текстуры.

## Чего избегать при выборе

**Щелочная формула для еженедельной мойки** — воск и силант смываются за 2-3 процедуры. Для регулярного ухода подходит только pH 6-8.

**Слишком слабое разведение (1:100 и больше)** — моющая способность становится минимальной. Рабочий диапазон для профессионального оборудования: 1:20-1:50.

**Концентрат без указания pH на этикетке** — невозможно оценить агрессивность состава и совместимость с защитой кузова.

**Нанесение на горячий кузов или хромированные молдинги** — состав высыхает до смывания, оставляя белёсые разводы. Работать в тени или охладить поверхность водой перед нанесением.

## FAQ

### Какой продукт выбрать для минимойки?

Нейтральный (pH 6-8), совместимый с бытовыми АВД. Разведение 1:5-1:10 в комплекте для нанесения. При давлении ниже 100 бар ориентироваться на липкость состава, а не на толщину слоя — жидкая консистенция при низком давлении это норма.

### Чем бесконтактная химия отличается от обычного шампуня?

Способом нанесения и задачей. Бесконтактная химия наносится через оборудование высокого давления, работает 5-10 минут (dwell time), удаляет рыхлую грязь без губки. Шампунь — контактный этап после предварительной очистки. Это две разные фазы мойки, продукты не взаимозаменяемые.

### Какой продукт подходит для авто с керамикой?

pH-нейтральный без фосфатов. Щелочной допустим только при сильном загрязнении с обязательным смыванием за 5 минут — высыхание на кузове недопустимо, иначе керамика теряет гидрофобные свойства быстрее.

### Как хранить концентрат?

При комнатной температуре в тёмном месте. Срок годности большинства составов — 12-24 месяца. Не смешивать разные продукты в одной ёмкости — это может нарушить pH и снизить эффективность.

### Почему химия быстро стекает и не держится на кузове?

Причины: давление оборудования ниже 100 бар, слабое разведение (крепче 1:30), горячая поверхность кузова. Решение: увеличить концентрацию до 1:5-1:10, охладить кузов водой перед нанесением, проверить давление минимойки.

---

**Итог — что покупать:**
- **Регулярная мойка** → нейтральный концентрат (pH 6-8)
- **Зимняя грязь, реагенты** → щелочная формула (pH 10-12), выдержка 7-10 минут
- **Керамика, воск, PPF** → pH-нейтральный без фосфатов
- **Минимойка до 120 бар** → состав с повышенной липкостью, разведение 1:5-1:10
