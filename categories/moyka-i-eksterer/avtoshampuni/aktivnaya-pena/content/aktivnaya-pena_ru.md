# Пена для мойки авто

Активная пена для мойки автомобиля — концентрат для бесконтактной очистки кузова через пеногенератор. Густой слой размягчает дорожную грязь и пыль без прямого контакта с лакокрасочным покрытием. Это первый этап мойки, который снижает риск появления микроцарапин при последующей ручной обработке.

## Как выбрать средство для бесконтактной мойки

| Тип по pH | Назначение | Влияние на защиту |
|-----------|------------|-------------------|
| Щелочная (pH 10-12) | Сильные загрязнения, зимние реагенты | Может смыть воск при высокой концентрации |
| Нейтральная (pH 6-8) | Регулярный уход, авто с керамикой | Безопасна для защитных покрытий |

**Сценарии выбора:**

- **Зимняя грязь, реагенты** → щелочной состав, более насыщенное разведение
- **Авто с керамикой или PPF** → pH-нейтральный продукт, безфосфатный
- **Еженедельный уход** → нейтральный концентрат
- **Домашняя минимойка** → концентрат с хорошей липкостью для длительного контакта

### На что смотреть на этикетке

| Маркер | Что означает | Кому подходит |
|--------|--------------|---------------|
| pH-neutral / pH 7 | Не смывает воск и керамику | Владельцам авто с защитой |
| Alkaline / щелочной | Агрессивная очистка въевшейся грязи | Для периодической глубокой мойки |
| Безфосфатный | Экологичность, безопасность для покрытий | Регулярное использование |
| Концентрат 1:50 | Высокая экономичность | Частые мойки |

## Как применять

1. **Развести концентрат** — согласно инструкции производителя (стандартно для пеногенератора)
2. **Предварительно ополоснуть** — сбить рыхлую грязь струёй воды
3. **Нанести снизу вверх** — начинать с нижних панелей, двигаясь к крыше
4. **Дать составу поработать** — выдержать время, не допуская полного высыхания
5. **Смыть** — давлением воды, пока слой ещё держится на поверхности

**Распространённые ошибки:**

- **Дать высохнуть на кузове** — появятся разводы и белёсые пятна
- **Нанести на горячий кузов** — состав быстро стечёт, не успев подействовать
- **Наносить сверху вниз** — средство стечёт, верхние панели останутся без обработки
- **Слишком крепкий раствор** — щелочь может повредить защитное покрытие

## Практические данные

| Параметр | Значение |
|----------|----------|
| Стандартное разведение (нейтральная) | согласно инструкции на флаконе |
| Разведение для тяжёлых загрязнений | более концентрированное |
| Время выдержки | зависит от степени загрязнения и температуры воздуха |
| Примерный расход концентрата | на одну мойку легкового авто |

## FAQ

### Чем активная пена отличается от обычного шампуня?

Бесконтактное средство наносится пеногенератором под давлением и работает без прямого контакта с кузовом. Обычный ручной шампунь используется с губкой на втором этапе. Активная пена — предварительный этап, снижающий количество абразива на поверхности.

### Можно ли мыть машину только бесконтактным способом?

Этот способ удаляет большую часть рыхлых загрязнений. Для полной очистки рекомендуется контактная мойка ручным шампунем — это завершающий этап перед сушкой.

### Смывает ли щелочной состав керамику?

При стандартном разведении и времени выдержки щелочной продукт безопасен. Риск появляется при слишком концентрированном растворе или длительном контакте. Для авто с керамикой предпочтительнее pH-нейтральное средство для регулярного ухода.

### Почему состав не держится и быстро стекает?

Возможные причины: недостаточное давление минимойки, горячий кузов, неправильная пропорция разведения. Убедитесь, что давление АВД достаточное, кузов не перегрет, концентрация соответствует рекомендациям.

### Что выбрать для минимойки?

Для домашней минимойки подойдёт концентрат с хорошими пенообразующими свойствами. Обращайте внимание на совместимость с вашим пеногенератором и рекомендуемое давление в характеристиках продукта.
