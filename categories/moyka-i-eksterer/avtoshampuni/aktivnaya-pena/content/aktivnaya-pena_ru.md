# Пена для мойки авто

Активная пена — первый этап бесконтактной мойки: состав размягчает 80–90% дорожной грязи без механического контакта с лаком, снижая риск микроцарапин. Главные критерии выбора: pH формулы, совместимость с защитным покрытием автомобиля.

## Как выбрать: сценарии

| Состояние авто | Что брать | Почему |
|----------------|-----------|--------|
| Зимняя грязь, реагенты | Щелочная (pH 10–12) | Разлагает соль и масляную плёнку за 5–7 минут |
| Регулярный уход раз в неделю | Нейтральная (pH 7–9) | Сохраняет воск и силанты |
| Керамика, PPF, свежий воск | pH 7–9 без фосфатов | Поддерживает гидрофобный слой |
| Пыль и лёгкие загрязнения | Любая с разведением 1:30 | Экономия концентрата |

**При керамическом покрытии** → нейтральная формула (pH 7–9). Щелочь безопасна для керамики, но при частом применении снижает гидрофобность.

**При редкой зимней мойке** → щелочной состав (pH 10–12) справится лучше. Dwell time 7–10 минут до смывания.

**При быстром стекании состава** → проблема в оборудовании или разведении. Увеличение концентрации до 1:5 в пенокомплекте решает вопрос.

**При свежем воске или силанте** → нейтральная химия сохранит защиту на 3–4 мойки дольше щелочной.

## На что смотреть при выборе

| Параметр | Значение |
|----------|----------|
| pH | Щелочной (10–12) мощнее, агрессивнее. Нейтральный (7–9) безопасен для любых покрытий |
| Концентрация | 1:50 = 50 моек из литра. Выше концентрация — экономичнее расход |
| Совместимость | Пенокомплект требует 1:5–1:10, пеногенератор — 1:20–1:50 |
| Расход | 200–400 мл концентрата на седан среднего размера |

## Выбор по оборудованию

**Минимойка 100–120 бар (Kärcher, Bosch, Nilfisk)** → нейтральный состав с хорошей липкостью, разведение по инструкции. При давлении ниже 100 бар густая шапка не получится — нормально для бытовых АВД.

**Пеногенератор профессиональный** → концентрат 1:20–1:50. Высокое давление взбивает густую шапку, удерживающуюся 5–10 минут на вертикальных панелях.

**Ручной распылитель** → готовый раствор (RTU). Без давления концентрат не вспенится — подходят только ready-to-use формулы.

**При давлении минимойки ниже 80 бар** → пена не генерируется, получается жидкий раствор. Причина — недостаточная мощность АВД.

## Чего избегать при выборе

**Щелочная химия для еженедельной мойки** → защитный слой смывается за 2–3 процедуры. Регулярный уход требует pH 7–9.

**Концентрат с разведением 1:100** → реальная моющая способность сопоставима с водой. Проверенный диапазон: 1:20–1:50.

**Составы без указания pH** → невозможно оценить агрессивность. Качественные производители указывают этот параметр.

## FAQ

### Какой состав выбрать для минимойки?

Нейтральный (pH 7–9) с указанием совместимости с бытовыми АВД. Разведение обычно 1:10–1:20 в пенокомплекте. При давлении ниже 100 бар густая шапка не образуется — ориентир: липкость состава, не толщина слоя.

### Чем активная пена отличается от обычного шампуня?

Способом нанесения и задачей. Пена работает бесконтактно через пеногенератор, удаляя рыхлую грязь за 5–10 минут dwell time. Шампунь наносится губкой после пены для финальной очистки. Два разных этапа мойки, не взаимозаменяемые продукты.

### Какой продукт взять для авто с керамикой?

Нейтральный без фосфатов (pH 7–9). Щелочной допустим при сильном загрязнении, но смывание за 5 минут обязательно — высыхание на кузове недопустимо. При регулярном использовании щелочи гидрофобность керамики снижается быстрее.

### Как определить слабую концентрацию?

Состав стекает за 1–2 минуты, не успевая размягчить грязь. Решение: увеличение дозировки вдвое или выбор продукта с более высокой концентрацией (1:20 вместо 1:50).

---

**Итог — что покупать:**
- **Регулярная мойка** → нейтральная пена (pH 7–9)
- **Зимняя грязь, реагенты** → щелочная (pH 10–12)
- **Керамика, воск, PPF** → нейтральная без фосфатов
- **Минимойка до 120 бар** → состав с хорошей липкостью
