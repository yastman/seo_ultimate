# Пена для мойки авто

Активная пена размягчает 80–90% грязи без контакта с кузовом — это первый этап бесконтактной мойки перед ручным шампунем. Главный критерий выбора: pH состава и тип защитного покрытия на автомобиле.

## Как выбрать: сценарии

| Состояние авто | Что брать | Почему |
|----------------|-----------|--------|
| Зимняя грязь, реагенты | Щелочная (pH 10–12) | Разлагает соль и масляную плёнку за 5–7 минут |
| Регулярный уход раз в неделю | Нейтральная (pH 7–9) | Не смывает воск и силанты |
| Керамика, PPF, свежий воск | pH 7–9 без фосфатов | Сохраняет гидрофобный слой |
| Пыль и лёгкие загрязнения | Любая с разведением 1:30 | Экономия концентрата |

**Если машина покрыта керамикой** → берите только нейтральную формулу (pH 7–9). Щелочь безопасна для керамики, но при частом применении снижает гидрофобность.

**Если моете раз в месяц зимой** → щелочная пена (pH 10–12) справится лучше. Дайте ей поработать 7–10 минут до смывания.

**Если состав стекает как вода** → проблема в оборудовании или разведении. Увеличьте концентрацию до 1:5 в пенокомплекте.

## На что смотреть при выборе

| Параметр | Что означает для вас |
|----------|----------------------|
| pH | Щелочной (10–12) мощнее, но агрессивнее. Нейтральный (7–9) безопасен для любых покрытий |
| Концентрация | 1:50 = 50 моек из литра. Чем выше — тем экономичнее |
| Совместимость | Пенокомплект требует 1:5–1:10, пеногенератор — 1:20–1:50 |

## Выбор по оборудованию

**Минимойка 100–120 бар (Kärcher, Bosch, Nilfisk)** → нейтральный состав с хорошей липкостью, разведение по инструкции.

**Пеногенератор профессиональный** → концентрат 1:20–1:50. Высокое давление взбивает густую шапку.

**Ручной распылитель** → готовый раствор (RTU). Без давления концентрат не вспенится.

## Чего избегать при выборе

**Щелочная химия для еженедельной мойки** → смоете защитный слой за 2–3 процедуры. Для регулярного ухода — только pH 7–9.

**Дешёвый концентрат с разведением 1:100** → реальная эффективность будет как у воды. Проверенный диапазон: 1:20–1:50.

## FAQ

### Какой состав выбрать для минимойки?

Нейтральный (pH 7–9) с указанием совместимости с бытовыми АВД. Разведение обычно 1:10–1:20 в пенокомплекте. При давлении ниже 100 бар густая шапка не получится — это нормально.

### Чем активная пена отличается от обычного шампуня?

Способом нанесения и задачей. Пена работает без контакта через пеногенератор, удаляя рыхлую грязь. Шампунь наносят губкой после пены для финальной очистки. Это два разных этапа мойки.

### Какой продукт взять для авто с керамикой?

Нейтральный без фосфатов (pH 7–9). Щелочной допустим при сильном загрязнении, но смывайте за 5 минут — не давайте высохнуть на кузове.

---

**Итог — что покупать:**
- **Регулярная мойка** → нейтральная пена (pH 7–9)
- **Зимняя грязь, реагенты** → щелочная (pH 10–12)
- **Керамика, воск, PPF** → нейтральная без фосфатов
