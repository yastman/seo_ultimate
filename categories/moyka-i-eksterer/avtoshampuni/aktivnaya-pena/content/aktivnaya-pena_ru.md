# Активная пена

Пена для мойки автомобиля защищает лак от микроцарапин: густой слой размягчает дорожную грязь без контакта с кузовом. Безопасная мойка минимизирует повреждения — особенно на машинах с керамикой или PPF. Наносится через пеногенератор на мойку высокого давления. Если у вас керамическое покрытие — выбирайте нейтральную формулу, для зимней грязи с реагентами — щелочную пену.

## Шампунь для бесконтактной мойки: сценарии выбора

| Состояние авто | Тип состава | Почему подходит |
|----------------|-------------|-----------------|
| Зимняя грязь, дорожные реагенты | Щелочной | Разлагает соль и масляную плёнку за несколько минут |
| Еженедельный уход | Нейтральный | Сохраняет воск, силант, керамическое покрытие |
| Авто с керамикой, PPF, свежим воском | pH-нейтральный без фосфатов | Поддерживает гидрофобность защитного слоя |
| Лёгкая пыль, птичий помёт | Любой, слабое разведение | Экономичный расход концентрата |

**Если у вас авто с керамическим покрытием** → нейтральная формула будет оптимальным выбором. Щелочные составы при регулярном применении снижают гидрофобный эффект защитного слоя.

**В зимний период с дорожными реагентами** → щелочной концентрат справляется лучше. Важно смывать до высыхания на кузове, иначе могут остаться белёсые разводы.

**Если на кузове свежий воск или силант** → нейтральная химия сохраняет защитный слой дольше. Щелочной состав при концентрированном разведении может смыть воск за несколько процедур.

## Химия для мойки авто: параметры для сравнения

| Параметр | На что влияет |
|----------|---------------|
| pH | Щелочные составы = агрессивная очистка, риск повреждения воска. Нейтральные = безопасность для любых покрытий |
| Липкость | Густая формула удерживается несколько минут на вертикальных панелях, давая время для работы |
| Расход | Зависит от оборудования и степени загрязнения — ориентируйтесь на инструкцию производителя |
| Температурный режим | На горячем кузове состав высыхает быстрее — работать в тени или охладить водой |

## Концентрат активной пены и оборудование

**Если у вас бытовая минимойка** → пена для минимойки с повышенной липкостью будет лучшим решением. При давлении ниже 100 бар получается жидкий раствор вместо густой шапки — норма для бытового оборудования.

**Профессиональное оборудование (120-150 бар)** → концентрат с разведением 1:20-1:50. Высокое давление обеспечивает густой слой, который держится 5-10 минут на вертикальных панелях. Направление нанесения — снизу вверх: гравитация помогает пене дольше работать. Расход концентрата — 200-400 мл на легковой автомобиль.

**Ручной помповый опрыскиватель (IK Foam)** → разведение слабее. Слой менее густой, но времени выдержки хватает для размягчения лёгких загрязнений.

**Готовый раствор RTU** → для распылителей без давления. Концентрат без взбивания не создаёт нужную текстуру.

## Чего избегать при выборе

**Щелочная формула для еженедельной мойки** — воск и силант смываются за несколько процедур. Для регулярного ухода подходит только нейтральный pH.

**Концентрат без указания pH на этикетке** — невозможно оценить агрессивность состава и совместимость с защитой кузова.

**Нанесение на горячий кузов** — состав высыхает до смывания, оставляя белёсые разводы. Охладите поверхность водой перед нанесением.

## FAQ

### Какой продукт выбрать для минимойки?

Нейтральный состав, совместимый с бытовыми АВД. Ориентируйтесь на липкость, а не толщину слоя — жидкая консистенция при давлении ниже 100 бар это норма.

### Чем бесконтактная химия отличается от обычного шампуня?

Способом нанесения и задачей. Бесконтактная химия наносится через АВД, работает 5-10 минут, удаляет рыхлую грязь без губки. Шампунь — контактный этап после предварительной очистки. Два разных продукта для разных фаз мойки.

### Какой продукт подходит для авто с керамикой?

pH-нейтральный без фосфатов. Щелочной допустим при сильном загрязнении — смывать за 5-10 минут, не допускать высыхания, иначе керамика теряет гидрофобность быстрее.

### Почему химия быстро стекает и не держится на кузове?

Причины: давление ниже 100 бар, слабое разведение, горячий кузов. Решение: увеличить концентрацию, охладить кузов водой, проверить давление минимойки.

---

**Итог — что покупать:**
- **Регулярная мойка** → нейтральный концентрат
- **Зимняя грязь, реагенты** → щелочная формула
- **Керамика, воск, PPF** → pH-нейтральный без фосфатов
- **Минимойка** → состав с повышенной липкостью
