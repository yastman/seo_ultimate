# Автошампуни для ручной мойки

Автошампунь для ручной мойки — концентрат для контактной мойки методом "ведро и губка". Три типа по pH: нейтральный (6-8), кислотный (3-5), щелочной/strip wash (10-12). Отдельная категория — составы с воском или SiO₂ для дополнительной защиты. Критерий выбора: состояние покрытия + цель (регулярный уход, снятие старой защиты, подготовка к полировке).

## Как выбрать: сценарии

| Задача | Что брать | Почему |
|--------|-----------|--------|
| Регулярный уход | Нейтральный pH 6-8 | Сохраняет защитные покрытия |
| Авто с керамикой | Нейтральный или с SiO₂ | Поддерживает гидрофобный слой |
| Снятие старой защиты | Strip wash pH 10-12 | Полностью удаляет воск/силант |
| Подготовка к полировке | Кислотный pH 3-5 | Убирает минеральные отложения |
| Мойка + защита | Состав с воском | Добавляет тонкий барьер |

**Защита на кузове** → только нейтральный, щелочной strip wash смоет покрытие.

**Перед нанесением новой защиты** → strip wash для "голого" лака, затем кислотный для удаления минералов.

**Еженедельный уход** → нейтральный концентрат 1:200 (литр = 50 применений).

## На что смотреть при выборе

| Параметр | Значение |
|----------|----------|
| pH 6-8 | Безопасность лака, керамики, силанта |
| Разведение 1:100-1:200 | Стандартный расход концентрата |
| Добавки (воск, SiO₂) | Дополнительный защитный барьер |
| Пенообразование | Удержание грязи в суспензии |

Смазывающие компоненты (lubricity) обеспечивают скольжение рукавицы по лаку без царапин. Двухфазная мойка методом двух вёдер снижает риск swirl-эффекта: wash bucket с раствором, rinse bucket — ополаскивание губки.

## Выбор по pH

| Тип | pH | Когда использовать | Особенность |
|-----|----|--------------------|-------------|
| Нейтральный | 6-8 | Регулярный уход, защищённый кузов | Не смывает защитные слои |
| Кислотный | 3-5 | После щелочной пены, перед защитой | Удаляет минеральные отложения |
| Щелочной (strip wash) | 10-12 | Снятие старого воска/силанта | Оголяет лак полностью |

**Регулярные мойки** → нейтральный, безопасен для любых покрытий.

**Полная подготовка к защите** → strip wash + кислотный последовательно.

## Выбор по добавкам

| Добавка | Эффект | Кому подходит |
|---------|--------|---------------|
| Базовый (без добавок) | Только очистка | Подготовка перед полировкой |
| С карнаубой | Глянец + тонкий барьер | Регулярный уход без отдельного нанесения |
| С SiO₂ | Гидрофобность | Владельцы керамики |
| С полимерами | Усиленный блеск | Выставочный вид |

**Глянец приоритет** → продукт с полимерами (gloss enhancer).

**Полировка запланирована** → базовый без добавок.

## Выбор по концентрации

| Тип | Разведение | Расход на мойку | Когда брать |
|-----|------------|-----------------|-------------|
| Высококонцентрированный | 1:200-1:500 | 5-10 мл | Регулярные мойки, профессионалы |
| Стандартный | 1:50-1:100 | 15-30 мл | Еженедельный уход |
| Ready-to-use (RTU) | Готовый | 200-300 мл | Редкие мойки, новички |

**Экономия приоритет** → концентрат 1:500 (литр = 100+ моек).

**Удобство приоритет** → готовый раствор без расчётов.

## Чего избегать при выборе

**pH >10 для регулярной мойки** → щелочной состав смоет защиту за 2-3 недели.

**Средство для посуды** → Fairy (pH >10) разрушает защитные покрытия, даёт тусклую поверхность.

Автошампунь с воском добавляет слой 0.5 мм vs 2-3 мм при отдельном нанесении воска. Полноценную защиту не заменяет.

## FAQ

### Какой шампунь для ручной мойки лучше?

Регулярный уход — нейтральный концентрат pH 6-8, разведение 1:150. Керамическое покрытие — аналогичный pH, опционально добавка SiO₂ для гидрофобности.

### Чем отличается от бесконтактного?

Шампунь для ручной мойки: ведро + рукавица (контактный метод), pH нейтральный 6-8. Активная пена: пеногенератор (бесконтактно), pH щелочной 10-12.

### Нужно ли разводить шампунь?

Обязательно. Стандарт 1:100-1:200. На 10-литровое ведро — 50-100 мл концентрата. Крепкий раствор (1:20-1:50) оставляет разводы.

### Как мыть машину вручную?

Two-bucket method с grit guard: wash bucket (раствор), rinse bucket (чистая вода). Ополаскивание рукавицы после каждой панели. Движения сверху вниз, волнообразные — не круговые.

**Итог — что покупать:**
- **Регулярный уход** → нейтральный pH 6-8
- **Керамика/воск** → нейтральный или с SiO₂
- **Снятие старой защиты** → strip wash pH 10-12
- **Подготовка к полировке** → кислотный pH 3-5
- **Мойка + защита** → состав с воском
