# Шампуни для ручной мойки

Автошампунь для ручной мойки — pH-нейтральное средство для контактной обработки кузова. Создаёт скользящий слой между варежкой и лаком: песчинки скользят, а не царапают ЛКП.

## Как выбрать: сценарии

| Тип | pH | Назначение | Особенности |
|-----|-----|------------|-------------|
| Нейтральный (Pure) | 7 | Регулярный уход, авто с керамикой | Не оставляет плёнки |
| С добавками (Wax/SiO₂) | 7–8 | Блеск и временный гидрофоб | Помыл и обработал за один шаг |
| Обезжиривающий (Strip Wash) | 9–10 | Подготовка к полировке | Смывает старые силанты |

**Авто с керамикой** → нейтральный автошампунь для ручной мойки (pH 7). Безопасен для слоя.

**Хотите гидрофоб сразу** → шампунь для ручной мойки с SiO₂. Эффект на 1–2 недели.

**Перед полировкой** → Strip Wash (pH 9–10). Удаляет всё, что мешает адгезии.

## На что смотреть

| Параметр | Значение |
|----------|----------|
| pH | 7 — безопасен для керамики и силантов. 9-10 — для подготовки |
| Лубрикация | Выше показатель — меньше царапин при контакте |
| Добавки | SiO₂ даёт гидрофоб. Без добавок — чистая очистка |

## Выбор по типу обработки авто

| Обработка | Что брать | Причина |
|-----------|-----------|---------|
| Керамика | Pure-шампунь (pH 7) | Добавки забивают поры |
| Силант | Любой с pH 7 | Сбалансированный pH безопасен |
| Без обработки | С SiO₂ | Получите базовую гидрофобность |
| Перед полировкой | Strip Wash | Нужна чистая поверхность |

## Метод двух вёдер: зачем нужна лубрикация

Два ведра — золотой стандарт ручной мойки: одно с шампунем, второе для ополаскивания варежки. Грязь остаётся во втором ведре, а не переносится обратно на кузов.

**Почему это важно для выбора шампуня:** высокая лубрикация позволяет песчинкам скользить по лаку вместо того, чтобы царапать его. Дешёвые составы без смазывающих добавок превращают даже правильную технику в лотерею.

**Grit Guard** — сетка на дне ведра. Задерживает песок внизу, не даёт ему подняться при полоскании варежки.

## Чего избегать

**Средство для посуды (Fairy)** → pH 10-11, агрессивные ПАВ. Высушивает резинки (белеют), убивает силанты за одну процедуру.

**Strip Wash для регулярного ухода** → смоете керамику/силант. Strip Wash — только перед полировкой.

**Составы без лубрикации** → дешёвые шампуни плохо скользят, больше микроцарапин даже при правильной технике.

## FAQ

### Можно ли мыть машину средством для посуды?

Нет. Fairy содержит агрессивные жироудалители и соли. Высушивают резиновые уплотнители (белеют), мгновенно смывают силанты и керамику. Автомобильный шампунь для ручной мойки создан с учётом лубрикации и pH-баланса.

### Какой шампунь для ручной мойки лучше?

Регулярный уход — нейтральный pH 7. Авто без обработки — с SiO₂ (гидрофоб на 1-2 недели). Керамика — нейтральный без добавок.

---

**Итог — что покупать:**
- **Регулярный уход** → нейтральный автошампунь для ручной мойки (pH 7)
- **Авто без обработки** → с SiO₂
- **Керамика** → нейтральный без добавок
