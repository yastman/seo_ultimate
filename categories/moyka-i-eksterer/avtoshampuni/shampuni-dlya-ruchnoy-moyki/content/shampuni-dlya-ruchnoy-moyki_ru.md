# Автошампунь для ручной мойки

Автошампунь для ручной мойки — концентрат для безопасной контактной очистки кузова методом «ведро + губка». В отличие от бытовых средств, автомобильный шампунь для ручной мойки имеет нейтральный pH и содержит смазывающие компоненты — губка скользит по краске без царапин. Выбор зависит от типа защитного покрытия и задачи: регулярный уход или подготовка к полировке.

## Как выбрать шампунь для ручной мойки авто

| Тип по pH | Особенности | Для кого |
|-----------|-------------|----------|
| Нейтральный (pH 6-8) | Безопасен для воска, керамики, силанта | Регулярная мойка, авто с защитой |
| Кислотный (pH 3-5) | Удаляет минеральные отложения | Подготовка перед полировкой или нанесением керамики |
| С воском | Добавляет тонкий защитный слой | Еженедельный уход с дополнительным блеском |
| С SiO2 (керамический) | Гидрофобный эффект после мойки | Владельцы керамических покрытий |

**Сценарии:**
- **Если авто с керамикой** → нейтральный или с SiO2 — сохраняет покрытие
- **Если нужна подготовка к защите** → кислотный — нейтрализует остатки щёлочи после pre-wash
- **Если хочется «2 в 1»** → с воском — моет и добавляет блеск за один этап
- **Если важна экономия** → высококонцентрированный нейтральный (разведение до 1:200)

### Чем автошампунь отличается от бесконтактной пены

| Параметр | Шампунь для ручной мойки | Активная пена |
|----------|--------------------------|---------------|
| Способ нанесения | Ведро + рукавица (контакт) | Пеногенератор (без контакта) |
| Этап мойки | Второй (основной) | Первый (подготовительный) |
| pH | Обычно нейтральный | Щелочной или нейтральный |
| Что делает | Финальная очистка | Размягчает грязь перед контактом |

## Как применять: метод двух вёдер

Метод двух вёдер (two-bucket method) — стандарт ручной мойки. Грязь с губки не возвращается в чистый раствор, царапины исключены.

1. **Подготовьте два ведра** — в одно налейте воду с шампунем, во второе — чистую воду для ополаскивания. На дно обоих установите грязеотделитель (grit guard)
2. **Смочите рукавицу** в ведре с шампунем, вымойте одну панель кузова
3. **Ополосните рукавицу** в ведре с чистой водой, потрите о грязеотделитель
4. **Повторите** — снова в шампунь, следующая панель. Двигайтесь сверху вниз
5. **Финальное ополаскивание** — смойте остатки шампуня чистой водой

**Ошибки:**
- **Мыть одной губкой кузов и пороги** — грязь с низа переносится наверх, появляются царапины
- **Круговые движения** — оставляют характерные круговые царапины (swirl marks). Двигайтесь прямыми линиями
- **Мойка на солнце** — шампунь высыхает до смывания, остаются разводы

## На что смотреть при выборе

| Маркер на этикетке | Что означает | Кому подходит |
|--------------------|--------------|---------------|
| pH-neutral / pH 7 | Не смывает воск и керамику | Авто с любым защитным покрытием |
| Wax / Carnauba | Содержит воск для дополнительного блеска | Регулярный уход без отдельного воскования |
| SiO2 / Ceramic | Добавляет гидрофобность | Керамические покрытия |
| Концентрат 1:100-1:200 | Экономичный расход | Частая мойка, профессионалы |

## FAQ

### Какой шампунь выбрать для авто с керамикой?

Нейтральный (pH 6-8) или специальный с SiO2. Керамика выдерживает широкий диапазон pH, но нейтральный состав оптимален для регулярного использования — не ускоряет износ покрытия.

### Почему нельзя мыть машину средством для посуды?

Бытовые средства имеют щелочной pH (выше 10) и смывают защитные покрытия — воск, силант, со временем и керамику. Автошампунь разработан с учётом совместимости с автомобильными поверхностями.

### Как разводить концентрат?

Зависит от степени загрязнения: для лёгкой грязи — более разбавленный раствор, для сильной — менее разбавленный. Ориентируйтесь на рекомендации производителя на этикетке.

### Зачем нужен грязеотделитель (grit guard)?

Пластиковая решётка на дне ведра удерживает частицы грязи внизу. Рукавица не собирает их со дна при следующем погружении — меньше риск поцарапать краску.

### Шампунь с воском заменяет отдельное воскование?

Частично. Шампунь с воском добавляет тонкий защитный слой и блеск, но полноценное воскование даёт более стойкий результат. Для регулярного ухода между основными обработками — вполне подходит.
