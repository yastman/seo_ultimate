# Шампуни для ручной мойки

Автошампунь для ручной мойки — концентрат с нейтральным pH для контактной обработки кузова методом «ведро и губка». Главное отличие от бытовых средств: высокая лубрикация (смазывающий эффект) позволяет песчинкам скользить по лаку, а не царапать его.

## Как выбрать: сценарии

| Ситуация | Что брать | Почему |
|----------|-----------|--------|
| Авто с керамикой | Нейтральный без добавок (pH 7) | Добавки забивают поры покрытия |
| Авто с воском/силантом | Нейтральный или с воском | Сохраняет защитный слой |
| Без защитного покрытия | Шампунь с SiO₂ | Получите базовый гидрофоб на 1-2 недели |
| Подготовка к полировке | Strip Wash (pH 9-10) | Смывает старые слои, поверхность готова к адгезии |
| Еженедельный уход | Нейтральный концентрат | Расход 15 мл на ведро, экономия до 50 процедур с литра |

**Если авто с керамикой** → только нейтральный pH 7, без добавок. Другие составы могут повлиять на гидрофобность.

**Если хотите блеск сразу после обработки** → шампунь для ручной мойки авто с воском или SiO₂. Эффект временный (до следующей процедуры), но визуально заметный.

**Если готовите авто к нанесению защиты** → Strip Wash. Удаляет всё, что мешает адгезии нового покрытия.

## На что смотреть при выборе

| Параметр | Что означает для покупателя |
|----------|------------------------------|
| pH | 7 — безопасен для любых покрытий. 9-10 — только для подготовки |
| Лубрикация | Выше показатель — меньше микроцарапин при контакте варежки с кузовом |
| Разведение | 1:100-1:200 стандарт. Высококонцентрированные (1:500) экономичнее |
| Добавки | SiO₂ — гидрофоб. Воск — блеск. Без добавок — чистая очистка |

## Чего избегать при выборе

**Если планируете керамику** → не берите составы с воском. Воск создаёт плёнку, керамика не схватится.

**Если у вас силант** → избегайте Strip Wash для регулярного ухода. Это средство для подготовки, смоет защиту за одно применение.

**Если нужна лубрикация** → не экономьте на дешёвых составах. Автомобильный шампунь без смазывающих добавок превращает даже правильную технику в лотерею с царапинами.

**Средство для посуды (Fairy)** — не альтернатива. pH 10-11 и агрессивные ПАВ высушивают резинки (белеют), убивают силанты за одну процедуру.

## FAQ

### Какой шампунь для ручной мойки лучше?

Зависит от задачи. Регулярный уход — нейтральный pH 7, расход 15-20 мл на ведро. Авто без обработки — с SiO₂ для базовой гидрофобности. Керамика — нейтральный без добавок, чтобы не забивать поры покрытия.

### Чем отличается от бесконтактного?

Бесконтактная пена (pre-wash) — щелочной состав pH 10-12 для первой фазы, работает без контакта. Автошампунь для ручной мойки — нейтральный pH 7, для второй фазы с варежкой. Пена убирает 70-80% грязи, шампунь — финальная очистка.

### Нужно ли разводить шампунь?

Да, концентрат разводится в ведре. Стандарт: 1:100-1:200 (15-20 мл на ведро воды). При сильном загрязнении — 1:50, при лёгком — 1:300. Готовые RTU-составы дороже в пересчёте на одну процедуру.

### Как мыть машину вручную?

Метод двух вёдер (two-bucket method): одно с шампунем, второе для ополаскивания варежки. Grit Guard на дне удерживает песок. Мыть сверху вниз, движения волнообразные. После каждой панели — ополоснуть варежку в чистом ведре.

**Итог — что покупать:**
- **Регулярный уход** → нейтральный автошампунь для ручной мойки (pH 7)
- **Авто без обработки** → шампунь для ручной мойки авто с SiO₂
- **Керамика** → нейтральный без добавок
- **Подготовка к полировке** → Strip Wash (pH 9-10)
