## Блок 1: Что это и зачем

### Определение
**Автошампунь** — специализированное концентрированное или готовое моющее средство с ПАВ, спиртом и добавками, разработанное для безопасной очистки автомобилей без повреждения ЛКП и защитных слоёв[1][2][3].

### Отличие от бытовых средств

| Параметр | Бытовое (Fairy, Мистер Мускул) | Автошампунь |
|----------|--------------------------------|------------|
| pH | 10-11 (щелочное) | 6-8 (нейтральное) |
| ПАВ | Агрессивные, разрушающие жиры | Сбалансированные |
| Воск/керамика | Смывает за 1 мойку | Безопасен |
| Остатки | Липкие, притягивают грязь | Чистое смывание |

**Почему нельзя мыть авто бытовым мылом:** Щелочь (pH >10) разрушает сальный слой ЛКП, деградирует воск полностью за одну мойку, разрушает керамику[3][4][23]. Научный тест: clearcoat деградирует за 24 часа при контакте с бытовой химией[23].

### Главное правило выбора
- **pH 6-8 для безопасности всех покрытий**[1][2][5][21]
- **БЕЗ воска, если авто имеет защиту**[20]

### Ось 1: По этапу мойки

| Тип | Назначение | Когда выбирать | pH |
|-----|-----------|---------------|----|
| Pre-wash (пена) | Первый этап, бесконтактно | Сильное загрязнение, зима | 10-12 |
| Ручной шампунь | Второй этап, контактный | Еженедельная мойка | 6-8 |

| Кислотный | Нейтрализация после щелочи | После pre-wash | 2-5 |

**ВАЖНО:** Pre-wash и ручной шампунь — это **разные этапы**, используются **последовательно**[10][14][19].

| Тип | pH | Плюсы | Минусы |
|-----|----|----|-------|
| Щелочной | 10-14 | Мощная очистка | ⚠️ Смывает воск за 1 мойку[22] |
| Нейтральный | 6-8 | ✅ Безопасен для всех, каждый день | Слабее на грязи |
| Кислотный | 2-5 | Нейтрализует щелочь | Только после щелочного |

### Ось 3: По форме

| Тип | Особенности | Выбирать когда |
|-----|-------------|-----------------|

## Блок 3: Как выбрать (сценарии)

### Сценарий 1: Авто с керамикой
**Рекомендация:** pH-нейтральный БЕЗ воска[5][20][21][30]  
**Частота:** 2 недели (идеально), месяц (минимум)[26][29][38][44][48]

### Сценарий 2: Новичок без защиты
**Рекомендация:** pH-нейтральный С воском/SiO2[1][2][21]

### Сценарий 3: Сильное загрязнение
**Рекомендация:** Pre-wash (щелочной) + ручной шампунь (нейтральный)[3][14]

### Сценарий 4: Еженедельная мойка
**Рекомендация:** Один pH-нейтральный концентрат[1][2]

### Сценарий 5: Профессиональная мойка
**Рекомендация:** Koch-Chemie Active Foam + специализированный шампунь[8][30]  
**Разведение:** 1:5-1:20 для пены[8][9]

## Блок 4: Как применять

### Pre-wash (активная пена)

| Шаг | Действие | Почему |
|-----|---------|--------|
| 1. Оборудование | Пенная пушка (foam cannon) | Требуется для создания пены[8][9][14] |
| 2. Разведение | 1:5-1:50 в зависимости от грязи | Проверить инструкцию[8][9] |
| 3. Нанесение | Снизу вверх (колеса → верх) | Грязь падает вниз[10][14] |
| 4. Выдержка | 5-15 минут (пока пушистая) | До опадания пены[8][9][14] |
| 5. Смывание | Мойка высокого давления | Полное смывание щелочи[10][14] |

### Ручной шампунь

#### Подготовка вёдер (TWO BUCKET METHOD)

| Компонент | Назначение | Почему |
|-----------|-----------|--------|
| Ведро 1 (Wash) | Шампунь + вода (1:10-1:30) | Чистый раствор |
| Ведро 2 (Rinse) | Чистая вода | Ополаскивание варежки |
| Grit Guard | Сепаратор в каждое ведро | Грязь не всплывает, остаётся внизу[10][12][13] |

**Результат:** Предотвращение царапин на 90%[12].

#### Разведение концентрата

| Тип авто | Разведение | Расход (в 15л ведро) |
|----------|-----------|-------------------|
| Без защиты, лёгкое | 1:20-1:30 | 20-30 мл |
| Среднее загрязнение | 1:10-1:15 | 40-50 мл |
| С керамикой | 1:15-1:20 | 30-40 мл |
| Очень грязное | 1:5-1:10 | 75-100 мл |

**Реальные расходы:**
- Kärcher RM 562: 150 мл в 5л (1:33.3)[53]
- Sonax: 10-15 мл per vehicle[52]
- Professional brush: 5-13 мл per car[51]
- Cherry Burst: 1.7p-8.6p per wash[50]

#### Последовательность (сверху вниз)

1. Крыша → капот → окна
2. Двери (передние → задние)
3. Юбки, пороги, колёса

#### Техника варежкой

1. Обмакнуть в Wash bucket
2. Помыть одну панель
3. Ополоснуть в Rinse bucket (конец в воду, работать на дне)
4. Отжать в Rinse bucket
5. Вернуться в Wash bucket (варежка чистая)

#### Смывание
- Мойка высокого давления снизу вверх
- Все пузырьки удалены (остатки = разводы)[1][11]

#### Сушка (Pat-and-Pull)
- Микрофибра мягкая
- Промокнуть (не тереть!)
- В тени (солнце = разводы)[15][16][55]
- Время: 20-40 минут

## Блок 5: Критические ошибки

| Ошибка | Последствие | Решение |
|--------|------------|---------|
| Мойка на солнце | Разводы, остатки щелочи | Мойте в тени, кузов 20-30°C[15][16][55] |
| Бытовая химия | Смыл воск, разрушил керамику | ТОЛЬКО автошампуни pH 6-8[3][4] |
| Передержка пены | Щелочь разрушает ЛКП | Смывайте за 5-15 минут[8][9][14] |
| Слишком крепкое разведение | Остатки ПАВ, липкая поверхность | 1:10-1:20 (лучше слабее)[1][11] |
| Мойка сверху вниз БЕЗ pre-wash | Грязь падает на чистые зоны | Ополоскать водой сначала[10][14] |
| Одна варежка | Грязь накапливается | Две варежки или частая смена[11][14] |
| Одно ведро | Грязь рециркулирует | **Two bucket с Grit Guard ОБЯЗАТЕЛЕН**[10][12][13] |
| Холодная вода на горячий кузов | Тепловой шок | Холодная вода или остудить кузов[15][55] |

## Блок 7: FAQ новичка

| Вопрос | Ответ |
|--------|-------|
| **Чем отличается автошампунь?** | pH 6-8 (нейтральный), безопасен для защиты, содержит смазывающие добавки[1][2][3] |
| **Можно ли средством для посуды?** | НЕТ. pH 10-11, смывает воск за одну мойку[3][4][20] |

| **Нужен отдельный шампунь для керамики?** | Нет, любой pH-нейтральный подходит. БЕЗ воска[20][21][30] |
| **Как разводить концентрат?** | 1:10-1:20 (проверить упаковку, разные бренды разные)[8][9][52] |
| **Можно зимой мыть?** | ДА, но осторожно. Тёплая вода, быстрая сушка[15][16] |
| **Что такое метод двух вёдер?** | Wash bucket (шампунь) + Rinse bucket (вода) + Grit Guard[10][12][13] |

| **Воск в шампуне — хорошо?** | ТОЛЬКО БЕЗ защиты. С керамикой — плохо (забивает на 80%)[20][21] |
| **Пену как единственное средство?** | Технически да, но не рекомендуется. После пены нужен шампунь для финала[1][14] |

| Пена быстро опадает | Слабое разведение, холодная вода | Больше концентрата, тёплая вода[8][9] |
| Шампунь не пенится | Старый концентрат, жёсткая вода | Проверить дату, добавить концентрат, мягкая вода[1][9] |
| Water spots остаются | Жёсткая вода (Ca, Mg) | DI вода, сушка в тени, water softener[37][40][49] |
| Матовые участки | Остатки ПАВ, высыхание на солнце | Повторное смывание, тень[1][15][16] |
| Шампунь не смывает | Слабое разведение, холодная вода | Больше концентрата, тёплая вода, для грязи — pre-wash[1][9][14] |
| Липкие остатки | Неправильное разведение или неавто средство | Повторная мойка, автошампунь[1][4][11] |
| Воск/керамика "исчезла" | Щелочь или бытовое средство | Переанести защиту, использовать pH-нейтральный[3][4][20] |

## Notes

### Что НЕ удалось найти
1. Точные долгосрочные данные (>5 лет) по воздействию щелочной пены на керамику
2. Лабораторное сравнение SiO2 в шампуне vs проф. SiO2 покрытие
3. Экологическое воздействие разных ПАВ на конкретные экосистемы

### Противоречивые данные (разрешено)
1. **Щелочной и воск:** Одни говорят "смывает за одну", другие "можно при слабом разведении". **Консенсус:** Да смывает, скорость зависит от концентрации[22].
2. **Двухвёдерный метод:** Некоторые используют Grit Guard, другие просто вёдра. **Истина:** Grit Guard эффективнее[12].
3. **pH для керамики:** Керамика выдерживает pH 1-13, но 6-8 безопаснее[30][31].

### Выводы для Buyer Guide (категории)

#### 1. Для новичка (~$10-15)
- pH-нейтральный концентрат (1:10-1:20)
- С воском/SiO2 (базовая защита)
- Примеры: Chemical Guys Maxi-Suds, Turtle Wax

#### 2. Для керамики (~$20-30)
- pH-нейтральный БЕЗ воска
- Сульфат-фри
- Примеры: Infinity Wax Synergy, Carpro Reset, Gyeon Restart

#### 3. Для сильного загрязнения
- Pre-wash (щелочной pH 10-12)
- + Ручной шампунь (нейтральный pH 6-8)
- + Кислотный (опционально)
- Примеры пены: Koch-Chemie Active Foam[8]

#### 4. Для еженедельной мойки
- Один pH-нейтральный концентрат (1:10-1:20)
- Двухвёдерный метод + Grit Guard
- Примеры: Chemical Guys, Turtle Wax, любой нейтральный

#### 5. Инструменты ОБЯЗАТЕЛЬНО
- Две варежки (кузов + колёса)
- Два ведра с Grit Guard в каждом[10][12][13]
- Микрофибра для сушки (мягкая, качественная)
- Пенная пушка (если используете pre-wash)
