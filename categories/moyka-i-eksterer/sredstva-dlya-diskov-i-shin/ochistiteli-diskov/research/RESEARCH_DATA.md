# Research: Очистители дисков (ochistiteli-diskov)

## 1. Базовая матрица поиска

| Данные | Результат |
| :--- | :--- |
| **Что это** | Химия для удаления тормозной пыли (железа) и дорожной грязи с дисков. |
| **Для чего** | Очистка от "въевшегося" нагара, который не берет обычный шампунь. |
| **Классификация** | **Индикаторные:** меняют цвет (фиолетовый). **Бескислотные:** безопасные. **Кислотные:** для старых запущенных дисков. |
| **Расход** | **100-150 мл** на комплект из 4-х колес. |
| **Инструкция** | 1. Остудить диски. 2. Распылить на сухую поверхность. 3. Подождать 2-5 мин (до красного цвета). 4. Смыть водой. |
| **Совместимость** | Нейтральные очистители безопасны для всех типов дисков и суппортов. Кислотные — опасны для хрома. |
| **Ошибки** | Нанесение на горячие тормоза, высыхание состава, использование кислот на неподходящих дисках. |
| **Критерии выбора** | Наличие цветовой индикации, pH-баланс, безопасность для декоративных покрытий. |

## 2. Углубление для детейлинга (E-E-A-T)

*   **Расход:** Примерно 100-150 мл на комплект из 4-х колес при средней степени загрязнения. Флакона 500 мл хватает на 3-4 полные обработки.
*   **Оборудование:** Триггерный распылитель, Ершик для внутренней части диска, Мягкая кисть для лицевой части.
*   **Safety / Ограничения:** Категорически запрещено наносить на горячие диски — риск термического шока металла и пригорания химии.
*   **Эффективность:** Удаляет металлические вкрапления химическим путем (растворением).

## 3. Полезные факты и лайфхаки

*   **Факт (Indicator):** "Кровавый" эффект индикатора — это реакция состава с вкраплениями железа. Это визуальный контроль: если цвет не изменился, значит железа на дисках нет.
*   **Лайфхак (Dry vs Wet):** На форумах советуют наносить индикаторные составы именно на **сухой холодный** диск. Вода разбавляет состав и снижает время контакта активных веществ с тормозной пылью.
*   **Лайфхак (Iron as Status):** Очиститель дисков работает как индикатор состояния суппортов. Если на одном колесе "крови" всегда гораздо больше — возможно, этот суппорт подклинивает и колодка изнашивается быстрее.
*   **Лайфхак:** Всегда мойте колеса первыми. Если мыть их после кузова, грязная вода и химия неизбежно полетят на чистый лак.
*   **Гайд (микро):** При мойке используйте 2 инструмента: жесткий ершик для барабана диска и мягкую кисть для спиц и болтов.

## 4. Меры предосторожности

*   **Температура:** Никогда не наносите химию на раскаленные тормозные диски. Это может привести к их деформации или выгоранию краски суппортов.
*   **Керамика:** На карбон-керамических тормозах использовать только pH-нейтральные составы без избыточных ПАВ.
