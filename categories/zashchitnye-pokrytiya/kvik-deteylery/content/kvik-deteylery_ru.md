# Полимер для авто

Полимер для авто (квик-детейлер) — спрей для экспресс-ухода за лакокрасочным покрытием. Убирает лёгкую пыль, восстанавливает блеск после мойки, поддерживает защитные покрытия. Технология работы: распылили на участок 50×50 см, протерли микрофиброй GSM 300-400, отполировали сухой салфеткой.

## Виды полимеров для авто

| Тип | Основа | Стойкость | Применение |
|-----|--------|-----------|------------|
| **Холодный воск** | Полимеры + силиконы | 3-7 дней | Регулярный уход между мойками, восстановление блеска |
| **Горячий состав** | Натуральный воск + ПАВ | До 14 дней | Финишный слой на автомойке, усиленная гидрофобность |
| **Быстрый блеск** | SiO2 (кварц) | До 30 дней | Поддержка керамических покрытий, максимальный глянец |

## Как выбрать полимер для авто

### Уборка пыли без мойки
Берите холодный воск для автомобиля с маркировкой «dust removal». Содержит добавки, поднимающие частицы с поверхности и связывающие их в микрофибре. Обычные средства царапают ЛКП при протирании по сухому кузову.

### Машина под открытым небом
Формулы с УФ-фильтрами и гидрофобными добавками. Холодный воск для авто создаёт плёнку, отталкивающую воду и снижающую воздействие солнца. Обновление раз в неделю сохраняет защитные свойства.

### Керамическое покрытие
Быстрый блеск для авто на базе SiO2. Совместим с керамикой, не перекрывает свойства основного покрытия, добавляет скольжения. Применение после каждой мойки продлевает жизнь керамике.

### Подготовка авто к выставке
Горячий воск для авто даёт максимальный глянец при нанесении на влажную поверхность. Технология: нанесение под давлением при 40-60°C — формула растекается равномерно и проникает в микропоры ЛКП.

## Сравнение с альтернативами

| Продукт | Время работы | Стойкость | Блеск | Сценарий |
|---------|--------------|-----------|-------|----------|
| Полимер (спрей) | 5-10 мин | 1-4 недели | Средний-высокий | Поддержание чистоты, экспресс-уход |
| Спрей-защита | 10-15 мин | До 8 недель | Высокий | После полировки, временная защита |
| Твёрдый состав | 30-60 мин | 2-4 месяца | Глубокий | Подготовка к сезону, долгая защита |
| Керамический спрей | 15-20 мин | До 6 месяцев | Очень высокий | Альтернатива нанокерамике |

## Технология нанесения

1. **Мойка и сушка.** Грязь на кузове размазывается полимером и царапает ЛКП.
2. **Встряхивание баллона.** Распыление на участок 50×50 см с расстояния 20-30 см.
3. **Протирание микрофиброй** (GSM 300-400) круговыми движениями.
4. **Полировка второй сухой салфеткой** до равномерного блеска.

Работа на солнце и горячем кузове даёт разводы — полимер высыхает до полировки.

## Холодный vs горячий воск

Холодный воск для автомобиля — полимерный спрей для ручного нанесения. Работает при любой температуре, подходит для домашнего ухода. Лёгкая защитная плёнка держится 5-10 моек.

Горячий состав — концентрированная эмульсия для автоматических моек. Нанесение под давлением в горячей воде (40-60°C) обеспечивает проникновение в микропоры ЛКП. Стойкость выше, требуется специальное оборудование.

Комбинирование: горячая формула на мойке как основа, холодная — поддержка между визитами.

## Совместимость с покрытиями

| Поверхность | Применение | Примечания |
|-------------|------------|------------|
| Глянцевая краска | Да | Основное назначение |
| Керамика | Да | Составы с маркировкой «ceramic safe» |
| PPF (плёнка) | Да | Сохранение свойств плёнки |
| Матовая краска | Нет | Жирные пятна, нужны специальные средства |
| Винил глянец | Да | Освежение цвета, дополнительная защита |
| Винил мат | Нет | Изменение структуры поверхности |

Стёкла обрабатывайте отдельными стеклоочистителями — полимеры для кузова дают жирную плёнку и блики.

## Частые вопросы

**Можно ли заменить мойку квик-детейлером?**
Нет. Быстрый защитный спрей убирает лёгкую пыль. Грязь, песок, реагенты смываются водой под давлением — протирание царапает ЛКП.

**Частота использования?**
После каждой мойки или раз в неделю для поддержания блеска. Машина в гараже — раз в 2 недели.

**Причины разводов?**
Нанесение на горячий кузов, грязная микрофибра, избыток продукта. Работайте в тени, меняйте салфетки, наносите тонким слоем.

**Применение на пластике и резине?**
Гладкий пластик — да, белые следы на резиновых уплотнителях. Избегайте неокрашенных молдингов и шин.

**Отличие от спрей-воска?**
Холодный воск для авто — упрощённая формула для экспресс-ухода. Спрей-защита содержит натуральные компоненты, даёт глубокий блеск, держится дольше, требует 10-15 минут полировки.
