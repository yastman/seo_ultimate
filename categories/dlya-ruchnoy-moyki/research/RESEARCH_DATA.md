# Research: Шампунь для ручной мойки (dlya-ruchnoy-moyki)

## 1. Базовая матрица поиска

| Данные | Результат |
| :--- | :--- |
| **Что это** | Специализированный шампунь для второй (контактной) фазы мойки с высокими смазывающими свойствами. |
| **Для чего** | Удаление статических загрязнений после бесконтактной фазы без риска поцарапать лак. |
| **Классификация** | **Чистые (нейтральные):** базовые. **С воском/полимерами:** блеск. **С керамикой (SiO2):** гидрофоб. |
| **Расход** | **20-50 мл** концентрата на 10 литров воды (зависит от жесткости воды). |
| **Инструкция** | 1. Предварительная мойка. 2. Метод 2-х ведер. 3. Мыть сверху вниз мягкой варежкой. 4. Смыть и высушить. |
| **Совместимость** | Нейтральные составы (pH 7) безопасны для всех типов защитных покрытий. |
| **Ошибки** | Метод "в одно ведро", использование бытовых средств, мойка под солнцем. |
| **Критерии выбора** | Степень скольжения (лубрикация), обилие пены, pH-баланс. |

## 2. Углубление для детейлинга (E-E-A-T)

* **Расход:** Очень экономичен. Обычно 20-50 мл концентрата на 10 литров воды. Одного флакона 500 мл хватает на 15-20 моек.
* **Оборудование:** Мягкая варежка (микрофибровая или овечья шерсть), 2 ведра (одно с шампунем, другое для полоскания), Решетка-фильтр (Grit Guard) на дно ведра.
* **Safety / Ограничения:** Не допускать высыхания шампуня на кузове. Мыть только в тени.
* **Эффективность:** Удаляет тонкую статическую пленку грязи, которую "бесконтакт" не берет на 100%.

## 3. Полезные факты и лайфхаки

* **Факт:** Бытовые моющие средства (для посуды) со временем разрушают восковую защиту и сушат резиновые уплотнители.
* **Лайфхак (Grit Guard Hack):** Всегда используйте 2 ведра и решетки-сепараторы на дно. Грязь с варежки оседает под решеткой и не возвращается обратно на лак при полоскании.
* **Лайфхак (Rinseless Wash):** Изучите составы типа Optimum No Rinse (ONR). Это "швейцарский нож" детейлинга: можно использовать как шампунь, лубрикант для глины или средство для быстрой очистки помета.
* **Лайфхак:** Для сушки используйте **огромное** микрофибровое полотенце, промакивая, а не вытирая поверхность.
* **Гайд (микро):** Мойте автомобиль сверху вниз. Крыша — первая, пороги — последние (самая грязная часть варежки не должна касаться верхних панелей).

## 4. Меры предосторожности

* **Песок:** Если варежка упала на землю — немедленно замените ее. Одна песчинка под давлением руки при мойке создаст глубокую царапину.
* **Стекло:** При использовании шампуня "с керамикой" будьте готовы к возможным налетам на стекле (протереть очистителем стекол).
