# Research: Очистители шин (ochistiteli-shin)

## 1. Базовая матрица поиска

| Данные | Результат |
| :--- | :--- |
| **Что это** | Сильнодействующая щелочная химия для глубокой очистки резины. |
| **Для чего** | Удаление коричневого налета (окисления) и старых чернителей из пор резины. |
| **Классификация** | **Пенные:** удобны. **Концентраты:** экономичны для проф. использования. |
| **Расход** | **100-200 мл** готового раствора на комплект из 4-х шин. |
| **Инструкция** | 1. Смойте рыхлую грязь. 2. Нанесите очиститель. 3. Потрите жесткой щеткой. 4. Смывайте до чистой воды. |
| **Совместимость** | Безопасны для резины. Щелочь может влиять на незащищенный алюминий дисков. |
| **Ошибки** | Нанесение на горячие шины, отсутствие механического воздействия (щетки), высыхание состава. |
| **Критерии выбора** | Способность удалять силикон, наличие индикатора (коричневая пена). |

## 2. Углубление для детейлинга (E-E-A-T)

*   **Расход:** Около 100-200 мл готового раствора на 4 колеса. Концентраты 1:10 очень экономичны.
*   **Оборудование:** Жесткая щетка для шин (Short Handle Tire Brush), Распылитель.
*   **Safety / Ограничения:** Работать в перчатках (щелочная среда). Не допускать попадания на кожаные элементы салона при чистке ковриков.
*   **Эффективность:** Без глубокой очистки новый чернитель продержится всего несколько дней. Очищенная шина — залог адгезии состава.

## 3. Полезные факты и лайфхаки

*   **Факт:** Коричневый налет на шинах (Blooming) — это результат выхода антиоксидантов на поверхность резины. Это естественный процесс защиты, но он портит вид и мешает чернителю.
*   **Лайфхак (Scrubbing Secret):** Потрите мокрую боковину шины жесткой щеткой с составом. Если пена стала бурой — значит грязь еще есть. Повторяйте процедуру до тех пор, пока пена на шине не станет **идеально белой**. Только это гарантирует 100% чистоту.
*   **Лайфхак (Matte Finish):** Тщательно очищенная шина без нанесения чернителя имеет благородный темно-серый "графитовый" матовый цвет, который ценят многие любители стокового вида.
*   **Лайфхак:** Используйте очиститель шин для резиновых ковриков. Он идеально вытягивает грязь, делая их абсолютно черными без эффекта жирности.
*   **Гайд (микро):** При работе с щеткой двигайтесь короткими круговыми движениями, чтобы "выбить" грязь из микропор резины.

## 4. Меры предосторожности

*   **Руки:** Щелочные очистители шин очень сильно сушат кожу. Маслобензостойкие нитриловые перчатки — обязательны.
*   **Тормоза:** Избегайте попадания концентрированной щелочи на суппорты и тормозные диски.
