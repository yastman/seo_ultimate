# Очиститель дисков

Очиститель дисков — средство, которое растворяет тормозную пыль и нагар за 3–5 минут без щётки. Составы с индикатором меняют цвет на фиолетовый, когда железо растворилось — видно, что пора смывать. **Главное правило:** для дорогих дисков и карбон-керамики — только pH-нейтральные составы, кислота разъедает покрытие.

## Виды химии для дисков

| Тип состава  | pH    | Назначение                    | Ограничения                                                |
| ------------ | ----- | ----------------------------- | ---------------------------------------------------------- |
| Нейтральный  | 6–8   | Регулярная мойка, лёгкая пыль | Нет                                                        |
| Iron Remover | 6–8   | Тормозная пыль, металл        | Нет                                                        |
| Щелочной     | 10–12 | Жир, масло, дорожная плёнка   | **Запрещено для керамических тормозов, анодировки**        |
| Кислотный    | 1–3   | Застарелый нагар              | **Запрещено для хрома, полир. алюминия, PCCB, анодировки** |

## Чек-лист покупателя: что нужно кроме банки

### Инструменты — разные для дисков и резины

**Для чёрного глянца (Black Gloss):** Жёсткие ершики (типа EZ Detail) царапают глянец мгновенно — «паутина» появляется после первой мойки. Решение: только Wheel Woolies (шерстяные «палочки») или микрофибровая варежка.

**Для обычных литых:** Щётки средней жёсткости, ершики для спиц и болтов.

**Для резины:** Здесь наоборот — нужна жёсткая щётка. Мягкая просто гладит грязь, не выбивая её из пор шины.

### Химия «2-в-1» — щелочные составы

Tire & Wheel Cleaner (щёлочь pH 10–12) — универсальный вариант для тех, кто не хочет две отдельные банки.

- **Плюс:** Идеально моет резину (вытягивает коричневую пену/антиозонанты) и подходит для регулярной мойки дисков
- **Минус:** Если диски запущены (жёлтый нагар годами), «2-в-1» не справится — нужна кислота

### Финишная защита — экономия на будущее

Нанесение керамики для дисков (Rim Coat) или силанта после глубокой очистки позволяет в будущем мыть диски просто водой и бесконтактным шампунем. Тормозная пыль не въедается, смывается струёй воды. **Это сэкономит бюджет на химии в следующем сезоне.**

## Как выбрать средство для чистки дисков

**Литые диски, регулярный уход** → нейтральное средство для дисков или Iron Remover.

**Въевшаяся тормозная пыль (рыжие точки)** → очиститель колесных дисков с индикатором.

**Толстый слой нагара** → кислотная химия для чистки дисков + обильный смыв.

**Хромированные диски** → только нейтральные средства для мойки дисков.

**Полированный алюминий** → нейтральная жидкость для чистки дисков.

**Карбон-керамика (PCCB), анодированные диски, Center Lock** → **ТОЛЬКО** нейтральный шампунь. Кислота и щёлочь уничтожат покрытие.

**Литые с лакировкой** → средство для мойки литых дисков без кислоты.

## Как применять

1. **Остудите диски!** Температура должна быть ниже +30°C. Проверяйте не шину, а тормозной диск (аккуратно рукой на расстоянии).
2. Смойте основную грязь водой.
3. Нанесите средство для мойки дисков, начиная с внутренней стороны.
4. Выдержите 2–5 минут, не давая высохнуть.
5. Пройдитесь щёткой по спицам и болтам.
6. **Обильно пролейте водой** суппорта, болты и ниппели — химия не должна остаться в скрытых полостях.

**Pro tip:** Нанесение через пенник или пеногенератор удлиняет цикл работы состава — пена дольше держится на поверхности и не стекает, химия успевает растворить больше нагара.

## Критические ошибки

**Нанесение на горячие диски** — химия для дисков высыхает мгновенно, оставляя несмываемые разводы. Суппорта Brembo могут побледнеть от химического ожога — яркий лак станет матовым.

**Кислота на хроме или PCCB** — необратимые матовые пятна. Карбон-керамические тормоза стоят сотни тысяч — их убивает любая агрессивная химия.

**Кислота на дисках со сколами** — если есть «бордюрка», кислота проникает ПОД лак через скол и поднимает покрытие изнутри. Лак начнёт слезать паутинкой за сезон.

**Попадание щёлочи на резину** — вызывает Tire Blooming (порыжение шин). Щёлочь вымывает антиозонанты из резины. Смывайте состав с шин немедленно.

**Flash Rust после мойки** — если помыли диски Iron Remover и оставили машину мокрой на ночь, тормоза могут «прикипеть». После мойки проедьте 100 метров и пару раз резко затормозите.

**Использование средств с HF** — плавиковая кислота опасна для жизни. Проверяйте состав перед покупкой.

## Итог выбора

- **Регулярный уход:** Нейтральный Iron Remover 500 мл — безопасен для любого покрытия
- **Толстый нагар:** Кислотный очиститель дисков 1 л (без HF!) + обильный смыв водой
- **Хром, полировка, PCCB:** Только нейтральная жидкость для чистки дисков
- **На потоке:** Концентрат 5 л, разводится согласно инструкции
- **Инструмент:** Для чёрного глянца — только микрофибра/шерсть; для резины — жёсткая щётка
- **Защита:** Керамика или силант для дисков — чтобы мыть реже и экономить на химии

## FAQ: Ответы эксперта

### Почему Iron Remover меняет цвет и пахнет тухлыми яйцами?

Фиолетовый цвет — реакция на ионы железа (тормозная пыль). Запах серы — норма для качественного состава. Работайте на улице.

### Тормозная пыль въелась намертво. Iron Remover не помогает?

Iron Remover убирает только поверхностную пыль. Запёкшийся нагар (жёлтые углы спиц) снимается только кислотой — локально на пятна, потёрли 30 сек, обильно смыли.

### Очиститель испортил колпачки (Center Caps). Почему?

Колпачки часто из анодированного алюминия — щёлочь/кислота разъедает покрытие. Снимайте перед агрессивной химией или используйте pH-нейтральный шампунь.

### Можно ли мыть диски зимой сразу после трассы?

Нет! Горячие диски + ледяная вода = деформация тормозного диска (биение в руль). Дайте остыть 5–10 минут.
