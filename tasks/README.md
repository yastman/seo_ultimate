## Workflow

1. **Смотри ROADMAP** → `reference/ROADMAP.md`
2. **Текущий этап** → выполняй по чеклисту
3. **Отмечай** → `[x]` выполненные пункты
4. **Переноси** → выполненные ТЗ в `completed/`
