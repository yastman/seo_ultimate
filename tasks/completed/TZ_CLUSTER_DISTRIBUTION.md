## Инструкция по заполнению

Для каждого кластера заполнить колонки **Решение** и **Комментарий**:

**Решение** — одно из:

- `L3` — сделать отдельной подкатегорией
- `General` — оставить как ключи для L2
- `Filter` — сделать фильтром
- `Relocate → [путь]` — перенести в другое место
- `Merge → [кластер]` — объединить с другим кластером
- `Special: Главная` — особая страница

**Комментарий** — пояснение почему такое решение.
