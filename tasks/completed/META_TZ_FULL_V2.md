## Правила

### Title

**Адаптивная формула:**

```
ЕСЛИ keywords[0] ≤ 20 chars:
  {keywords[0]} — купить в интернет-магазине Ultimate

ИНАЧЕ:
  {keywords[0]} — купить, цены | Ultimate
```

**IRON RULE:** keywords[0] из `_clean.json` — ДОСЛОВНО. Не менять!

### Description

**Формула:**

```
От производителя автохимии Ultimate. {Назначение} — {типы/формы} в таре {объёмы}. Опт и розница. Доставка.
```

**Правила:**

-   НЕ дублировать keywords[0] (ключ уже в Title)
-   Длина: 100-160 chars
-   Объёмы точные из товаров

---

## Чеклист валидации

-   [ ] Title содержит keywords[0] ДОСЛОВНО
-   [ ] Title 30-60 chars (до `| Ultimate`)
-   [ ] Description 100-160 chars
-   [ ] Description содержит "от производителя" или "автохимии Ultimate"
-   [ ] Description содержит "Опт и розница"
-   [ ] Description НЕ содержит keywords[0] (не дублируем)
-   [ ] H1 = keywords[0] или близко
