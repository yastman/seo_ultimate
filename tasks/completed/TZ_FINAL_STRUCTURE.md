# ТЗ: Финальная структура категорий

**Дата:** 2026-01-02
**Статус:** Утверждено
**Принцип:** ХАБ (L2) → L3 по интенту. Разный интент = разная страница.

---

## Логика структуры

```
L2 (ХАБ) — собирает весь трафик категории
└── L3 — отдельная страница по интенту/типу товара
```

**Правило создания L3:**

- Разный интент покупки
- Разные товары (не пересекаются)
- Достаточный спрос (>100 vol)
