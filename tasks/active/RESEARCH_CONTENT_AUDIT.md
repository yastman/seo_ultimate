## Шаблон "Требования к ответам" (для ручной вставки)

```markdown
## Требования к ответам

1. **Каждый факт = источник.** Формат: утверждение → URL → цитата из источника.
2. **Без источника — не писать.** Если данных нет — указать "данные не найдены".
3. **Приоритет источников:**
    - Официальные сайты производителей
    - Научные статьи / тесты с измерениями
    - Детейлинг-форумы — только для практических советов
    - YouTube-тесты с измерениями
4. **Язык источников:** английский и русский.
5. **Актуальность:** предпочтительно 2020-2025.
6. **Спорные утверждения:** особое внимание блоку 6а — не подтверждать без proof.
```
