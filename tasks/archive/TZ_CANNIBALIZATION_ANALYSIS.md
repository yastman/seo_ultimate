# ТЗ: Анализ каннибализации и объединение кластеров

**Дата:** 2026-01-02
**Цель:** Предотвратить создание дублирующих страниц с одинаковыми товарами
**Принцип:** Одна страница = один интент = уникальный набор товаров

---

## Проблема

**Каннибализация** — когда несколько страниц сайта конкурируют за одни и те же ключи в поиске.

Результат:

- Google не понимает какую страницу ранжировать
- Размывается вес ссылок
- Падают позиции всех страниц
- Одни товары на разных страницах = дубли контента

**Правило:** Если на двух страницах будут одни и те же товары — это одна страница.

## Анализ кластеров на каннибализацию

### Группа 1: Микрофибра и тряпки

| #   | Кластер              | Vol  | Топ-ключи                                 | Товары             |
| --- | -------------------- | ---- | ----------------------------------------- | ------------------ |
| 8   | Тряпка для авто      | 4310 | тряпка для авто, тряпки для автомобиля    | Микрофибры, тряпки |
| 9   | Тряпка микрофибра    | 1500 | микрофибра для авто                       | Те же микрофибры   |
| 10  | Без разводов         | 820  | тряпка без разводов, микрофибра для сушки | Те же микрофибры   |
| 11  | Микрофибра для мойки | 260  | микрофибра для полировки                  | Те же микрофибры   |
| 12  | Для стекла           | 110  | тряпка для стекла                         | Те же микрофибры   |
| 13  | Для сушки            | 150  | тряпка для вытирания после мойки          | Те же микрофибры   |

**Проблема:** 6 кластеров = потенциально 6 страниц с ОДНИМИ И ТЕМИ ЖЕ товарами (микрофибры).

**Решение:**

```
БЫЛО: 6 кластеров → 6 страниц (каннибализация)
СТАЛО: 1 страница L2 "Микрофибра и тряпки" + фильтры
```

| Действие               | Кластеры                                |
| ---------------------- | --------------------------------------- |
| **General L2**         | #8, #9, #10, #11 → объединить в General |
| **Filter: Для стекол** | #12 → фильтр по назначению              |
| **Filter: Для сушки**  | #13 → фильтр по назначению              |

**Фильтры не создают отдельные страницы**, они фильтруют товары на одной странице.

### Группа 2: Щетки и кисти

| #   | Кластер              | Vol  | Топ-ключи            | Товары           |
| --- | -------------------- | ---- | -------------------- | ---------------- |
| 14  | Щетка для мойки авто | 1470 | щетка для мойки авто | Щетки            |
| 15  | Щетка для автомобиля | 1050 | щетка для автомобиля | Те же щетки      |

| 17  | Щетки для химчистки  | 580  | щетка для салона     | Кисти для салона |

**Анализ:**

- #14 и #15 — **ОДИН ИНТЕНТ** ("щетка для мойки" vs "щетка для авто" = синонимы)
- #16 и #17 — **БЛИЗКИЙ ИНТЕНТ** (кисти для детейлинга часто = кисти для химчистки)

**Решение:**

```
БЫЛО: 4 кластера → 4 страницы
СТАЛО: 1 страница L2 + 1 L3 (или только L2)
```

| Действие                     | Кластеры                                   |
| ---------------------------- | ------------------------------------------ |
| **General L2**               | #14, #15 → объединить (основные ключи L2)  |
| **L3: Кисти для детейлинга** | #16, #17 → объединить (если разные товары) |

**Вопрос исполнителю:** Товары в #16 и #17 пересекаются? Если да → объединить в General.

### Группа 4: Средства для кожи

| #   | Кластер       | Vol  | Топ-ключи                             | Товары       |
| --- | ------------- | ---- | ------------------------------------- | ------------ |
| 32  | Уход за кожей | 1710 | уход за кожей авто, полироль для кожи | Кондиционеры |
| 33  | Чистка кожи   | 340  | средство для чистки кожи              | Очистители   |
| 34  | General       | 190  | средство для кожи                     | Все          |

**Анализ:**

- #32 vs #33 — **РАЗНЫЙ ИНТЕНТ** (Conditioner vs Cleaner = разные товары)
- #34 — **ДУБЛЬ** общих ключей

**Решение:**

```
СТАЛО: L2 + 1-2 L3 (если товары реально разные)
```

| Действие                | Кластеры                          |
| ----------------------- | --------------------------------- |
| **General L2**          | #32, #34 → объединить             |
| **L3: Очистители кожи** | #33 → оставить ЕСЛИ товары разные |

**Вопрос:** Есть ли в ассортименте отдельно "очистители кожи" и "кондиционеры кожи"? Если нет → всё в General L2.

### Группа 5: Воски

| #   | Кластер         | Vol  | Топ-ключи             |
| --- | --------------- | ---- | --------------------- |
| 35  | General (Воски) | 2840 | воск для авто         |
| 36  | Твердый воск    | 1700 | твердый воск для авто |
| 37  | Жидкий воск     | 1120 | жидкий воск для авто  |

**Анализ:**

- Твердый и Жидкий — это **РАЗНЫЕ ПРОДУКТЫ**, не просто атрибуты:
  - **Твердый воск:** карнауба, пасты → ручная располировка → "защита надолго"
  - **Жидкий воск:** спреи, быстрые воски → быстрое нанесение → "быстрый блеск"
- Разный способ применения, разный результат
- Покупатель выбирает ИЛИ твердый ИЛИ жидкий — разный интент
- Высокий спрос: 1700 + 1120 = 2820

**Решение:**

```
СТАЛО: L2 "Воски" + 2 отдельные L3 подкатегории
```

| Действие              | Кластеры                 |
| --------------------- | ------------------------ |
| **General L2**        | #35                      |
| **L3: Твердый воск**  | #36 → отдельная страница |
| **L3: Жидкий воск**   | #37 → отдельная страница |

**СОЗДАТЬ** отдельные страницы L3 — это разные товары с разным интентом покупки.

## Итоговые правила

### Когда создавать L3 (отдельную страницу)

✅ Создавать L3 если:

- Уникальный набор товаров (не пересекается с другими)
- Высокий поисковый спрос (>500 vol)
- Разный интент (покупка разных вещей)

❌ НЕ создавать L3 если:

- Те же товары что в родительском L2
- Низкий спрос (<300 vol)
- Это атрибут/свойство товара (цвет, размер, тип) → Filter

### Когда использовать Filter

- Для стекол/для кузова → Filter
- Подарочный/профессиональный → Filter
- Аккумуляторный/сетевой → Filter
- Цвет, размер → Filter

**Исключение:** Твердый/Жидкий воск → **L3** (разные товары, разный интент)

## Чеклист для проверки каннибализации

Перед созданием любой новой L3 страницы проверить:

- [ ] Товары на этой странице УНИКАЛЬНЫ (не дублируют родителя)?
- [ ] Поисковый спрос > 300?
- [ ] Интент ОТЛИЧАЕТСЯ от родительской категории?
- [ ] Это НЕ атрибут товара (цвет, размер, тип)?

Если хоть один ответ "НЕТ" → использовать Filter или General.
