## December 2025 Update (v20.0)

Ключевые изменения в SEO:

| Изменение | Было | Стало |
|-----------|------|-------|
| **Keyword Density** | 0.5-1% safe zone | НЕ фактор ранжирования |
| **Title length** | 40-60 chars | **50-60 chars / 600px** |
| **Commercial modifiers** | Опционально | **Обязательно в Title** |
| **FAQ Schema** | Использовать | **Убрать** (не работает для e-commerce) |
| **Content length** | По количеству ключей | **По анализу конкурентов** |
| **og:title** | Дублировать title | **Оптимизировать отдельно** |
| **AI Overviews** | — | **87% e-commerce запросов** |

Источники: John Mueller (Dec 2025), Zyppy Q1 2025, Onely 2025
